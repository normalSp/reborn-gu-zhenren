Runtime promotion gate:
- Runtime file truth source is `src/canon/audio-source-manifest.json`.
- Public mirror is `public/audio/audio-source-manifest.json`.
- `npm run check:runtime-assets` verifies enabled audio manifest references exist and are non-empty.
