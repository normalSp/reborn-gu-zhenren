- 音频校验：所有 `runtimeEnabled=true` 的 BGM/SFX 文件存在且非 0 字节；Lightning Moment 已覆盖柳贯一与无极魔尊。

- PPTX 校验：`宣发素材/RebornG-v0.7.1-项目介绍.pptx` 必须能作为标准 PowerPoint 包打开。
