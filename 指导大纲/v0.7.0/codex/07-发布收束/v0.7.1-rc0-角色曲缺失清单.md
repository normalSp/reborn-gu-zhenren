# v0.7.1-rc.0 角色曲缺失清单

更新时间：2026-05-09

## 执行规则

- 根目录 `bgm/` 只作为用户原始素材暂存区，不作为 runtime 路径。
- 运行时只引用 `public/audio/bgm/...` 下整理后的音频文件。
- 下列曲目均已在 `src/canon/audio-source-manifest.json` 与 `src/canon/character-bgm-manifest.json` 中登记为 `user_supplied_reference`，但当前本地文件不存在，因此保持 `runtimeEnabled=false`。
- 文件补齐前，不允许启用 `nonprofit_fan_pack` 的私用角色曲 runtime；免费公布包 fallback 仍可运行。

## 补齐后验收

- 每个目标文件存在且长度大于 0。

- `src/canon/character-bgm-manifest.json` 中对应角色曲条目才可切换 `runtimeEnabled=true`。
- 跑 `npm test -- src/engine/audio-resource-policy.test.ts`、`npm run build`、`npm run test:e2e:long`。
