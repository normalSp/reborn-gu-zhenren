# RebornG v0.7.1-rc.0 发布说明

更新时间：2026-05-10

## 版本信息

- 版本：`0.7.1-rc.0`
- 存档格式：`SAVE_FORMAT_VERSION=15`
- 分支：`codex/v070-release-closure`
- 目标：作为进入 v0.8-a 前的发布候选版本，收束 v0.7 系统可信度、走查体验、角色曲接入、测试存档和宣发材料。

## 主要完成项

- 经济闭环：炼蛊、升炼、拆炼、喂养、商会、宝黄天、势力五账本、小队战和外派奖励均走真相源与闸门。
- 战斗与小队：单人战、小队战、`BattleTrace`、杀招/仙蛊使用、奖励回流和 AI 战斗候选已形成可验证地基。
- 行动与身份：开局身份路由、行动入口、气运可见性、十绝体显示、主修流派软倾向、声望中文化和叙事硬事实已收束。
- AI 闸门：DeepSeek 只能提出剧情、对话、战斗和传闻候选，不能直接写入未知资产、正式任务、地图或战斗奖励。
- 炼蛊/物资/商会：炼蛊筛选、升炼/拆炼预览、本命蛊说明、物资袋超载、商会卖蛊材和移动端窄屏可用性已做发布级修复。
- 音频：用户提供的角色曲已接入私用二创包，免费 BGM/SFX 保持可运行并有 manifest/attribution。
- 测试存档：新增中文测试存档与索引，旧英文存档继续保留供 e2e 兼容。
- 宣发物料：新增 B 站介绍 Markdown 与 16:9 PPTX。

## 测试计划

发布候选最终验收需要重新执行：

```bash
npm test
npm run build
npm run test:e2e:long
```

本轮执行结果：

- `npm test`：通过，55 个测试文件 / 357 个用例。
- `npm run build`：通过，最大 chunk 为 `combat-squad` 约 426.12 kB，未出现 500KB+ 警告。
- `npm run test:e2e:long`：通过，15 条 Playwright 长测路径。
- 长测中修正了“角色曲方源年轮”中文存档的玩家名污染，玩家仍为原创角色“听潮散修”。

人工走查建议覆盖：

- 七出身开局：古月、熊家、白家、商家、武家、铁家、散修。
- 调息、修行、突破、行动面板、野外行动。
- 人物图鉴、NPC 对话行动卡、事件日志。
- 炼蛊筛选、升炼、拆炼、蛊虫喂养、本命蛊说明。
- 商会买卖蛊材、物资袋超载、宝黄天全品类。
- 单人战、小队战、外派、地形/阵法、十绝体压力。
- 角色曲、杀招闪图、成就音效、移动端 375/430/768px smoke。

## 已知非阻塞项

- 完整昼夜修行、伏击全链路、完整升仙失败、复杂灾劫、高级采集生态、完整移动端系统、本命蛊深成长均归入 v0.8。
- 真实公开视频发布如需要“纯免费素材包”，应通过 audio manifest 切换 runtime 包或关闭私用二创曲包。
- 角色曲曲目审美仍可继续微调，但不再阻塞 v0.8-a 的工程开发。

## 发布判断

若自动回归和人工走查无 P0 问题，v0.7.1-rc.0 可以作为进入 v0.8-a 的基线。后续不建议继续把 v0.8 深系统回塞到 v0.7.1，避免再次扩大发布范围。
