# RebornG v0.7.1-rc.0 发布说明

更新时间：2026-05-09

## 版本与存档

- 运行版本：`package.json = 0.7.1-rc.0`
- 存档格式：`SAVE_FORMAT_VERSION = 15`
- 本版本是进入 v0.8-a 前的发布候选扫尾版本，重点是清理旧文档冲突、锁定开局身份路由、保留 v0.8 深系统承接边界，并把私用二创角色曲缺文件明确列为 v0.8-a 前阻塞项。

## 已收束内容

- v0.7.0-pre/a/b/c 的经济、炼养用、宝黄天、战斗、小队、势力、外派、十绝体、地形、闪图、免费音频包和长测地基已落地。
- v0.7.1-polish 已修复行动入口误导、气运/十绝体显示、势力声望中文化、主修流派软倾向、战斗触发说明、叙事数值一致性、炼蛊筛选和 AI 奖励一致性。
- v0.7.1 开局身份路由已接入 `StartProfile`，商家、武家、熊家、铁家、散修等开局不会再被章节默认位置写成古月族人。
- `game-dev-text` 已更新到 `2.2.0`，`reverend-insanity-lore` 已更新到 `0.2.0`，补齐当前版本文档入口、RC 闸门、构建警告回归规则、出身身份、宿命、十转与永生边界。

## 当前阻塞

- 用户确认的 19 个角色曲/名场面曲目尚未提供本地音频文件，私用二创包仍不得启用。
- 缺文件清单见：`指导大纲/v0.7.0/codex/07-发布收束/v0.7.1-rc0-角色曲缺失清单.md`。
- 放置规则：根目录 `bgm/` 只作为原始素材暂存区；运行时只引用整理后的 `public/audio/bgm/...` 路径。
- 在文件齐备前，免费公布包仍可运行；但按用户最新决策，进入 v0.8-a 前必须补齐私用角色曲文件或重新决策解除阻塞。

## v0.8 承接边界

- 不回塞 v0.7.1：完整 `SceneTimeContext`、昼夜修行、伏击全链路、完整大境界突破/升仙失败、复杂灾劫、高级采集生态。
- v0.8 将承接：出身深线、正史锚点、IF 宿命双路线、天意/因果账本、剧情候选闸门和结局解析。
- 永生、十转、永生蛊只作为未证悬念、求道余韵或远景线索，不作为 v0.8 可达成运行时资产或胜利结局。
