## 已完成
- C0 Playwright 基线：安装 `@playwright/test` 与 Chromium，新增 `playwright.config.ts`、`src/e2e/installE2eHarness.ts`、`tests/e2e/v070-long.spec.ts`。
- C1 音频资源库重建：移除旧运行时 AI/FFmpeg BGM 文件，新增 `public/audio/audio-source-manifest.json`，BGM/SFX 均以 manifest 为来源清单。
- C2 角色曲触发：新增 `src/canon/character-bgm-manifest.json`，首批登记方源《年轮》、龙公《春泥》，默认 `runtimeEnabled=false`，等待用户放入确认音频文件后启用。
- C3 闪图特效：扩展 `battle-asset-manifest.json`，新增 `BattleFlashOverlay`，单人战和小队战在使用杀招/重要仙蛊时可入队闪图事件；缺图时使用流派色 fallback，不阻塞 `DuelMove`、`BattleTrace` 和奖励结算。
- C4 扩展系统协议：新增地形/阵法配置、小队外派任务配置、十绝体灾劫/空窍压力画像、宝黄天稀有线索池。
- C5 音频诊断：`AudioManager` 扩为 `master/bgm/sfx/voice/ui` 五通道，voice 播放时通过 `soundSlice.voiceActive` 触发 BGM -6dB ducking；战斗/成就缺真实 SFX 时记录诊断，不再使用合成音冒充。
- C6 专项存档：新增 16 个 v0.7.0-c 手测存档，覆盖角色曲、闪图、宝黄天、地形、外派、十绝体、配音 ducking、诊断等。

## 剩余风险
- B站视频 BV1khRLBUEpk 当前需要人工在浏览器或 B站客户端复核曲目与时间戳；工程只登记用户确认的角色曲映射，不把未落地音频启用。
- 方源《年轮》、龙公《春泥》音频文件尚未放入 `public/audio/bgm/character/`，所以 manifest 仍禁用运行时播放。
- 免费 SFX 素材仍待用户或素材库确认；战斗/成就音效缺资源时现在会记录诊断并静默降级，不会再播放合成占位。
- 小队外派本轮落纯引擎协议，完整 UI 与多线剧情回流需要后续继续做。

## 后续建议
- 先由用户提供或确认角色曲、五域氛围 BGM、战斗 SFX、成就 SFX 文件，再把对应 manifest `runtimeEnabled` 打开。
- 小队外派 UI 应接入 `evaluateSquadDispatch/resolveSquadDispatchOutcome`，并把结果统一走经济、声望、传闻和关系闸门。
- 地形/阵法应在地图和战斗入口显示可解释预览，避免玩家只在结算后才知道为什么命中/逃跑变化。
