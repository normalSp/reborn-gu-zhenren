## 本轮已落地

### M5 炼蛊来源矩阵接入 UI

- `src/components/game/RefinePanel.tsx` 接入 `findMaterialSources`。
- 炼蛊详情页新增“来源矩阵”，显示库存、需求、来源。
- 若材料没有真相源来源，或库存不足，炼蛊按钮禁用，不能显示为可完成。
- 合炼仍不开放；当前界面保持炼蛊、升炼、拆炼。

### M6 蛊虫主动使用系统

- 新增 `src/canon/gu-use-registry.json`。
- 新增 `src/engine/gu-use-registry.ts` 与测试。
- `GuUseRegistryEntry` 登记 `useMode / targetRule / cost / cooldown / consumesGu / effects / sideEffects / loreRef / provenance / balanceTier`。
- `妇人心蛊` 登记为 `scene_gated`，不能普通点击使用，必须由剧情场景触发。
- `鸿运齐天蛊` 登记为 `consumable`，支持主动使用并给目标气运增益；当前 UI 先接 self，后续小队完成后扩展队友/NPC目标选择。
- `GuInventoryPanel` 将“激活”语义改为“启用/休眠”；主动使用另走“使用/剧情”按钮。

### M7 道心、声望与剧情双向闭环

- 新增 `src/engine/dao-reputation-policy.ts` 与测试。
- 事件类型包括杀戮、救援、背叛、交易、欺瞒、守诺、牺牲、夺宝、庇护、勒索。
- 同一事件可同时影响道心倾向和势力声望。
- `CharacterPanel` 的势力声望不再只显示数字，新增最近原因、当前权益、当前风险。
- `context-builder` 注入 `daoHeartNarrativeBias`，让 DeepSeek 在选项生成时感知玩家道心倾向。

### M8 四维资质可变但严格稀有

- 新增 `src/engine/attribute-mutation-policy.ts` 与测试。
- 资质、体魄、心智、气运都可变，但必须来自血颅蛊、力道蛊、智道蛊、运道蛊、梦境、灾劫、传承或引擎白名单。
- 血颅蛊提升资质必须绑定同族杀戮和重大声望/道心代价。
- 气运变更可影响自己、队友、NPC、boss；无察运手段时不直接明示。

### M9 舆图传闻层

- 扩展 `MapLocation` 字段：介绍、类型、危险度、设施、关联势力、资源线索、来源、可信度、传闻状态、行动。
- `mapSlice` 新增 `rumorLocations / addRumorLocation / promoteRumorLocation`。
- `state-update-applier` 将 AI 回传的新地名写入传闻地点，不直接升级为已知地点。
- `SVGMapPanel` 展示更详细的已知地点信息，并新增传闻地点列表。

### M10 元石调息重做

- 新增 `src/engine/primeval-meditation.ts` 与测试。
- 移除 `advanceTurn` 中低真元自动吸收元石的逻辑。
- 新增 `meditateWithPrimevalStone`：主动消耗 1 回合调息，按境界比例恢复真元。
- 候选比例：1转 35%，2转 25%，3转 18%，4转 12%，5转 8%。
- `StatusBar` 新增“调息”入口。当前默认按商路场景风险处理，后续可根据当前地点安全度切换 `safe / caravan / field / aperture`。
