## 一、小队/势力完整系统（核心新系统）

### 1.2 势力系统（`src/store/slices/factionSlice.ts` 重构）

#### 1.2.1 新增状态

```typescript
interface PlayerFaction {
  id: string;
  name: string;           // 势力名称（玩家自命名）
  domain: string;          // 所属域
  type: '正派' | '魔道' | '散修联盟' | '家族';
  level: number;           // 势力等级 1-10
  reputation: number;      // 声望 0-10000
  resources: { 元石: number; 仙元石: number; 蛊材: Record<string, number> };
  members: SquadMember[];  // 成员列表
  maxMembers: number;      // 最大成员数（随等级增长：等级1=3人, 等级10=12人）
  foundedAt: number;       // 创建回合
}

interface FactionEvent {
  id: string;
  type: 'recruitment' | 'conflict' | 'opportunity' | 'crisis';
  description: string;
  choices: { id: string; text: string; risk: string; outcome: string }[];
  resolved: boolean;
}
```

#### 1.2.2 势力创建条件

- 境界 >= 6转（蛊仙）
- 拥有至少一片福地/洞天
- 消耗**3000仙元石**（修正自50000，参见数值审计报告§1.5）
- 势力创建后获得势力旗帜、初始3个招募槽

#### 1.2.2.1 势力命名

按五域文化提供预设模板 + 自由输入：

| 域 | 模板示例 | 文化特色 |
|----|---------|---------|
| 南疆 | "XX山寨""XX洞府""XX山" | 山寨文化，以地名命名 |
| 北原 | "XX部""XX王庭""黄金XX" | 游牧部落，黄金家族影响 |
| 东海 | "XX岛""XX盟""XX楼" | 海岛散修联盟 |
| 西漠 | "XX绿洲""XX古城""XX沙海" | 沙漠绿洲文明 |
| 中洲 | "XX派""XX门""XX院" | 十大古派宗门体系 |

玩家可从模板中选择，也可自定义（8字限制，无敏感词）。模板选中后自动填入文本输入框。

#### 1.2.3 势力管理

| 功能 | 机制 | 消耗 |
|------|------|------|
| 升级势力 | 提升maxMembers+1、解锁高级功能 | 仙元石+声望 |
| 势力任务 | AI生成势力相关叙事（招募/冲突/机会/危机） | 每10回合触发 |
| 成员分配 | 指派成员执行任务（探索/守卫/贸易） | 成员时间（冷却） |
| 势力外交 | 与其他势力建立关系（同盟/敌对/中立） | 声望+外交行动 |

### 1.3 NPC招募系统

#### 1.3.1 招募来源

| 来源 | 条件 | 成功率 | 忠诚基础值 |
|------|------|--------|-----------|
| 现有NPC（已结识） | 好感度 >= 60 + npcs.json 中有该角色 | 好感度/100 | 好感度/2 |
| 道场/势力任务 | 完成试炼型道场后概率遇到 | 30-50% | 40-60 |
| 宝黄天招募令 | 购买"招募令"后在势力面板使用 | 保底1人 | 随机30-70 |
| 剧情AI叙事 | AI判定玩家声望/境界后引入 | 叙事决定 | 叙事决定 |
| 成就解锁 | 达到特定成就后自动解锁特殊NPC | 100% | 80-100 |

#### 1.3.2 NPC性格系统

| 性格 | 战斗行为 | 忠诚变化 | 特殊效果 |
|------|---------|---------|---------|
| **loyal(忠诚)** | 优先保护玩家 | 不易下降 | 濒死时替玩家挡刀 |
| **cunning(狡诈)** | 优先偷袭 | 势力弱时快速下降 | 侦察/情报成功率+20% |
| **reckless(莽撞)** | 优先强攻 | 胜利+忠诚, 失败-忠诚 | 暴击率+10% |
| **cautious(谨慎)** | 优先防御 | 稳定不波动 | 逃脱成功率+30% |
| **selfless(无私)** | 优先治疗友方 | 玩家受伤时+忠诚 | 治疗效率+25% |

**性格来源**：
- 原著NPC：从 `npcs.json` 中读取预设性格（如太白云生=无私、黑楼兰=狡诈）
- 随机招募NPC：性格随机，但受势力类型影响（魔道→更多狡诈/莽撞, 正派→更多忠诚/谨慎）
- 性格可在长期相处中微调（每50回合有1次性格转变事件）

#### 1.3.3 NPC死亡处理

| 战斗类型 | HP归零结果 | 恢复方式 |
|---------|-----------|---------|
| **训练**（道场对决） | 重伤（冷却恢复，10回合不可出战） | 自动恢复 |
| **实战**（剧情/势力战/白天黑天） | **阵亡**（永久不可用） | 不可恢复 |

> 设计理由：实战阵亡增加战术深度和情感重量，训练重伤降低挫败感。

#### 1.3.3 忠诚度系统

```
忠诚度 0-100

影响：
  100-80：全力战斗，不使用逃跑，有概率触发"舍身保护"
  79-60：正常战斗，偶尔防御
  59-40：战斗中概率防御/逃跑（30%）
  39-20：战斗中高概率逃跑（60%），可能拒绝执行危险任务
  19-0： 背叛——离开势力，可能带走装备/蛊虫

忠诚变化事件：
  +忠诚：每10回合+1（基础）、战斗胜利+2、给予蛊虫/资源+5、救治疗伤+3
  -忠诚：每10回合未出战-2、战斗失败-3、势力声望下降-5、长期未分配资源-1/回合
```

### 1.4 小队编成与战斗

#### 1.4.1 小队编成

- 出战上限：**4人**（含玩家）
- 编队界面：从势力成员中拖拽至出战位
- **战术姿态**（替代阵型——原著战斗为"杀招流"非"阵型流"，参见原著考证）:

| 战术姿态 | 效果 | 原著依据 |
|---------|------|---------|
| **合击** (Combined Strike) | 两人攻击同一目标+15%伤害 | 铁家四老合击、三兄弟合击 |
| **牵制** (Pinning) | 一人牵制敌方最强单位，其闪避-20% | 全书最高频战术(128处) |
| **掠阵** (Flank Support) | 压阵者每回合自动恢复最低HP队友5%HP | 铁若男掠阵、方源掠阵熊力 |
| **斩首** (Decapitation) | 全队集火最先击杀最弱敌人，首击+10%暴击 | 全书第一大反奴道战术 |

战术姿态在编成时选择，战斗中可消耗1回合切换。4人以上改为群像战斗（原 NarrativeCombatPanel 逻辑）。

### 1.5 白天/黑天升级

v0.6.0中白天/黑天为1v1连战（依次挑战荒兽）。v0.7.0升级为：
- 白天：3人小队 vs 2-3只荒兽（光翼兽+罡风隼+幻智猴）

- 仙材/仙蛊掉落率不变，但增加"小队奖励"（每多存活1个队友+5%仙材掉落率）

## 二、成就系统重做

### 2.1 当前问题

- 现有成就仅2个道心相关条件
- 成就与杀招、蛊虫、道场、势力等系统完全脱节

- 成就无奖励机制

### 2.2 新成就体系

#### 2.2.1 "里程碑"式成就替代"达到X"式（解决时间线冲突）

线性的"realmNum >= N"成就改为"首次突破至X转"的里程碑成就：

```json
{ "id": "realm_2", "condition": "flags._experienced_realm_2 === true" }
```

`flags._experienced_realm_N` 仅在通过正常突破流程（`state-update-applier.ts` 中境界提升）时写入。时间线起点直接赋予的境界不触发。**高转开局玩家需"转世重来"或开新低转档才能收集**——增加重复可玩性。

## 三、宝黄天全品类完成

### 3.1 杀招传承交易

**设计原则**：原著杀招极难获得，以**残方为主、完整传承稀少**。

| 商品 | 形式 | 频率 | 定价 |
|------|------|------|------|
| 杀招**残方** | 市场（与仙蛊残方同标签页） | 每期2-3个 | `杀招等级 × 300`仙元 |
| 杀招**完整传承** | 拍卖（稀有） | 每3-4期1个 | `杀招等级² × 300`仙元 |

凡级杀招便宜（3转残方=900仙元），仙级贵（6转完整=10800仙元）。玩家也可上架杀招传承出售。

### 3.2 仙蛊方交易

**交易形式**：市场（固定价格）| 上架数量：每期3-5个 | 刷新周期：每10回合 | 价格：`目标仙蛊tier² × 300`仙元。

### 3.3 影响文件

- 修改 `src/engine/auction-engine.ts`（杀招传承池生成）
- 修改 `src/components/game/AuctionPanel.tsx`（"传承"+"仙蛊方"标签页）

## 四、势力事件权重与地形战斗

### 4.1 势力事件可配置权重

玩家在势力面板调节"我更喜欢什么类型的势力事件"：

| 权重调节 | 效果 |
|---------|------|
| **招募倾向↑** | 更多NPC拜访/投靠事件 |
| **资源倾向↑** | 更多仙材采集/贸易机会事件 |
| **战斗倾向↑** | 更多剿匪/狩猎/势力冲突事件 |
| **外交倾向↑** | 更多与其他势力互动的叙事事件 |
| 默认 | 四者均等（各25%） |

每10回合触发一次势力事件。AI叙事根据权重偏向生成对应类型的事件。

### 4.2 地形战斗
| 地形 | 效果 |
|------|------|
| 平地 | 无影响 |
| 山地 | 力道/土道+10%效果 |
| 沼泽 | 毒道/水道+10%, 速度-10% |
| 洞穴 | 暗道+15%命中 |
| 高空(白天) | 风道/光道+15% |
| 深渊(黑天) | 暗道/魂道+15% |

地形数据存储在 `combat-constraints.json` 的 `scene.terrain` 字段中。小队战引擎在初始化时读取并应用。
