**公式**：`维护成本 = tier × rankMultiplier × pathMultiplier`
- `rankMultiplier`: divine=1.7, legendary=1.3, epic=1.0, rare=0.7
- `pathMultiplier`: 战斗流派(炎/雷/血/力/剑/暗)=1.2, 辅助(光/木/水/风/变化)=0.8, 特殊(宇/宙/智/律/运)=1.0
- `周期`: `20 + tier × 5`（回合）

**结算**：`immortalSlice.tickAperture()` 中追加逻辑——遍历 `apertureStorage.immortalGu`，`turn % upkeep.cycle === 0` 时扣除。余额不足→仙蛊进入"饥饿"(同凡蛊四态模型)。

## 缺口2：荒兽池扩展——从7种到15种

### 2.1 原著依据

**原文出处明确提及的荒兽种类**（非穷举）：
- 泥沼蟹（荒兽，方源狐仙福地地灾）——高防御荒兽代表
- 戾血龙蝠（上古荒兽，血海老祖九道真传之一，宋紫星持有）——血道荒兽代表
- 黑狱龙（游戏中已采用）——暗道荒兽BOSS
- 飓风狐/凤翼狐（芸芸提及）——风道荒兽
- 三尾狐、九尾狐（太古荒兽，人祖传）——奴道/幻道荒兽
- 桃太狼（荒兽级的狼，琅琊福地）——力道荒兽
- 彩鹿（荒兽，"神圣璀璨"，琅琊福地阁外）——光道荒兽

**原文分类体系**：荒兽 → 上古荒兽 → 太古荒兽（三档），荒兽身上可能寄生仙蛊。

## 缺口3：combatStatus字段

核心战斗蛊虫追加可选 `combatStatus` 字段（缺省使用流派默认映射）：

```json
{ "月光蛊": null, "火鸦蛊": {"type":"burn","chance":0.15,"potency":1},
  "毒液蛊": {"type":"poison","chance":0.20,"potency":1},
  "寒冰蛊": {"type":"freeze","chance":0.10,"potency":1},
  "血蛊": {"type":"bleed","chance":0.18,"potency":1},
  "春雷蛊": {"type":"stun","chance":0.12,"potency":1},
  "幽影随行蛊": {"type":"blind","chance":0.12,"potency":1} }
```

缺省行为：`combatStatus === undefined` → 使用 `PATH_STATUS_MAP` 流派默认映射。
