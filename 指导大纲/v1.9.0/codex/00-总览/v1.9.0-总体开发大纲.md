# v1.9.0 总体开发大纲

状态：draft；等待用户批准 D-190。
日期：2026-05-22
主题：v2.0 区域活世界预备与门禁收束

## 目标

v1.9 的目标不是做完区域活世界，而是为 v2.0 做好可审计的准备：

- 选定第一个 v2.0 核心区域。
- 设计区域事件、身份路线、路线状态、生存经济、社会压力、冲突后果的合流方式。
- 严肃评估 `SAVE_FORMAT_VERSION = 25`、`regionalEventLedger`、`runFingerprint` 和相关持久字段。
- 建立 MiroFish v2 prelude topic-slice request/intake 计划。
- 形成 v2.0 长测、live probe、Player Advocate 和测试矩阵硬门。

## 不做事项

- 不默认新增 save 字段。
- 不默认改 `SAVE_FORMAT_VERSION = 25`。
- 不默认新增 runtime tab。
- 不默认导入 MiroFish。
- 不默认让 DeepSeek 看全书摘要或 RAG。
- 不默认启用子代理。
- 不自动部署 EdgeOne。
