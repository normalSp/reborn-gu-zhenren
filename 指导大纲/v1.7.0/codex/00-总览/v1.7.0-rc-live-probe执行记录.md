## P2 风险

live 输出偶发“木牌/令牌/登记/成员身份”等偏正式道具词。它们没有形成正式身份、奖励、地点、阵营或 NPC 命运，但容易让玩家误读为准正式系统。

处理：

- v1.7 不阻断，但写入 future prompt/eval hardening 样本。
- 若 v1.8 继续区域活世界，应把“偏正式道具降级为口头试探/普通杂务/观察机会”写入 prompt/evaluator。

## 工具修正

- fetch/network error 进入 retry；最终失败计入 P1。
- forbidden claim evaluator 区分否定、问题复述、未来门禁和实际结论。
- 正式地点/价格表/库存等词不再作为单词硬禁，改用结论模式判定。

## 结论

v1.7-rc live gate 通过。通过范围仅限本轮 50 轮区域活世界 live probe，不代表 300+ 轮长线区域活世界质量已完成。
