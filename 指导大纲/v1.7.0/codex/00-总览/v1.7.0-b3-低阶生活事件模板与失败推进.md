## 失败推进原则

| 失败类型 | 可显示反馈 | 禁止升格 |
|---|---|---|
| 被盘问 | 公开证据不足、需解释/旁证 | 正式通缉/封锁 |
| 加入商队被拒 | 短工、货物、报到前置 | 正式商队身份 |
| 市场压价 | 观察、打听、换窗口 | 成交、价格表、库存 |
| 临时遮蔽 | 人情压力、低阶劳作 | 债务 ledger、NPC 命运 |
| 路途事件 | 守规矩、补给压力 | 正式路线成功 |
| 远城目标 | 门槛、外缘压力 | 完整商家城开放 |
