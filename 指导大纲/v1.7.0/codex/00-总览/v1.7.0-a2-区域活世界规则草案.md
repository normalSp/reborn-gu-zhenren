## Same-start replayability rule

同一开局重复运行时，v1.7 允许差异来自：

- 抽到不同 pressure deck。
- 同一压力使用不同公开前置表达。
- 玩家被引向不同候选：解释身份、绕路、短工、卖货、求助、等待、打听。
- DeepSeek 对同一公开压力的叙事措辞差异。

禁止差异来自：

- 正式地点、阵营、奖励、NPC 生死变化。
- hidden/private 泄露。
- 未经 runtime gate 的 MiroFish item 直接进入叙事。
- 同一存档内前后事实矛盾。

## b1 projection hint

若 b1 获批 runtime 第一刀，建议只实现纯 projection：

`routeLocationState + survivalEconomyState + livingWorldState + localActionLedger + v1.3/v1.4/v1.5 projection -> regional life pressure candidates`

输出只显示：

- 当前外缘压力。
- 可见前置。
- 失败/拒绝后的下一步。
- 禁止写入说明。

不写：

- 新 save 字段。
- 正式区域状态。
- 商队身份、交易结算、奖励。
- DeepSeek visible MiroFish summary。
