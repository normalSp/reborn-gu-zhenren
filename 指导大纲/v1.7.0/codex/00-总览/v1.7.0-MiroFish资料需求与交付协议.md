## 何时升级 blocking

以下任一项出现，MiroFish 需求升级为 `blocking`：

- 正式 topic-slice 要进入 a2 intake。
- 需要命名地点、命名 NPC、正式势力规则。
- 触碰 hidden-adjacent、方源公开证据、后期高阶事实。
- 任意内容准备进入知识索引、测试样本、rule draft 或 canon draft。
- DeepSeek 可见摘要或 prompt context 候选。
