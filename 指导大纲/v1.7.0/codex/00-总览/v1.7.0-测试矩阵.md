# v1.7.0 测试矩阵

状态：completed
日期：2026-05-21

## 矩阵目标

v1.7 的风险不是单个 UI，而是“活世界纵切”把多个旧地基合流时越权：

- 区域样板被误写成正式地点。
- 商队/散修/城门外缘被误写成正式阵营或身份。
- MiroFish topic-slice 候选直接变 canon。
- DeepSeek 把 hidden/private 或正式结论写进叙事。
- 玩家目标在多轮 replay 中丢失。
- 失败、拒绝、缺补给被写成死路，而不是低阶生活后果。
- 同一开局重复运行时只有换皮文本，没有可审计的低阶压力差异；或反过来，差异来自正式事实漂移。
