# v1.8.0-b2 同开局差异度可测化

状态：已完成
日期：2026-05-21

## 结论

b2 把“同一开局也有不同体验”收束为可测 projection 指标，而不是新增 `runFingerprint`。

## 可变项

允许变化：

- 当前强调的身份候选。
- 压力卡排序。
- 风险 framing。
- 安全下一步措辞。
- DeepSeek 叙事表达。

必须稳定：

- 存档版本。
- 玩家正式身份/职业。
- 路线、地点、阵营、奖励、库存、价格、NPC 生死。
- hidden/private。
- 本地引擎权威。

## 实现方式

本地 engine 用只读 evidence 与 `variantIndex` 生成不同候选强调；不把差异写入存档，也不让 DeepSeek 决定正式事实。

## 测试样本

| 样本 | 期望 |
|---|---|
| 同一开局三次展开 | 候选强调不同，稳定事实一致 |
| 要求固定职业身份 | 降级为候选身份压力 |
| 要求正式地点/阵营 | 阻断，转为前置条件或观察 |
| 要求奖励/库存/价格 | 阻断，转为资源压力 |

## 结论

b2 通过。`runFingerprint` 仍保留在 future_sample_pool，只有当后续证明持久差异收益大于 schema/回滚风险时，才进入单独 save-format 门禁。
