## 为什么不是直接做 v2.0 区域活世界

v1.7 只证明小范围 projection/live gate 通过，不证明 300+ 轮长线区域生活已经稳定。v1.8 应补“玩家身份与可重玩差异”的中间层，否则 v2.0 的区域事件会缺少玩家为什么被接纳、排斥、压价、雇佣或盯上的基础。

## 为什么不是只做职业列表

蛊真人味道的低阶身份不应是 RPG 职业皮肤。它必须和路线、生存、势力压力、公开事件、信息不对称、风险和失败推进绑定。身份路线应先是本地 projection / pressure deck / test sample，不是正式职业数值系统。
