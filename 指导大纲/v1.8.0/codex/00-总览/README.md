## 专家团推荐主线

推荐主线：

`低阶身份路线与同开局差异度地基`

推荐目标：

在 v1.7 已有的区域活世界 projection 上，增加“玩家是谁、靠什么活、被谁接纳或排斥、同一开局为什么会有不同低阶压力”的第一层解释。

推荐身份路线候选：

| 身份路线 | 当前定位 | 禁止误写 |
|---|---|---|
| 商队学徒 / 临时帮工 | 跟随商队、做杂务、换取庇护和信息 | 不等于正式商队成员、商家城入籍或稳定工资 |
| 散修短工 | 靠零散劳力、低阶资源、临时人情活下去 | 不等于自由强者或完整散修势力 |
| 低阶护卫 / 护送候选 | 接触巡路、护送、站岗、护货风险 | 不等于正式佣兵系统、战斗奖励池或 NPC 生死结论 |
| 采集跑腿 | 山路、野外、低阶资源线和补给压力 | 不等于稳定材料农场或正式价格表 |
| 情报跑腿 / 传话人 | 听风、递话、观察、避祸 | 不等于隐藏事实揭示、方源私密因果或正式间谍身份 |
