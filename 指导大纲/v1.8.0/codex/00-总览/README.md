## 专家团推荐主线

推荐主线：

`低阶身份路线与同开局差异度地基`

推荐目标：

在 v1.7 已有的区域活世界 projection 上，增加“玩家是谁、靠什么活、被谁接纳或排斥、同一开局为什么会有不同低阶压力”的第一层解释。

推荐身份路线候选：

| 身份路线 | 当前定位 | 禁止误写 |
|---|---|---|
| 商队学徒 / 临时帮工 | 候选标签 / 测试样本 | 不等于正式商队成员、商家城入籍、稳定工资或正式任务 |
| 散修短工 | 候选标签 / 测试样本 | 不等于自由强者、完整散修势力或正式据点 |
| 低阶护卫 / 护送候选 | 候选标签 / 测试样本 | 不等于正式佣兵系统、战斗奖励池或 NPC 生死结论 |
| 采集跑腿 | 候选标签 / 测试样本 | 不等于稳定材料农场、正式价格表、库存或炼蛊成功率 |
| 情报跑腿 / 传话人 | 候选标签 / 测试样本 | 不等于隐藏事实揭示、方源私密因果或正式间谍身份 |
