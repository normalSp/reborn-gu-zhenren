# v1.8.0-b3 身份特异失败推进

状态：已完成
日期：2026-05-21

## 结论

b3 将身份路线候选的失败推进限定为“解释压力与下一步”，不生成正式惩罚、奖励、身份、地点或 NPC 命运。

## 失败推进样式

| 候选 | 失败或阻断 | 安全推进 |
|---|---|---|
| 商队临时帮工 | 被盘问、被压价、报到前置不足 | 继续观察、做短工、等待管事窗口 |
| 散修短工 | 被拒绝、缺少担保、夜路风险 | 找临时遮蔽、换零活、降低目标 |
| 低阶护卫 | 战斗风险、雇主不信任 | 描述战术交给本地引擎、转为守夜/绕路 |
| 采集跑腿 | 资源点不稳定、受伤或被驱赶 | 记录补给压力、换路线或等待 |
| 情报跑腿 | public evidence 不足、靠近 hidden/private | 只做公开观察/传话/避祸 |

## 边界

- 不给正式职业。
- 不给正式商队/散修/护卫身份。
- 不创建稳定材料农场。
- 不写价格表、工资、任务奖励。
- 不裁决战斗胜负或 NPC 生死。
- 不揭示方源私密因果或 hidden/private。

## 结论

b3 通过。失败推进成为玩家理解压力的文本地基，不是正式失败系统。
