## 硬边界

本阶段不授权：

- runtime / UI / store / prompt 行为变更。
- `SAVE_FORMAT_VERSION`、新增持久字段或 migration/defaults。
- live DeepSeek、模型切换、DeepSeek visible knowledge / RAG 或 authority 扩张。
- BFF/backend。
- subagents。
- MiroFish export。
- 第三方 agent SDK/framework 依赖或 PoC。
- 外部 SDK/agent 文件读取、命令执行、patch artifact 或 git 操作。
- 正式地点、阵营、奖励、NPC 生死、canon promotion、hidden/private body 可见化。

## MiroFish 分类

startup/a0 默认：`not_needed`。

原因：

- 当前只做抽象低阶 NPC / 势力 archetype 和工程门禁。
- 不新增命名 NPC。
- 不触碰 hidden-adjacent。
- 不产生正式 lore 结论。

升级 blocking 的触发：

- 命名原著 NPC 或原著关键人物进入样本。
- L4/L5 事实锚点进入样本。
- hidden-adjacent、方源公开证据或正式 lore 结论进入样本。
- 正式地点、阵营、身份、奖励或 NPC 生死进入样本。
