状态：completed。

## 目标

把 b1 runner 从“能跑样本”推进到“可复核、可分类、可回放”的 Agent Lab 证据格式。

本阶段仍是 report-only / offline：

- 不调用 live DeepSeek。
- 不启用 subagents。
- 不引入外部 SDK/framework。
- 不写 runtime、save、canon、DeepSeek context 或知识库正文。

## multi-turn / rescore 设计

runner 通过以下字段形成离线多轮结构：

- `roundId`：10 个离线 round。
- `agentSlot`：20 NPC + 1 L5。
- `replay_lane_marker`：用于确定性回放对照。
- `factionPressureRefs`：3 个抽象 pressure source。
- `sourceEventRefs` / `visibleFactRefs`：只使用 synthetic 或既有 public boundary ref。

runner 对同一批样本执行二次 scoring，并比较：

- `worldCoreResult`。
- finding severity/code 序列。

本次结果：`rescoreStable = true`。

## failure taxonomy

当前 v2.2 taxonomy 分为：

| 分类 | 判定 | 本次结果 |
|---|---|---|
| schema failure | 字段缺失、非法 layer/kind/visibility/memoryStatus/effect | 0 |
| source/visible/faction ref drift | ref 不在样本 allowlist | 0 |
| hidden leak | hidden ref 越权、hidden/private body key、forbidden text | 0 |
| formal authority drift | candidateEffect 或文本宣称正式地点/阵营/奖励/NPC 生死/凭信/结局 | 0 |
| memory contamination | subjective/rumor/reflection 被写成 canon/save/persistent memory | 0 |
| L5 overreach | L5 不使用 `heaven_will_pressure` 或不只输出 macro pressure | 0 |
| idle loop | 缺少 `idle_breaker` 或 `safe_next_step` | 0 |
| rescore instability | 二次 scoring 结果不一致 | 0 |
