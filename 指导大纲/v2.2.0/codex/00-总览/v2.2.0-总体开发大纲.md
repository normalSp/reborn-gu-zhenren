## 关键设计原则

- agent 不是事实源。
- memory 不是 canon。
- reflection 不是正式结论。
- L5 不是可聊天 NPC。
- 势力 pressure 不是正式势力状态。
- report-only artifact 不是正式存档。
- runner 通过不等于 runtime agent 获批。

## v2.2 不做

- 不接 runtime agent。
- 不调用 live DeepSeek，除非后续单独批准。
- 不引入第三方 agent framework。
- 不做外部 framework PoC。
- 不启用子代理。
- 不建 BFF/backend。
- 不写正式存档或新 save fields。
- 不新增 MiroFish export。
- 不开放正式地点、阵营、奖励、NPC 生死或 canon promotion。
