## 实现范围

新增 runner 继承 v2.1 自有 zero-dependency / dry-run / report-only 思路，但扩大到 v2.2 设计规模：

- scenario：`outer_edge_agent_lab_synthetic_v1`。
- 样本：21 条。
- 非 L5 agent：20 个 unique NPC。
- 抽象 pressure source：3 个。
- L5：1 个 `l5_heaven_will_pressure`。
- round：10 个离线 round。

runner 检查：

- schema 100%。
- WorldCore post-check 结果符合预期。
- `authority` 必须是 `proposal_only`。
- `requiresWorldCoreAdjudication` 必须为 true。
- `agent_private_candidate` 不得变成 hidden/private lore、save、canon、runtime 或 DeepSeek visible。
- 每条 proposal 必须包含 `safe_next_step`、`idle_breaker`、`replay_lane_marker`。
- hidden leak、formal authority drift、memory contamination 为 blocking。
- L5 只能输出 `macro_pressure_constraint`，不能宣判结局、宿命或正史。
