# v3.2.0 总体开发大纲

日期：2026-05-24
状态：startup completed；pending user approval

## 版本目标

v3.2 的目标是完成一次不接入 runtime 的完整 rehearsal：

1. 构造 synthetic world snapshot。
2. 生成 L2/L3 proposal batch。
3. 通过 WorldCore post-check rehearsal 做接受、拒绝、人工复核、future gate 分类。
4. 输出 approved expression sample，但不进入 UI 或存档。
5. 形成 rejection/manual_review/future_gate 趋势报告。
6. 为 v3.3 准备 F-300 reopen 决策包。

## 工程原则

- 只用 RebornG 自有零依赖报告型工具。
- 样本只用 synthetic / redacted / abstract fixtures。
- 所有输出都是 report-only。
- WorldCore final authority 必须在报告字段里体现。
- 任何候选都不能自裁决、自写存档、自升格成 canon。

## 通过标准

- P0/P1 falseNegative 必须为 0。
- resultMismatch 必须为 0。
- missingRequiredFamilies 必须为 0。
- 所有 boundary assertions 必须为 false。
- `runtimeImplementationApproved=false` 必须保持。
- 若输出 v3.3 准入建议，只能是决策包建议，不是 implementation approval。

## 不运行项

startup 不运行 runtime 测试。D-320 获批前，也不新建 checker 或样本。
