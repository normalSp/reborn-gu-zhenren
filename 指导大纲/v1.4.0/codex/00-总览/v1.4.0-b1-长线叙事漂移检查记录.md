# v1.4.0-b1 长线叙事漂移检查记录

状态：T0-lite deterministic drift 通过；live probe 未执行
日期：2026-05-21

## 检查范围

v1.4-b1 触碰 route/location/region 的玩家可见体验，因此触发长线叙事漂移门禁。

本轮风险是：

- 区域样板逐步说成正式地点。
- 商队窗口逐步说成交易/加入成功。
- 散修落脚提示逐步说成正式身份或据点。
- 商家城外缘门槛逐步说成进入商家城核心。
- DeepSeek / UI 文案泄露隐藏事实或写奖励、NPC 生死。
