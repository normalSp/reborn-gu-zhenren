# v1.4.0-b3 商队/散修/城外缘接触窗口

状态：已完成
日期：2026-05-21

## 目标

把玩家离开青茅后的“下一步接触对象”降成受控窗口，而不是直接开放地点、身份或势力。

## 窗口

| 窗口 | b3 表达 | 禁止 |
|---|---|---|
| 商队接触窗口 | 询价、递话、担保、补给风险 | 加入商队、正式交易、价格/库存 |
| 散修落脚提示 | 落脚传闻、补给压力、身份风险 | 正式据点、散修身份、奖励 |
| 商家城外缘门槛 | 城外缘公开理由、担保、追查风险 | 进入城市核心、正式地点、商店 |

## Runtime 位置

- `postureCards.caravan_contact_window`
- `postureCards.rogue_settlement_hint`
- `postureCards.city_outer_threshold`

每张卡都固定：

- `canUnlockLocation = false`
- `canTransferFaction = false`
- `canGrantReward = false`
- `canSetNpcFate = false`
- `canPatch = false`
- `statePatchApplied = false`

## 结论

b3 完成。玩家可看见区域接触方向，但不会获得正式身份、地点、交易或奖励。
