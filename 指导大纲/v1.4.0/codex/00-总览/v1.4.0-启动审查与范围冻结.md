# v1.4.0 启动审查与范围冻结

状态：草案已建立；等待用户审批
日期：2026-05-21

## 版本定位

v1.4 不是“开启完整南疆”，而是建立第一个南疆早期低阶区域样板，让玩家在离开青茅后的外缘空间里看到可行动、可受限、可反馈的区域格。

## 建议范围

v1.4 可以讨论并逐步实现：

- 商队路线候选与商队接触窗口。
- 山路/荒野/散修聚点/城外缘等低阶区域格。
- 区域危险、补给、身份检查、追踪压力。
- 区域公开事实卡与公开传闻。
- 区域事件模板的 projection-only 第一刀。
- route/location、survival economy、social pressure 的组合可读性。

## 暂不开放

- 完整南疆地图。
- 完整商家城。
- 正式进入城市、正式地名结论或长期区域 ownership。
- 正式阵营转移、招揽成功、通缉解除、封锁解除。
- 正式交易结算、价格表、库存、奖励。
- NPC 生死、捕获、背叛。
- DeepSeek 写正式地点、区域状态或经济结算。
- BFF/backend、自动部署、公测新口径。

## 用户输入分类

本阶段需要用户输入类型：

- direction：是否批准 v1.4 主线为南疆早期低阶区域样板。
- boundary：完整南疆/商家城/正式区域字段是否继续禁止。
- priority：第一刀偏 route、survival、social 还是区域事件。
- tradeoff：继续 projection-first，还是评估最小持久区域聚合对象。
- cost：rc 是否需要 live probe，成本到 rc 前再确认。

## 当前结论

先建立 a0 文档与门禁，不进入 runtime。D-140 系列获批后进入 a1 save-format / authority 设计门禁。
