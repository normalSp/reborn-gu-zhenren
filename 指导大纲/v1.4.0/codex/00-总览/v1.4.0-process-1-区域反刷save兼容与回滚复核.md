## 反刷复核

v1.4 不新增收益结算，因此反刷风险集中在 UI/叙事误导：

| 风险 | 处理 |
|---|---|
| 反复刷商队窗口获得交易/奖励暗示 | forbiddenWrites 阻断 `formal_trade`、`inventory_delta`、`reward` |
| 反复刷散修落脚获得身份/据点暗示 | 阻断 `formal_rogue_identity`、`location_unlock` |
| 反复刷城外缘获得入城结论 | 阻断 `full_shang_clan_city`、`location_unlock` |
| DeepSeek 生成正式区域结论 | 本轮无 DeepSeek context/authority 变化；后续如引入必须 live probe |
