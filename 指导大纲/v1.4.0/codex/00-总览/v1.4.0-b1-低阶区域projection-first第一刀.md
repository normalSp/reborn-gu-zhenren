# v1.4.0-b1 低阶区域 projection-first 第一刀

状态：已完成
日期：2026-05-21

## 目标

把 v1.1 `routeLocationState`、v1.2 `survivalEconomyState`、v1.3 社会压力投影和 v0.18 route/multi-region reviewed evidence 组合成玩家可读的南疆早期低阶区域样板。

## Runtime 变更

| 文件 | 内容 |
|---|---|
| `src/engine/v140-region-sample-projection.ts` | 新增纯函数 `buildV140RegionSampleProjection()`。 |
| `src/components/game/RegionSamplePanel.tsx` | 新增世界面板 `区域` tab。 |
| `src/components/game/WorldHubPanel.tsx` | 接入 `region_sample` tab 和总览卡片。 |

| `tests/e2e/v140-region-sample-projection.spec.ts` | 浏览器路径验证 UI 可读、无区域存档字段、无正式结论。 |

## 明确没有做

- 没有 bump `SAVE_FORMAT_VERSION = 25`。
- 没有新增 `regionSampleState` / `regionalSampleState`。
- 没有 store action。
- 没有 migration/defaults。
- 没有 DeepSeek prompt、context 或 authority 变化。
- 没有正式地点、完整南疆、完整商家城。
- 没有正式交易、价格、库存、奖励、阵营、通缉、NPC 生死。

## 区域 posture

| posture | b1 口径 |
|---|---|
| 山路外缘 | 读取路线/地点与 v0.18 山路候选证据。 |
| 商队接触窗口 | 读取市场窗口、商队边界和社会压力；不成交、不写商队身份。 |
| 散修落脚提示 | 读取补给、身份、追索压力；不写据点或身份。 |
| 商家城外缘门槛 | 只显示城外缘门槛；不进入城市核心或正式地点。 |
