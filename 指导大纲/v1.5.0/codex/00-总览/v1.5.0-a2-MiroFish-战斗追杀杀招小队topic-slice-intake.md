## 3. 晋升边界

允许进入 b1 的只有 RebornG 自己转写后的：

- posture id。
- source pointer id。
- 只读风险提示。
- 反制 / 小队 / 阵法前置提示。
- 测试样本。
- UI 边界文案。

禁止进入 b1：

- 原文摘录、quote、original prose。
- hidden/private body。
- 全书基础包原始条目。
- 正式 `conflictConsequenceState` 或同类持久字段。
- 正式追杀、通缉、封锁、阵营敌对结论。
- 正式掉落、奖励、稀有蛊、仙蛊、完整杀招传承。
- NPC 死亡、捕获、背叛、永久伤势。
- 方源私密因果、重生、春秋蝉、隐藏事实正文。
- DeepSeek 战斗 / 奖励 / 生死 / 追杀 authority。

## 5. blocking 升级条件

以下任一项出现时，MiroFish 从 `preferred` 升级为 `blocking`：

- 正式命名 NPC 追杀、截杀、死亡、捕获、背叛。
- 原著角色参与战斗结论。
- 特定杀招传承、归属、效果 canon 晋升。
- 隐藏事实邻近冲突、方源私密因果、重生 / 春秋蝉相关诱导。
- 正式势力通缉、围剿、招揽、封锁、敌对关系。
- 任何内容进入 runtime canon、DeepSeek visible context、玩家可见事实或硬测试事实。
