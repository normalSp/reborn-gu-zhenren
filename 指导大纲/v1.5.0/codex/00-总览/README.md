## 专家团建议

v1.5 的主线建议为：`冲突、追杀、杀招与小队后果深化`。

优先做：

- 冲突/追杀风险的 projection-first 解释层。
- 战斗后果如何影响路线、生存压力、社会压力和区域姿态的只读归因。
- 杀招反制、失败代价、小队/阵法前置条件的可读性样板。
- 旧战斗系统、v0.17 战斗深化、v1.1-v1.4 状态地基之间的权威归并。

暂不做：

- 正式战斗后果持久字段。
- 正式掉落池、稀有蛊、仙蛊、完整杀招传承。
- NPC 生死、捕获、伤残、背叛或永久关系结论。
- 势力通缉/追杀的正式结论。
- DeepSeek 战斗、奖励或生死 authority。
