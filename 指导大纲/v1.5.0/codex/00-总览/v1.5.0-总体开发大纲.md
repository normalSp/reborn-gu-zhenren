# v1.5.0 总体开发大纲

状态：completed
日期：2026-05-21
主题：冲突、追杀、杀招与小队后果解释层第一阶段

## 目标

把冲突从“战斗入口/战斗 UI”推进到“活世界后果解释层”：

- 冲突从哪里来。
- 玩家为什么被追、被截、被盘查或被威胁。
- 杀招/小队/阵法为什么能用或不能用。
- 战斗后果如何影响路线、生存、社会和区域压力。

## 第一刀建议

b1 默认使用 projection-first：

- 读取 existing combat trace / action ledger / routeLocationState / survivalEconomyState / livingWorldState / region sample。
- 输出冲突压力、追杀风险、反制缺口、可选准备。
- 写 UI，不写 store。
- 写 tests，不写 migration。
