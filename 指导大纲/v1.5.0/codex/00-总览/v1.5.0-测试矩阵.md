# v1.5.0 测试矩阵

状态：a1 active
日期：2026-05-21

## 矩阵目标

v1.5 的高风险不是“UI 能显示冲突”，而是：

- 是否暗中开放掉落/奖励。
- 是否暗中写 NPC 生死或捕获。
- 是否把追杀/通缉变成正式阵营结论。
- 是否让 DeepSeek 结算战斗或生死。
- 是否让杀招/小队/阵法绕过 engine/canon 前置。
