## 发现

- 玩家最容易误解的是“Auto-Theater Combat 是否等于已经做了新战斗 runtime”。当前文案需要持续强调：v4.0 是 design-gate / report-only / checker-first。
- 第二个高风险误解是“自走棋是否替代全部战斗”。当前结论应持续写成：不采用纯自走棋；凡阶棋盘保留为局部镜头；高阶主表达是 theater / stack / ledger。
- 第三个高风险误解是“天道/宿命是否可以裁决玩家结局”。当前结论应持续写成：pressure / constraint / risk，不是 verdict。
