## v4.0 范围锁定

v4.0 主线锁定为：`Auto-Theater Combat 与 HeavenWill/Fate Pressure 高阶世界表达模型`。

允许自动推进：

- a1 冻结 Auto-Theater Combat、Auto-Theater Lite、battle theater、killer-move stack、Combat Ledger、Immortal Gu House boundary、美术与交互语法。
- a2 冻结 Fate state、HeavenWill pressure、era anchor、causality debt、L5 macro director。
- b1 新增自有 zero-dependency report-only v400 readiness checker。
- b2 新增 synthetic/generic negative fixtures 和 deterministic hardening evidence。
- b3 新增 60 轮 Player Advocate、old-save/no-save/rollback 证据。
- process-2 与 rc 完成长线漂移、知识边界、Skill sync、项目入口和 Git/CI 收束。

不允许自动推进：

- 高阶战斗 runtime、凡阶战斗 runtime 迁移、纯自走棋 runtime、theater UI、Auto-Theater 素材生成。
- 杀招栈数值结算、仙蛊屋 runtime 状态、环境破坏结算、NPC 生死、正式奖励。
- HeavenWill/Fate runtime 裁决、L4/L5 runtime、原著关键人物 agent。
- save field、`SAVE_FORMAT_VERSION` bump、`runFingerprint`。
- live DeepSeek、DeepSeek prompt/context/model/authority 扩大、DeepSeek visible lore/RAG。
- MiroFish export/intake、真实原著事实、hidden-adjacent、正式 lore 结论。
- backend/BFF/service、外部框架 PoC/dependency/subagents/read-only scan/patch artifact。
- public/legal/EdgeOne/main auto-merge。
