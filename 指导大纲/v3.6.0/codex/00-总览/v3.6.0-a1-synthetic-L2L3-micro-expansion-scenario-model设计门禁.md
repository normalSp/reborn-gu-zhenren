# v3.6.0-a1 synthetic L2/L3 micro-expansion scenario model 设计门禁

日期：2026-05-25
状态：completed
冻结版本：`v360_a1_synthetic_l2_l3_micro_expansion_model_v1`

## 模型目标

在 v3.5 lifecycle v2 外增加多个 synthetic/generic L2/L3 lane，让同开局下的候选表达更丰富，同时不让 agent 拥有事实、数值、存档、地点、阵营或 NPC 命运裁决权。

## Lane 规则

| 类型 | 允许 | 禁止 |
|---|---|---|
| synthetic L2 | 路人、短工、补给摊、旁观者等泛化低阶意图候选 | 命名 NPC、真实势力、正式任务、奖励 |
| synthetic L3 | 当前场景压力源、盘问、临场小队谨慎等泛化场面候选 | L4/L5、天道/宿命、原著关键人物 |
| rejected lane | 只显示安全阻断原因 | 显示被拒正文或让玩家误解为事实 |
| expired lane | 只提示重新取样 | 保留旧候选为存档事实 |
| needs_user_decision lane | 显示 future gate 原因 | 自动开放 future gate |
