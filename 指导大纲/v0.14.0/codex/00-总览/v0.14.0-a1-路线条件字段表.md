# v0.14.0-a1 路线条件字段表

日期：2026-05-17
状态：草案

## 目标

定义 v0.14 route condition preview 的字段口径。a1 不写 runtime，这张表用于后续 a2/b2 实现。

## Route Condition Preview

```ts
type RouteEligibility = 'blocked' | 'needs_preparation' | 'candidate' | 'ready';

interface RouteConditionPreview {
  routeKey: string;
  displayName: string;
  eligibility: RouteEligibility;
  reason: string;
  missingConditions: RouteMissingCondition[];
  riskFactors: RouteRiskFactor[];
  socialModifiers: RouteSocialModifier[];
  availablePreparations: RoutePreparationAction[];
  sourceRefs: RouteSourceRef[];
  boundaries: RouteBoundary[];
}
```

## 字段说明

| 字段 | 含义 | 来源 | 可见性 |
|---|---|---|---|
| `routeKey` | 路线 id | 本地 canon/rules | 玩家可见可翻译 |
| `displayName` | 路线名 | 本地 canon/rules | 玩家可见 |
| `eligibility` | 当前资格 | 本地 engine | 玩家可见 |
| `reason` | 简短解释 | 本地 engine，可交给 DeepSeek润色 | 玩家可见 |
| `missingConditions` | 缺什么 | 本地 engine | 玩家可见 |
| `riskFactors` | 风险 | route/supply/pursuit、factionPressure | 玩家可见摘要 |
| `socialModifiers` | NPC/势力影响 | v0.13 projection | 玩家可见摘要 |
| `availablePreparations` | 可做前置 | 本地 action protocol | 玩家可见 |
| `sourceRefs` | 来源引用 | fact/anchor/action refs | 开发/测试可见 |
| `boundaries` | 权限边界 | 本地 rules | 玩家可见部分 |

## Missing Condition

```ts
interface RouteMissingCondition {
  id: string;
  type: 'supply' | 'social_cover' | 'public_reason' | 'faction_pressure' | 'route_knowledge' | 'risk_acceptance';
  label: string;
  severity: 'soft' | 'hard';
  canPrepareNow: boolean;
  preparationActionId?: string;
}
```

## Risk Factor

```ts
interface RouteRiskFactor {
  id: string;
  axis: 'pursuit' | 'resource' | 'hidden_fact_probe' | 'faction_pressure' | 'local_survival' | 'canon_anchor_pressure';
  label: string;
  level: 'low' | 'medium' | 'high' | 'blocked';
  visibleSummary: string;
}
```

## Social Modifier

```ts
interface RouteSocialModifier {
  id: string;
  subjectRef: string;
  modifierType: 'opportunity' | 'suspicion' | 'cover' | 'obstruction' | 'rumor';
  visibleSummary: string;
  sourceActionIds: string[];
}
```

## Preparation Action

```ts
interface RoutePreparationAction {
  id: string;
  label: string;
  actionType: 'cover_tracks' | 'send_message' | 'avoid_attention' | 'gather_supply' | 'ask_public_evidence';
  rewardPolicy: 'none';
  canChangeLocation: false;
  canChangeFaction: false;
}
```

## Boundary

```ts
interface RouteBoundary {
  id: string;
  type: 'no_location_unlock' | 'no_faction_transfer' | 'no_reward' | 'no_npc_life_result' | 'hidden_ref_only' | 'deepseek_no_authority';
  visibleText: string;
}
```

## 写入策略

a1 默认只定义字段。

a2/b1 可写 existing v22 字段：

- `playerGoals`
- `knownFacts`
- `npcMemories`
- `factionPressure`
- `actionConsequences`
- `worldClock.lastActionId`
- local action ledger

新增持久 route state 需要用户决策。
