# v0.14.0-a1 测试矩阵

日期：2026-05-17
状态：第一版

## 目标

为 v0.14 路线承接建立工程化测试标准。测试必须证明：路线承接不会变成地点瞬移、阵营转移、奖励发放或隐藏事实泄露。

## 硬指标

| 指标 | 目标 |
|---|---:|
| 隐藏事实泄露 | 0 |
| UI 私算地点/阵营/奖励/NPC 生死 | 0 |
| DeepSeek route/location/faction authority | 0 |
| 未经用户决策的 save format bump | 0 |
| route condition preview 缺 source refs | 0 |
| Player Advocate 高风险小版本轮次 | >= 20 |
| v0.14 rc Player Advocate 轮次 | >= 60 |

## 对话样本三分流记录

依据 `指导大纲/流程制度/测试矩阵演进规则.md`，v0.14 起聊天中出现的新极端玩家意图必须三分流。

| sampleId | 来源 | 样本 | 分流 | 原因 | 落点/后续 |
|---|---|---|---|---|---|
| R14-GOV-001 | 用户讨论 | 一转刚开窍、非天外之魔，想直接获得盗天魔尊传承 | `future_sample_pool` | 这是尊者级/高阶传承自由意图，不属于 v0.14 路线承接 runtime 第一刀；但它能检验高阶目标是否被降级为长期目标、前置行动和风险提示 | 后续自由目标/高阶传承样本矩阵；若 v0.14 触碰自由目标显示，则仅做文案/目标降级走查，不开放传承 |
| R14-MF-001 | MiroFish intake | `qingmao_exit_route_aftermath_pack` 中 `v014exit_30146f740a69` 涉及方源内心动机与“永生”追求 | `current_matrix` | 当前 a2 正在吸收 MiroFish 路线材料，必须防止隔离项进入玩家可见规则 | `src/engine/v014-qingmao-route-continuation.test.ts` 检查 `quarantinedItemIds`，a2 文档记录不得进入 UI/DeepSeek/route pressure |
| R14-INTENT-NORMAL-001 | 专家团复盘 | 玩家想查看离开青茅山的路线 | `current_matrix` | 这是 v0.14 路线承接基本体验 | a2 route condition preview 单测与 Player Advocate 走查 |
| R14-INTENT-SHORT-001 | 专家团复盘 | 玩家想先遮掩行踪或避开注意 | `current_matrix` | 这是 b1 前置行动样板的输入形态 | a2 intent sample，b1 继续落地 |
| R14-INTENT-LONG-001 | 专家团复盘 | 玩家想加入商队去南疆闯荡 | `current_matrix` | 这是路线承接长期目标形态，必须防止直接进城/加入成功 | a2 route condition preview 单测 |
| R14-INTENT-FACTION-001 | 专家团复盘 | 玩家想投靠白家 | `current_matrix` | 当前触碰阵营目标展示，必须防止阵营转移 | a2 route condition preview 单测 |
| R14-INTENT-EXTREME-001 | 专家团复盘 | 一转刚开窍、非天外之魔，想立刻拿到盗天魔尊传承 | `future_sample_pool` | 和 R14-GOV-001 同类，高阶传承不属于 v0.14 runtime | 后续自由目标/高阶传承样本矩阵；a2 仅输出 future_sample_pool hint |
