# v0.14.0-a1 测试矩阵

日期：2026-05-17

## 目标

为 v0.14 路线承接建立工程化测试标准。测试必须证明：路线承接不会变成地点瞬移、阵营转移、奖励发放或隐藏事实泄露。

| 指标 | 目标 |
|---|---:|
| 隐藏事实泄露 | 0 |
| UI 私算地点/阵营/奖励/NPC 生死 | 0 |
| DeepSeek route/location/faction authority | 0 |
| 未经用户决策的 save format bump | 0 |

| Player Advocate 高风险小版本轮次 | >= 20 |
