## 定位

`v0.14.0` 承接 `v0.12.0 青茅山正史锚点与 IF 框架` 与 `v0.13.0 NPC 与势力反应系统`，目标是让玩家不再被青茅山单一区域卡死，而是能带着前期账本进入南疆后续人生。

v0.14 的关键不是“开完整南疆大地图”，而是把已有地基接起来：

`玩家目标 -> 路线条件 -> 前置行动 -> 青茅后果 -> 路线承接状态 -> 后续区域入口`

## 主线

v0.14 主线冻结为：

`青茅后续路线承接：把玩家从青茅山带向南疆人生`

重点交付：

- 路线承接设计门禁。
- 路线条件表与账本读取规则。
- 从一个 `局势后续候选` 升级为正式前置行动样板。
- 青茅离开路线第一刀。
- 阵营目标前置条件展示。
- 路线承接 UI 和 Player Advocate 修正。

## 非目标

除非用户另行批准，v0.14 不做：

- 完整南疆开放世界。
- 完整商家城系统。
- 正式白家投靠、古月叛逃、阵营身份变化。
- 正式声望数值、通缉生效、招揽成功、任务网络和任务奖励。
- NPC 生死、抓捕、追杀成功或关键正史结局。
- 新增蛊虫、蛊方、材料池、稳定资源刷取。
- DeepSeek 获得地点、阵营、奖励、NPC 生死或正史结果写入权。
- 大规模后端化、EdgeOne 自动部署、branch protection。

## Player Advocate 轮次

v0.14 起执行升级规则：

- 纯文档/CI/Git/internal 脚本阶段可豁免，但必须写明。
- 普通玩家可见小版本：10 轮。
- 涉及正式路线承接、地点入口、阵营前置、隐藏事实或旧档迁移的小版本：20 轮。
- 普通 rc：50 轮。
- v0.14 rc：60 轮。

## MiroFish 总需求

v0.14 需要先准备三类资料，但不等于一开始全部 blocking：

1. `qingmao_exit_route_aftermath_pack`：青茅离开后的公开后果、追击、遮掩、三寨残余压力。
2. `southern_border_low_rank_route_pack`：商队、散修、山路、坊市、初级落脚点候选。
3. `shang_clan_city_public_entry_pack`：商家城公开入口候选，暂不开放完整城市系统。

当前 Codex 线程不能直接联系 MiroFish 会话；需要用户转交请求给会话 `019e207b-c55d-7e23-b450-efa7a054a165`。
