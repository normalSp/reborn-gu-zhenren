## 交付要求

每个包必须：

- JSON 为主。
- 不包含原文正文。
- 不包含 quote、originalText、excerpt、verbatim 等字段。
- 使用 source pointer 而不是原文引文。
- 标记 item 类型、可见性、可信度、适用阶段。
- 标记 hidden fact 只为 hidden_ref，不给玩家可见正文。

## 建议字段

```json
{
  "id": "route_item_id",
  "category": "route_aftereffect | route_candidate | supply_gap | pursuit_pressure | public_entry | faction_pressure | hidden_ref",
  "summary": "RebornG-owned short summary",
  "visibility": "public | player_visible | hidden_ref_only",
  "sourcePointers": ["chapter_or_sample_pointer"],
  "suggestedUse": ["candidate_pool", "rule_draft", "test_sample"],
  "blockedUse": ["runtime_truth", "deepseek_authority", "player_hidden_fact_body"],
  "riskTags": ["faction_pressure", "route_escape", "canon_anchor_pressure"]
}
```
