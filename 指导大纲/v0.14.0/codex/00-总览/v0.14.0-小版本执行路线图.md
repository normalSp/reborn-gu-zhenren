## b1 候选后续到正式前置行动桥

目标：

- 从 v0.13 `social followups` 中选一个低风险候选，升级为正式前置行动。

优先候选：

1. 遮掩逃离痕迹。
2. 递话商队。
3. 避开古月视线。

默认推荐：`遮掩逃离痕迹`，因为它不会直接开放阵营、奖励、地点或 NPC 生死。

Player Advocate：20 轮。

## b2 青茅离开路线第一刀

目标：

- 将路线准备推进到路线承接状态。
- 允许玩家看到可走路线、风险、缺口和下一步入口。

边界：

- 不开放完整南疆地图。
- 不瞬移商家城。
- 不结算追杀/抓捕成功。
- 不改变阵营身份。

Player Advocate：20 轮。

## b3 阵营目标前置条件展示

目标：

- 玩家输入“投靠白家”“加入商队”“离开古月”等目标时，显示长期条件、风险、当前小步行动。

输出：

- 当前不能做的原因。
- 能做的低风险前置。
- 未来可能需要的条件。
- 明确“不开放 faction_transfer”。

Player Advocate：20 轮。

## b4 路线承接 UI 与摘要优先级

目标：

- 让路线承接状态在 UI 中可读。
- 移动端默认只显示关键摘要，其余折叠。

验收：

- 玩家能看懂自己为什么能走/不能走。
- 玩家能看懂哪个账本影响了路线。
- 移动端不遮挡、不淹没关键按钮。

Player Advocate：20 轮。
