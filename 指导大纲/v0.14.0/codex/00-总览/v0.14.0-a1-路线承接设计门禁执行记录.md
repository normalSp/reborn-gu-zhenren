## a2 默认实现边界

允许进入 `v0.14.0-a2` 的范围：

- 建立 RebornG-owned route-continuation rules。
- 建立只读 route condition preview engine。
- 从 `playerGoals`、`knownFacts`、`hiddenFactRefs`、`factionPressure`、`npcMemories`、`actionConsequences`、`worldClock.lastActionId` 读取条件。
- 输出 `eligibility`、`missingConditions`、`riskFactors`、`socialModifiers`、`availablePreparations`、`sourceRefs`、`boundaries`。
- 加入正常、短期、长期、极端玩家意图样本。

不允许进入 `v0.14.0-a2` 的范围：

- 不写 `route_entered`。
- 不改变地点、阵营、奖励、NPC 生死或正史锚点。
- 不新增持久字段，不 bump `SAVE_FORMAT_VERSION`。
- 不让 DeepSeek 决定路线、地点、阵营、奖励或隐藏事实。
- 不吸收 MiroFish 隔离项为玩家可见规则。
