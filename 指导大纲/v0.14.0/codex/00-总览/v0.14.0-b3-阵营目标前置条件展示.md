# v0.14.0-b3 阵营目标前置条件展示

日期：2026-05-17
状态：本地质量门已通过，待提交、推送与远端 CI
MiroFish：not_needed，现有 v0.14 三包与 v0.13 势力压力包足够支撑只读前置展示
Player Advocate：20 轮

## 目标

让玩家输入或记录类似：

- `我要投靠白家`
- `我要加入商队`
- `我要去商家城`
- `我要脱离古月/成为散修`

时，能在自由目标面板看到：

- 当前身份。
- 为什么这是长期目标或跨阵营目标。
- 还缺哪些公开前置。
- 有哪些风险。
- 当前只能做哪些低风险小步。

本阶段重点是“解释前置”，不是“开放阵营系统”。

## 非目标

本阶段不做：

- `faction_transfer`
- `standing_delta`
- `warrant`
- `formal_recruitment`
- `formal_task`
- `task_reward`
- `route_entered`
- `location_unlock`
- 正式加入白家、商队或商家城。
- 正式散修系统。
- NPC 生死、抓捕、追杀成功/失败。
- 隐藏事实正文展示。
- DeepSeek 新权限。
- 新存档字段或 `SAVE_FORMAT_VERSION` bump。

## 设计口径

`buildQingmaoFactionGoalPrerequisites()` 是纯函数，只读取：

- `livingWorldState`
- 玩家输入文本。
- `selectedStartProfileId`
- 当前 `playerFactionId`
- 既有 `qingmao-route-continuation-rules.json`
- 既有 `qingmao-region-board.json`

它输出 `read_only_prerequisite_preview`，不会写 store。

## 输出字段

每张前置卡包含：

| 字段 | 说明 |
|---|---|
| `routeKey` | 对应路线/身份入口，如 `baijia_contact_window` |
| `disposition` | 当前目标性质，如 `cross_faction_prerequisites` |
| `currentFactionId` | 当前阵营只读展示 |
| `currentIdentitySummary` | 当前身份摘要 |
| `publicSummary` | 玩家可见解释 |
| `prerequisiteLines` | 缺少的硬/软前置 |
| `riskLines` | 势力、追踪、身份、城市延期等风险 |
| `allowedNextSteps` | 当前只允许的小步 |
| `blockedUpgrades` | 明确禁止升级 |
| `canPatch` | 恒为 `false` |
| `canChangeFaction` | 恒为 `false` |
| `createsFormalTask` | 恒为 `false` |
| `grantsReward` | 恒为 `false` |

## UI 表现

`FreeGoalPanel` 新增：

- `data-testid="free-goal-faction-prerequisites"`
- 标题：`阵营/身份目标前置`
- 关键提示：`只显示前置，不转阵营`

玩家输入 `我要投靠白家` 后，面板展示：

- `白家接触前置`
- `跨阵营前置`
- 当前身份，如 `古月山寨 · guyue_clan_member`
- 前置条件、风险、可做小步
- `禁止 faction_transfer`

玩家输入 `我要加入商队去商家城` 后，面板展示：

- `商队接触前置`
- `商家城公开入口前置`
- `不直接加入商队`
- `不开放完整城市系统`

## MiroFish 判定

本阶段 MiroFish 需求为 `not_needed`。

原因：

- 当前只做 RebornG-owned 前置展示，不新增原著事实。
- v0.14 三包已提供路线、商队、商家城公开入口候选材料。
- v0.13 势力压力包已支撑“白家接触窗口/古月疑心/信任缺口”的候选推演。
- 如果后续要做正式投靠、通缉、招揽、任务网络、命名 NPC 反应或新地域进入，再重新向用户请求 MiroFish 包。

## 边界审查

| 风险 | 本阶段处理 |
|---|---|
| 玩家误以为可以一键投靠白家 | UI 文案使用“只显示前置，不转阵营”，并禁止 `faction_transfer` |
| 玩家误以为加入商队成功 | UI 文案写明“商队目标先走公开理由、递话窗口、补给和身份盘问，不直接加入商队” |
| 玩家误以为进入商家城 | `shang_public_entry_deferred` 仍是延期入口，不开放完整城市系统 |
| UI 私算声望/任务/奖励 | 引擎输出 `canPatch=false`、`createsFormalTask=false`、`grantsReward=false` |
| 隐藏事实泄露 | 只使用 route rules 与公开身份，不输出隐藏事实正文 |
| DeepSeek 越权 | 不改 prompt/schema/model/context-builder |
