# v0.14.0-a2 路线条件读取第一刀

日期：2026-05-17
状态：已实现，待验证收束

## 目标

把 v0.14 的路线承接规则落成只读 runtime helper：

`livingWorldState + 玩家意图 -> route condition preview`

本阶段只展示路线条件、缺口、风险、社会影响和可做前置，不写状态，不进入地点，不改阵营，不给奖励。

## 新增文件

- `src/canon/qingmao-route-continuation-rules.json`
- `src/engine/v014-qingmao-route-continuation.ts`
- `src/engine/v014-qingmao-route-continuation.test.ts`

## Route Archetype

第一刀支持 5 类路线承接预览：

| routeKey | 口径 |
|---|---|
| `mountain_pass_escape` | 山路逃离路线，只做条件和风险预览 |
| `merchant_caravan_contact` | 商队接触路线，只做递话、公开理由和补给前置 |
| `baijia_contact_window` | 白家接触窗口，只做试探，不投靠成功 |
| `rogue_cultivator_path` | 散修过渡路线，只做低阶生存和身份遮掩缺口 |
| `shang_public_entry_deferred` | 商家城公开入口候选，只做远期入口条件评估 |

## 输出字段

`buildQingmaoRouteContinuationPreview()` 输出：

- `status = read_only_preview`
- `previews[]`
- `eligibility`
- `missingConditions`
- `riskFactors`
- `socialModifiers`
- `availablePreparations`
- `sourceRefs`
- `boundaries`
- `intentRulingHints`
- `intentSamples`
- `forbiddenWrites`
- `quarantinedItemIds`
- `statePatchApplied = false`
