# v0.19.0-b2 长测 runner 与 Player Advocate 大走查第一刀

日期：2026-05-19
状态：已完成
阶段性质：deterministic walkthrough + matrix gate

## 目标

把 v0.19 的代表性路径从“建议测试”变成可审计走查：

- 覆盖 8 条 playthrough anchor。
- 覆盖 route/location、hidden fact、reward、faction、NPC life、release copy 等越权边界。
- 不调用 live DeepSeek，不产生 API 成本。
- 不新增存档、不新增 gameplay runtime。

## 本轮策略

本轮不是自动 AI 玩家完整游玩，而是 deterministic Player Advocate walkthrough：

1. 以 `buildV019LongPlaythroughMatrix()` 的 8 条路径为覆盖源。
2. 以玩家视角记录“目标 / 上下文 / 输入 / 反馈 / 是否理解”。
3. 使用 `scripts/check-player-advocate-gate.mjs` 校验记录不是空模板。
4. 发现的 P2/P3 进入后续池，不直接改 gameplay。
