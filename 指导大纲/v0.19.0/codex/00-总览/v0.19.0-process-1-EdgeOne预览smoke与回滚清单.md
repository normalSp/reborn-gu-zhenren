# v0.19.0-process-1 EdgeOne 预览 smoke 与回滚清单

日期：2026-05-19
状态：已完成
阶段性质：流程清单，不自动部署

## 原则

本专项只做预览 smoke、证据记录和回滚清单，不做 EdgeOne 自动部署，不改 CodeBuddy/EdgeOne 集成。

## 发布前 smoke

发布前至少确认：

1. `npm run build` 通过。
2. `npm run check:production-preview` 通过。
3. `npm run check:player-visible-copy` 通过。
4. `npm run check:runtime-assets` 通过。
5. `npm run check:v019-content-governance` 通过。
6. Git 当前 commit 和 GitHub Actions run 已记录。
7. `指导大纲/项目仪表盘.md` 已更新。
8. handoff 已记录 branch / commit / push / CI。

## 发布后 smoke

用户或 CodeBuddy 部署 EdgeOne 后，人工检查：

1. 首页不黑屏。
2. console 无启动期 `Cannot read properties of undefined`。
3. 标题页可见版本。
4. 进入游戏后底栏六入口存在。
5. 行动/自由目标/蛊道/世界/记录关键入口可打开。
6. 公开文案没有“完整南疆/完整商家城/仙蛊奖励/宝黄天交易”等越权承诺。

## 黑屏排查顺序

1. 看浏览器 console 第一条 runtime error。
2. 对照本地 `npm run check:production-preview`。
3. 对照最近 CI run。
4. 检查是否生产拆包循环依赖。
5. 检查是否 module 顶层同步调用 engine 默认状态。
6. 检查 runtime assets 0 字节。
7. 回滚到最近通过的 commit。

## 回滚清单

| 项 | 需要记录 |
|---|---|
| 本地 commit | 当前 phase commit hash |
| 远端 branch | 当前 branch |
| GitHub Actions run | deterministic quality gate run id |
| EdgeOne build | 用户侧/CodeBuddy 构建记录 |
| 回滚目标 | 最近通过 production-preview smoke 的 commit |
| 数据风险 | v0.19 不新增 save field，回滚不涉及 v23 migration |

## 当前 v0.19 结论

- 本阶段不自动部署。
- 本阶段不新增 EdgeOne 配置。
- 本阶段不新增 BFF/backend。
- 本阶段只把 smoke 和回滚证据纳入 v1.0 readiness。
