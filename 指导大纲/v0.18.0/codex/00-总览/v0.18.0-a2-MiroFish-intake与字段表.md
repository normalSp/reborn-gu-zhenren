## RebornG 本地候选字段

### RouteEntryCandidate

| 字段 | 说明 |
|---|---|
| `id` | RebornG 本地 id |
| `sourcePackageId` | MiroFish 包或本地规则来源 |
| `sourceItemIds` | 来源 item ids |
| `routeId` | 本地路线 id |
| `stage` | 候选阶段 |
| `publicMilestone` | 玩家可读公开里程碑 |
| `prerequisites` | 本地前置条件 |
| `blockedReasons` | 阻断原因 |
| `riskHints` | 补给、追索、身份风险 |
| `allowedWrite` | 第一刀默认 `existing_v22_only` |

### RegionFactDraft

| 字段 | 说明 |
|---|---|
| `id` | RebornG 本地事实卡 id |
| `sourcePackageId` | 来源包 |
| `sourceItemIds` | 来源 item ids |
| `regionRef` | 区域/外缘引用 |
| `visibility` | `public/player_visible/hidden_ref_only` |
| `summary` | RebornG 重写后的公开摘要 |
| `hiddenRefs` | 仅本地引用，不能入 DeepSeek |
| `allowedPromptUse` | 仅公开摘要可 true |
| `blockedImplications` | 不得暗示完整地图/完整城市/高阶路线 |

### RoutePressureDraft

| 字段 | 说明 |
|---|---|
| `id` | RebornG 本地压力 id |
| `sourcePackageId` | 来源包 |
| `sourceItemIds` | 来源 item ids |
| `pressureAxis` | `supply/pursuit/identity/faction_residual/caravan_guarantee/npc_memory` |
| `publicReason` | 玩家可读公开理由 |
| `severity` | `low/medium/high/blocked` |
| `ledgerInputs` | 读取的 existing v22 ledger |
| `blockedEscalations` | 通缉、招揽、阵营、NPC 生死等阻断 |
