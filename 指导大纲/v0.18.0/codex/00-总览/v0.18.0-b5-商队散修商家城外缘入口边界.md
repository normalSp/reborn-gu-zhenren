# v0.18.0-b5 商队、散修、商家城外缘入口边界

日期：2026-05-19
状态：已完成
性质：runtime read-only projection

## 目标

把玩家常见的后青茅路线愿望拆成三类公开入口边界：

1. 商队同行/递话外缘。
2. 散修/独行落脚外缘。
3. 商家城外缘公共入口。

## 实现

`buildV018QingmaoRouteMultiRegionOverview()` 输出 `entryBoundaries`，UI 展示在 `free-goal-v018-entry-boundaries`。

每个入口只显示：

- 公开摘要。
- 前置条件。
- 可做小步。
- 禁止升级项。

## 边界

- 商队：不加入商队，不正式雇佣，不正式交易。
- 散修：不洗白身份，不改阵营，不写正式通缉或 NPC 结果。
- 商家城外缘：不开放完整商家城、拍卖、演武场、内城、任务网络。

## 验收

- e2e 验证三类入口卡可见。
- e2e 验证 UI 不出现 `加入商队成功`、`进入商家城`、`奖励已发放`。
- Player Advocate 20 轮记录：`v0.18.0-b5-Player-Advocate-20轮走查记录.md`。

## 结论

b5 通过。玩家能看到后青茅人生路线的入口形状，但不会被误导成正式开放完整城市或阵营系统。
