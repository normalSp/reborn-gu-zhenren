日期：2026-05-19
状态：本地开发里程碑完成；等待提交、推送与远端 CI 记录
主题：`南疆路线与多区域承接`

## 定位

`v0.18.0` 承接 `v0.14` 的青茅路线候选、`v0.15` 的低阶经济/补给、`v0.16` 的 UI 减法与权威归并、`v0.17` 的战斗深化。它的目标不是直接开放完整南疆大地图，而是让玩家从青茅山之后的人生路线开始有正式承接：路线状态、低阶区域事实、前期账本影响、补给/追索/身份压力、商队或散修等低阶入口。

推荐玩家心智：

`行动 -> 路线门槛/离开承诺 -> 本地路线裁决 -> 路线账本/区域状态 -> 后续行动与叙事回流`

## 明确非目标

除非用户另行批准，v0.18 不做：

- 完整南疆大地图。
- 完整商家城系统、内城/演武场/拍卖/商会正式经济网络。
- 正式阵营转移、通缉、招揽、任务网络或长期声望系统。
- NPC 生死、捕获、永久追杀成败。
- 正式路线奖励、地点奖励、蛊虫/材料奖励。
- 蛊仙期、仙蛊、仙材、福地/洞天、宝黄天、尊者线索。
- DeepSeek 直接决定路线进入、地点解锁、阵营、奖励或 NPC 命运。
- 后端/BFF、云存档、自动部署；只做设计评估，除非用户单独批准。

## 完成结论

`v0.18.0` 已完成为本地开发里程碑：玩家可以看到青茅离开门槛、候选路线承接、南疆低阶公开事实、前期账本压力回流，以及商队/散修/商家城外缘入口边界。所有 runtime 写入都限制在 existing v22 字段和本地 action ledger 内；未新增 `route_entered`、新地点字段、`SAVE_FORMAT_VERSION = 23`、正式地点/阵营/奖励/NPC 生死、BFF/backend 或 DeepSeek 权限。
