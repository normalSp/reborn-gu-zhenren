## 必须停机

- 新增 save field、migration、`SAVE_FORMAT_VERSION` bump 或 `runFingerprint`。
- 新增 persistent agent memory、agent state、store slice 或跨回合 agent 私有状态。
- 调用 live DeepSeek，或修改 DeepSeek prompt/context/model/authority。
- 新增 DeepSeek visible lore/RAG。
- MiroFish export/intake，或触发命名 NPC、真实原著势力/地点、hidden-adjacent、方源公开证据、正式 lore 结论。
- 新增 backend/BFF/service/job queue/eval archive/cloud save。
- 引入 external framework PoC/dependency/read-only scan/subagent/patch artifact。
- 开放 L4/L5 runtime、天道/宿命 runtime、原著关键人物 runtime agent。
- 写正式地点、阵营、身份、奖励、NPC 生死、通缉、招揽、封锁。
- 让 self-learning 直接改 canon/runtime/save/prompt/knowledge-index。
- Player Advocate 或 drift 出现 P0/P1。
- CI/测试失败且需要改变方案。
- public wording、release、EdgeOne、法律/版权边界或成本边界变化。

## 不构成停机

只要仍在 D-350 范围内，以下事项可以继续：

- 补充文档、测试矩阵、PA 记录和 drift 记录。
- 硬化 transient proposal lifecycle。
- 改善 candidate/rejected/expired 文案。

- 增加 no-save-write / rollback / old-save 证据。
