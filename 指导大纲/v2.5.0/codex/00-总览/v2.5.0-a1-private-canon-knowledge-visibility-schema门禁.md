## Visibility Levels

草案名：`v250_a1_private_canon_visibility_schema_v1`

| Level | 含义 | v2.5 允许 | 禁止 |
|---|---|---|---|
| `public_visible` | 玩家已知或公开世界压力 | level 名、说明、未来安全摘要占位 | 自动生成正式地点/阵营/奖励/NPC 生死 |
| `player_visible_safe_summary` | 可被玩家或 DeepSeek 看到的安全摘要类别 | 只定义类别和要求 | 当前不写真实摘要正文 |
| `source_pointer_only` | 只可保留来源指针，不展开内容 | source pointer class、id-only 规则 | 原文、邻近段落、hidden 解释 |
| `agent_private_candidate` | Agent/NPC 私有记忆候选 | synthetic marker 与候选状态 | 写入 save、canon、正式 NPC 记忆 |
| `hidden_marker_synthetic` | 合成 hidden leak 测试标记 | `SYNTH_*` 标记 | 解释为任何原著真实事实 |
| `hidden_body_locked` | 真实隐藏正文锁定态 | 只可作为禁区名称 | UI、DeepSeek、报告正文、artifact、知识库正文 |
| `formal_decision_locked` | 正式裁决等待用户决策 | 只可作为禁止项 | 自动 canon promotion 或 runtime 写入 |
| `deepseek_visible_denied` | 明确不得进入 DeepSeek visible context | 拒绝原因、审计状态 | prompt/context 注入 |

## Knowledge Item Lifecycle

v2.5 只定义生命周期，不创建知识库正文。

```text
archive_source_pointer
  -> index_candidate_stub
  -> rule_or_test_candidate
  -> future_human_review_gate
  -> future_canon_gate
  -> future_runtime_canon_subset
```

每一层的权威：

| Stage | 权威 | v2.5 状态 |
|---|---|---|
| `archive_source_pointer` | MiroFish / 原始档案层只提供指针 | 只读概念 |
| `index_candidate_stub` | RebornG 知识索引候选壳 | 只定义字段，不写正文 |
| `rule_or_test_candidate` | 规则草案或测试样本 | 可作为未来样本池说明 |
| `future_human_review_gate` | 用户/专家团复核 | future gate |
| `future_canon_gate` | 当前版本单独批准 | future gate |
| `future_runtime_canon_subset` | `src/canon/*.json` | v2.5 不触碰 |

## DeepSeek Visible Denial Rules

DeepSeek visible context 在 v2.5 不新增任何内容。未来若申请 DeepSeek 可见知识，必须另走 F-250-003。

当前拒绝规则：

- `hidden_body_locked` 永远拒绝。
- `agent_private_candidate` 默认拒绝，除非未来被改写成玩家可见安全摘要并另批。
- `source_pointer_only` 只可作为本地审计指针，不进入 prompt。
- `formal_decision_locked` 永远拒绝，直到用户单独批准正式裁决。
- `player_visible_safe_summary` 当前只定义类别，不写正文，因此也不进入 prompt。

## WorldCore Post-Check 草案

未来 private canon / AgentProposal / eval archive 输出必须由 WorldCore 或等价本地 post-check 拦截：

P0 blocking：

- hidden/private body 出现在 UI、DeepSeek visible、报告正文、artifact、save、runtime canon、外部框架输入或 patch。
- source pointer 被展开成正文、邻近段落或隐藏解释。
- AgentProposal 把候选记忆写成正式 NPC 事实、态度、身份或剧情结论。
- L5 宏观压力宣判结局、宿命结果、NPC 生死或 canon outcome。
- private canon service / backend / BFF 自称拥有最终裁决权。

P1 manual review：

- 缺少 visibility level。
- 缺少 deniedConsumers。
- 使用疑似正式凭证、身份、地点、奖励、阵营归属词，但未宣称为事实。
- source pointer class 与真实 fact id 命名混淆。

P2 warning：

- 报告没有写清 MiroFish `not_needed` 原因。
- schema 没有记录 future_gate_required 决策号。
- 没有明确 live DeepSeek / Player Advocate 豁免理由。
