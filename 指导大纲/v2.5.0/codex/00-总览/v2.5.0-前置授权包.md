## 例外停机清单

触发以下任一条件，立即停止 `/goal` 自动推进：

- 需要任何 F-250 事项。
- 文档阶段发现 v2.5 设计必须改成 runtime/backend/prompt/save 实现才能成立。
- 文档扫描发现授权包本身存在越权措辞且无法局部修正。
- CI/测试连续失败且需要改变方案。
- 用户中途提出新方向或更高优先级。
- 专家团发现 private canon / visibility 方案会损害原著边界、游戏性或未来 v3.0 agent 架构。
