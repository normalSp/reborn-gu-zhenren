v2.5-b2 冻结 `v250_b2_hidden_leak_memory_contamination_agentproposal_risk_model_v1`：AgentProposal、NPC 记忆候选、private canon 分类和 self-learning 候选都必须先通过 hidden leak / memory contamination / formal authority drift 风险模型；它们只能生成候选，不能自裁决。

## Severity Rubric

### P0 blocking

- hidden/private body 或其可复原解释出现在 UI、DeepSeek visible、report body、artifact、save、runtime canon、知识库正文、外部框架输入或 patch。
- AgentProposal 写入正式地点、阵营、身份、奖励、NPC 生死、canon promotion。
- `agent_private_candidate` 被当成玩家已知事实或正式 NPC 记忆。
- L5/Heaven's Will/Fate 输出结局、宿命结果、原著关键人物命运或 canon outcome。
- source pointer 展开成原文、邻近段落或隐藏解释。
- self-learning 直接改 canon/runtime/save/prompt/knowledge-index body。

### P1 manual review / blocking

- 缺少 visibility level 或 allowed/denied consumer。
- AgentProposal 使用疑似正式凭证、令牌、成员身份、封锁、招揽、通缉、奖励词。
- report 保存了可复原 hidden body 的 hash/source neighborhood。
- framework adapter 或 future service 没有 redaction policy。
- MiroFish need level 未写或触发条件判断不清。

### P2 warning

- synthetic marker 命名与真实 fact id 混淆。
- cost/model/prompt-hash 字段缺失 future placeholder。
- Player Advocate/live DeepSeek 豁免理由不完整。
- 手工复核链路缺少责任人或 next gate。
