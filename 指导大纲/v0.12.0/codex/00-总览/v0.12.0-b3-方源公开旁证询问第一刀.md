## 运行时行为

玩家输入类似：

- “我要打听方源在客栈和族学的公开记录”
- “我要观察方源”
- “我要跟踪方源”

裁决为可尝试调查后，自由目标面板会提供“旁证询问”入口。

点击后会：

- 根据公开询问文本匹配方源公开旁证面。
- 写入 `knownFacts`，只包含玩家角色可以从公开渠道得到的表象。
- 写入 `hiddenFactRefs`，只作为本地保护引用。
- 写入 `npcMemories`，记录有人打听方源公开痕迹。
- 写入 `factionPressure`，记录山寨内部警觉。
- 写入 `actionConsequences`。
- 更新 `worldClock.lastActionId`。
- 更新 `sceneSessionState.localActionLedger`。
- 更新 `flags.lastWorldActionReturnContext`。

## MiroFish 吸收方式

MiroFish 主包没有直接进入 runtime。

吸收方式：

1. intake review 确认包内无 quote/originalText/excerpt/verbatim 和禁词泄露。
2. RebornG 将候选条目改写成本地 public evidence / rule draft。
3. runtime 只读取 RebornG 本地 canon JSON。
4. hidden boundary 只转成 `hiddenFactRefs`，不进入 UI 和 DeepSeek visible context。
5. source item id / source pointer id 只保留为来源指针。
