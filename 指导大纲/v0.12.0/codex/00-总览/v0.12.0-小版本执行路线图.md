## v0.12.0-a2 青茅低阶 IF 矩阵、偏离规则和代价

状态：第一刀已完成，记录见 `v0.12.0-a2-青茅低阶IF矩阵.md`。

目标：

- 定义玩家能偏离哪里、不能偏离哪里、偏离需要什么代价。
- 让自由意图裁决器未来有明确 IF 规则来源。
- 把旧 v0.8 的剧情锚点/IF 思想低阶化到青茅山，而不是直接复用宿命战、尊者博弈或永生级矩阵。

交付：

- 青茅 IF 轴：`npc_attention`、`faction_pressure`、`resource_control`、`route_escape`、`hidden_fact_probe`、`local_survival`、`canon_anchor_pressure`。
- IF 偏离等级：`blocked`、`rumor_only`、`precondition_required`、`minor_deviation`、`major_deviation_candidate`。
- 偏离代价类型：资源、时间、身份风险、势力压力、NPC 记忆、追击风险。
- 测试：重大锚点不得被普通自由输入直接改写。
- 新增 `src/canon/qingmao-low-rank-if-matrix.json` 与 `src/engine/v012-qingmao-if-matrix.ts`。

验收：

- 青茅 IF 轴不包含 `protect_fate`、`break_fate`、`venerable_balance` 等高阶轴。
- 每个轴都有可写字段、不可写字段、DeepSeek 权限和至少 2 条测试样本。
- `major_deviation_candidate` 只能进入候选、压力或准备链，不直接改变正史核心因果。
