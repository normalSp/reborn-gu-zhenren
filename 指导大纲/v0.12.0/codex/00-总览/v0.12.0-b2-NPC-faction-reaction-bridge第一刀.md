## 运行时行为

玩家完成公开行动后，例如：

- 记录 `region:outside_qingmao` 长期目标。
- 执行逃离青茅山路线准备。
- 形成公开 `knownFacts`、`actionConsequences`、`factionPressure` 或 `playerGoals`。

点击自由目标面板中的“局势反应 / 推演”会：

- 根据公开痕迹匹配青茅轻量反应规则。
- 写入 `factionPressure`。
- 写入 `npcMemories`，只保存公开记忆，不保存隐藏事实正文。
- 写入一条 `actionConsequences`。
- 更新 `worldClock.lastActionId`。
- 更新 `sceneSessionState.localActionLedger`。
- 更新 `flags.lastWorldActionReturnContext`。

## 规则覆盖

当前第一刀吸收 12 条候选规则，覆盖：

- 古月内部竞争。
- 族学/炼化考核压力。
- 家老与内务堂安全警惕。
- 客栈、商队、酒类窗口。
- 三寨外部互疑。
- 小组任务权威。
- 资源分配和信任危机。
- 内务堂流程痕迹。

这些规则只产生轻量反应，例如：

- 警惕。
- 观望。
- 试探。
- 盘问。
- 交易窗口。
- 记录痕迹。

## 禁止项

本刀没有做：

- 新增存档字段。
- 提升 `SAVE_FORMAT_VERSION`。
- 正式声望系统。
- 阵营身份变化。
- 白家/熊家投靠。
- NPC 生死、抓捕、追击成败结论。
- 奖励、元石、酒、材料、蛊虫、蛊方。
- 新地点或路线成功解锁。
- 正史锚点修改。
- hidden fact 显示。
- DeepSeek 权限扩张。

## MiroFish 吸收方式

MiroFish 主包没有直接进入 runtime。

吸收方式：

1. intake review 确认包内无 quote/originalText/excerpt/verbatim。
2. RebornG 将候选条目改写成本地 reaction rule draft。
3. runtime 只读取 RebornG 本地 canon JSON。
4. MiroFish source item id / source pointer id 只保留为来源指针。
5. UI 和 DeepSeek 不接收隐藏事实正文或 MiroFish 权限字段。
