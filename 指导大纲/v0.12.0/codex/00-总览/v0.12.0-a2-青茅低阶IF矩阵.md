## IF 等级

当前矩阵固定 5 个等级：

| 等级 | 含义 |
|---|---|
| `blocked` | 直接阻断，只给原因和安全改写方向 |
| `rumor_only` | 只能作为公开传闻或无证据讨论 |
| `precondition_required` | 需要路线、资源、身份、证据或风险条件 |
| `minor_deviation` | 允许低阶局部偏离，但必须由本地引擎结算 |
| `major_deviation_candidate` | 重大 IF 候选，只能进候选池、压力链或长期准备 |

## IF 轴

当前矩阵只使用青茅低阶轴：

- `npc_attention`
- `faction_pressure`
- `resource_control`
- `route_escape`
- `hidden_fact_probe`
- `local_survival`
- `canon_anchor_pressure`

不使用：

- `protect_fate`
- `break_fate`
- `venerable_balance`
- `immortal_trade`
- `rank_nine_acquisition`
- `treasure_yellow_heaven_transaction`

这些高阶轴保留到后期大锚点网络，不能在青茅凡阶开局直接使用。

## 样本规则

第一刀已覆盖这些代表性输入：

| 玩家输入类型 | 当前裁决 |
|---|---|
| “我要拿九转蛊/仙蛊/宝黄天交易/永生” | `blocked` |
| “我要拿大量/无限元石” | `blocked` |
| “我要杀死白凝冰/阻止他自爆” | `blocked` |
| “我要揭露古月一代/天鹤/血道真相” | `blocked` |
| “我要调查灵泉枯竭秘密” | `blocked` |
| “我要直接找到花酒传承位置” | `blocked` |
| “我要跟踪方源” | `precondition_required`，只能公开观察 |
| “我要投靠白家” | `major_deviation_candidate`，只能进入接触窗口/风险链 |
| “我要跟商队逃离青茅山” | `precondition_required`，只能路线准备 |
| “我要侦查山道/临时藏身点” | `precondition_required`，只能路线情报或暴露风险 |
| “我要问花酒行者传闻” | `rumor_only` |
| “我要省元石买月兰花喂月光蛊” | `minor_deviation`，本地经济结算 |
| “我要参加狼潮巡逻/三寨战功” | `minor_deviation`，本地战斗/压力结算 |

## DeepSeek 权限

DeepSeek 仍只能写：

- 叙事。
- 候选。
- 线索。
- 传闻。
- 压力表达。
- 本地裁决结果解释。

DeepSeek 不能写：

- 正式奖励。
- 蛊虫、元石、地点解锁。
- NPC 生死。
- 阵营身份变化。
- 逃离成功。
- 隐藏事实 body。
- 正史锚点改写。
