# v0.13.0-b4 Player Advocate 可读性 UI 第一刀

日期：2026-05-16
状态：第一刀完成
入口决策：用户批准方案 A，放进 `FreeGoalPanel` 作为“社会影响 / 局势后续”折叠区。

## 目标

b4 第一刀只回答：

玩家能不能在自由目标面板里看懂自己已经影响了谁、哪些势力正在形成压力、哪些公开行动已经被记录、哪些后续仍只是候选？

## 本次交付

新增 / 修改：

- `src/components/game/FreeGoalPanel.tsx`
- `tests/e2e/v013-social-impact-panel.spec.ts`

`FreeGoalPanel` 新增“社会影响 / 局势后续”折叠区，按当前 `livingWorldState` 与 `sceneSessionState.localActionLedger` 只读计算：

- `buildQingmaoNpcMemoryProjection()`
- `buildQingmaoFactionStanceProjection()`
- `buildQingmaoPublicEventChronicle()`
- `buildQingmaoSocialFollowups()`

## 玩家可见内容

折叠区展示四类信息：

- `谁记住了你`：NPC / 小组 / 流程视线的候选记忆。
- `势力态度`：古月、白家、商队等势力的压力、机会或风险候选。
- `公开行动摘要`：可公开压缩的行动摘要。
- `局势后续`：解释、递话、遮掩、避险、旁证等候选后续。

## 边界

本刀仍然不做：

- store 写入。
- 新增持久化字段。
- 提升 `SAVE_FORMAT_VERSION`。
- 关系分数、声望数值。
- 正式通缉、招揽、投靠、任务、奖励。
- 地点解锁、阵营变化、NPC 生死。
- hidden fact id/body 展示。
- 方源隐藏因果、春秋蝉、重生/回溯泄露。
- DeepSeek 新写入权。

UI 只消费本地 engine 输出，不私算关系、不私算势力态度、不从 MiroFish 原始包读数据。
