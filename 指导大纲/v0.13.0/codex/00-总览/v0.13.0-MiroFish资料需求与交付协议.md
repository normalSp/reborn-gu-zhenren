日期：2026-05-16

定位：为 v0.13 NPC/势力社会记忆系统提供 quote-redacted 候选材料。

## 结论

v0.13 需要 MiroFish，但 MiroFish 仍不是 canon，不是 runtime 权威，不是 DeepSeek 权限源。

所有交付包必须：

- quote-redacted。
- 不含原文正文。
- 不含 `quote`、`originalText`、`excerpt`、`verbatim` 等字段。
- 标注 source pointers。
- 标注 review status。
- 标注 RebornG gate。
- hidden facts 只能 `hidden_ref_only`。

## 全版本候选包

### 1. qingmao_npc_memory_motive_pack

目标阶段：`v0.13.0-a2 NPC 记忆投影引擎第一刀`
阻塞等级：`preferred`；如果要写命名 NPC runtime 规则，则升级为 `blocking`。

用途：

- NPC/小组/家老/商队人物会记住什么公开行为。
- 什么行为会引发怀疑、信任、试探、利益驱动或避让。
- 同一行为对不同主体的解释差异。

禁止：

- NPC 生死结论。
- 隐藏事实 body。
- 方源隐藏因果、春秋蝉、重生/回溯。
- 正式任务、奖励、地点、阵营变化。

### 2. qingmao_faction_reputation_pressure_pack

目标阶段：`v0.13.0-b1 势力态度 / 压力投影第一刀`
阻塞等级：`preferred`；如果写正式势力态度 rule draft，则升级为 `blocking`。

用途：

- 古月、白家、熊家、商队、族学、内务等对玩家公开行为的压力、机会、封锁、招揽、通缉、任务来源候选。
- 给 v0.13 的 stance / pressure projection 提供候选轴。

禁止：

- 公开声望数值结论。
- 正式通缉生效。
- 正式招揽成功。
- 阵营身份变化。
- 任务奖励。

### 3. qingmao_public_event_chronicle_pack

目标阶段：`v0.13.0-b2 事件编年史与公开行动摘要`
阻塞等级：`preferred`。

用途：

- 青茅公开事件的可见顺序和公共记忆。
- 玩家公开行动如何被记录成摘要。
- 哪些事件适合作为 DeepSeek prompt-safe public summary。

禁止：

- 隐藏事实 body。
- 内心独白式确定性推断。
- 未公开的正史因果。
