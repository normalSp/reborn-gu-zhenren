# v0.13.0-a1 字段表与写入权限

日期：2026-05-16
状态：第一版
定位：约束社会记忆候选如何读取、投影、写入和展示。

## 总原则

第一阶段复用 `SAVE_FORMAT_VERSION = 22` 的既有字段，不新增持久化关系账本。

允许复用：

- `livingWorldState.knownFacts`
- `livingWorldState.hiddenFactRefs`
- `livingWorldState.actionConsequences`
- `livingWorldState.factionPressure`
- `livingWorldState.npcMemories`
- `livingWorldState.localActionLedger`
- `livingWorldState.playerGoals`
- `livingWorldState.lastWorldActionReturnContext`

不在 a1/a2 第一刀新增：

- `relationshipLedger`
- `factionStandingLedger`
- `publicEventChronicle`
- `socialMemoryState`

## MiroFish 候选输入字段

MiroFish 原始 item 只能进入 intake/review 阶段。runtime 只能消费 RebornG 重写后的字段。

| 字段 | 可否吸收 | 说明 |
|---|---|---|
| `requestId` / package id | 可作为来源指针 | 仅用于追溯 |
| `item.id` | 可作为来源指针 | 不作为 runtime 事实 id |
| `category` | 可映射 | 映射到 RebornG 本地候选类型 |
| `summary` / prompt-safe summary | 可改写 | 必须重写为 RebornG 本地摘要 |
| `sourcePointerIds` | 可保留 | 仅作来源追溯，不展示正文 |
| `review.status` | 可保留 | 必须为 `export_ready` 或更高 |
| `reborngGate.runtimeAuthority` | 必须检查 | 只能是 `candidate_only` |
| `reborngGate.runtimeVisible` | 必须检查 | 必须为 `false` |
| `reborngGate.deepSeekVisible` | 必须检查 | 必须为 `false` |
| 原文正文、quote、excerpt | 禁止 | 出现即阻断 |

## 本地候选类型

### SocialMemoryCandidate

用途：a2/a3/b1/b2 的通用候选载体。

建议字段：

| 字段 | 说明 |
|---|---|
| `id` | RebornG 本地候选 id |
| `kind` | `npc_memory` / `faction_pressure` / `public_event_summary` |
| `sourcePackageId` | MiroFish 包或本地规则来源 |
| `sourceItemIds` | 来源 item id 列表 |
| `triggerActionId` | 本地行动或后果 id |
| `subjectRef` | 主体引用，先可匿名或半结构化 |
| `visibility` | `player_visible` / `subject_visible` / `hidden_ref_only` |
| `reason` | 本地重写的公开原因 |
| `riskLevel` | `low` / `medium` / `high` / `blocked` |
| `recommendedFollowUps` | 后续行动候选 id |
| `blockedReasons` | 被门禁阻断的原因 |

### NpcMemoryProjection

用途：a2 read-only engine 输出。

建议字段：

| 字段 | 说明 |
|---|---|
| `subjectRef` | NPC/小组/身份层 |
| `memoryAxis` | `attention` / `suspicion` / `trust` / `debt` / `avoidance` / `interest` |
| `publicReason` | 玩家可读公开原因 |
| `confidence` | `low` / `medium` / `high` |
| `canPatch` | 第一刀默认 `false`，后续受控动作才可写 |
| `patchPreview` | 若未来可写，展示会写什么 |

### FactionPressureProjection

用途：b1 read-only engine 输出。

建议字段：

| 字段 | 说明 |
|---|---|
| `factionRef` | 势力/组织/职能层 |
| `pressureAxis` | `watch` / `opportunity` / `blockade` / `recruitment_hint` / `pursuit_risk` / `task_source_hint` |
| `publicReason` | 公开理由 |
| `severity` | `low` / `medium` / `high` / `blocked` |
| `escalationBlocked` | 是否被正式通缉/招揽/任务门禁挡住 |

### PublicEventSummaryCandidate

用途：b2 公开行动摘要。

建议字段：

| 字段 | 说明 |
|---|---|
| `eventRef` | 本地事件/行动/后果 id |
| `publicSummary` | 可给玩家和 DeepSeek 的公开摘要 |
| `visibleTo` | `player` / `local_group` / `faction` / `region_public` |
| `hiddenRefs` | 仅本地引用，不能进入正文 |
| `promptSafe` | 是否允许进入 DeepSeek prompt |

## 写入权限表

| 写入目标 | a1 | a2 第一刀 | 后续受控行动 |
|---|---|---|---|
| `npcMemories` | 禁止 | 禁止，read-only projection | 仅通过 `living_world_engine` patch |
| `factionPressure` | 禁止 | 禁止，read-only projection | 仅通过批准的 action engine |
| `actionConsequences` | 禁止 | 禁止，read-only projection | 仅通过行动结算 |
| `localActionLedger` | 禁止 | 禁止 | 仅通过统一行动协议 |
| `knownFacts` | 禁止 | 禁止 | 仅公开事实门禁通过后 |
| `hiddenFactRefs` | 禁止 | 禁止 | 仅本地隐藏事实门禁 |
| DeepSeek prompt | 禁止 MiroFish 原始包 | 只允许本地过滤后的公开摘要 | 仍无写入权 |

## 必停用户决策

遇到以下事项必须停下：

- 新增持久化字段或提升 `SAVE_FORMAT_VERSION`。
- 把某个主体升级为命名 NPC runtime rule。
- 正式公开关系数值或势力声望数值。
- 正式招揽、投靠、叛逃、通缉、追杀、抓捕、任务或奖励。
- 把 MiroFish 候选升级为 canon anchor。
- 让 DeepSeek 获得新写入权。

## a2 实现建议

a2 第一刀可以只做 read-only projection：

- 输入本地公开状态。
- 使用少量 RebornG-owned rule draft。
- 输出候选解释。
- 不写 store。
- 不加存档字段。
- 单测先覆盖门禁和排序。

这样可以先验证玩家能否看懂“谁为什么记住我”，再决定是否开放受控写入。
