## 版本定位

`v1.3.0` 是 v1.x 的第三块地基：在路线/地点和低阶生存经济之后，让 NPC、势力、公开事件和社会压力能长期承接玩家行为。

冻结主线：

`玩家行为 -> 关系证据/公开事件 -> NPC/势力压力投影 -> UI 显示谁受影响与下一步 -> DeepSeek 只写社会叙事 -> drift/live probe 验证`

## 当前禁止范围

除非用户另行批准，v1.3 当前不得：

1. 新增或修改持久社会字段。
2. bump `SAVE_FORMAT_VERSION = 25`。
3. 新增 `socialRelationState` 或同类正式聚合对象。
4. 开正式命名 NPC allowlist runtime 规则。
5. 写正式阵营转移、通缉结论、招募成功、任务奖励、NPC 生死或捕获结果。
6. 让 DeepSeek 写正式关系、阵营、通缉、招揽、奖励、地点或隐藏事实。
7. 把 MiroFish 基础包整包导入知识库、runtime、canon 或 DeepSeek visible context。
8. 把 `mirofish-reborng-export` 并入专家团 skill，或让它直接改 RebornG canon/runtime。
9. 自动部署 EdgeOne 或写公开发布承诺。

- 正式 NPC 关系或势力关系持久字段。

- NPC 生死、捕获、背叛、正式任务奖励或阵营转移。
