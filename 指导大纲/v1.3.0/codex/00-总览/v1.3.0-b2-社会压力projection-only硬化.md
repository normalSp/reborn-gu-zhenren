## 结论

b2 没有进入 `SAVE_FORMAT_VERSION = 25`，也没有新增 `socialRelationState`。本阶段把 b1 的边界从文案强化为 helper 返回的 `projectionAudit`，让 UI、测试和后续 b3/b4 都能机械确认：

- 当前仍是 v24。
- 当前无持久社会写入。
- 旧字段不是权威。
- MiroFish 只作为已审查规则来源。
- DeepSeek 无新权限。
- 任何 ledger/promote 都必须重新停下来让用户决策。
