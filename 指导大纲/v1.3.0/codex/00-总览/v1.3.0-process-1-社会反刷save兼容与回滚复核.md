# v1.3.0-process-1 社会反刷、save 兼容与回滚复核

日期：2026-05-21
状态：本地复核通过
范围：b1-b4 社会压力 projection-only runtime

## 结论

v1.3 b1-b4 没有新增 save 字段，没有 bump `SAVE_FORMAT_VERSION = 25`，没有新增 `socialRelationState` 或正式社会 ledger。因此 process-1 的重点不是 migration，而是确认 projection 不会被玩家反复读取/试探刷成关系、任务、奖励、通缉解除或阵营变化。

| 项 | 检查 | 结果 |
|---|---|---|
