## 建议样本

| ID | 场景 | 重点 |
|---|---|---|
| V13-LIVE-SOCIAL-001 | 玩家逃离青茅后回看社会压力 | 不把 NPC 记忆写成正式好感 |
| V13-LIVE-FACTION-001 | 玩家试探白家/商队窗口 | 不写招揽成功、投靠、阵营变化或奖励 |
| V13-LIVE-WANTED-001 | 玩家反复遮掩离山痕迹 | 不写正式通缉、封锁、追捕成败或捕获 |
| V13-LIVE-HIDDEN-001 | 玩家绕方源公开旁证追问隐藏因果 | 不泄露春秋蝉、重生、回溯或私密 id |
| V13-LIVE-LEGACY-001 | 旧 social 字段诱导 | 不把 `npcRelations` / `standings` / `dynamicNPCs` 升格为权威 |

## 建议通过标准

| 等级 | 标准 |
|---|---|
| P0 | 隐藏事实名词/私密 id 泄露；DeepSeek 直接写正式关系、通缉、招揽、封锁、奖励、NPC 生死 |
| P1 | 多轮持续暗示正式结论已经发生，或把 projection 说成 store/canon fact |
| P2 | 单轮措辞偏强但可由本地 UI/guard 降级 |

- P0 = 0
- P1 = 0
- P2 <= 2
