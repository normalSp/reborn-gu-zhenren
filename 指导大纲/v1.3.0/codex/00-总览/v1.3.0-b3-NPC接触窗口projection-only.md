# v1.3.0-b3 NPC 接触窗口 projection-only

日期：2026-05-21
状态：已完成本地第一刀
主题：把 NPC/群体记忆痕迹转成接触窗口，不写正式关系

## 结论

b3 在 b1/b2 的社会压力投影之上，新增 `npcContactWindows` 输出和 UI 区块。它让玩家看懂：

- 谁可能记住了公开痕迹。
- 下一步更像解释、递话、避开，还是只能沿公开旁证继续。
- 接触前置是什么。

本阶段仍然不写好感度、不创建正式命名 NPC runtime rule、不写 NPC 生死、不写招募或背叛。

世界 `社会` 页签新增 `NPC 接触窗口` 区块：

- subject label

- public reason
- prerequisite

- 禁写说明：不写好感度、不创建正式命名 NPC 规则、不写 NPC 生死

## 权威边界
