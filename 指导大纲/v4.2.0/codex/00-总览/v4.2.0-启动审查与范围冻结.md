## 范围冻结

`v4.2` 推荐冻结为：

`Auto-Theater Lite / 凡阶战斗映射设计门禁`

允许讨论和后续在获批后实现：

- 现有凡阶战斗 action 到 Lite preparation / transcript / ledger 的映射。
- 月光蛊、白玉防御、酒虫支援、低阶兽战、失败原因等 synthetic/generic 样本。
- Combat Ledger Lite lane 与玩家可读原因标签。
- reduced-motion、移动端可读性、文本不过载验收。
- report-only checker 草案或后续自有 checker。

不允许在未另批前实施：

- 凡阶战斗 runtime 迁移。
- theater UI implementation。
- 纯自走棋 runtime。
- 高阶战斗 runtime。
- Auto-Theater 素材生成或 runtime 晋升。
- DeepSeek live 战斗表达。
- save/migration/`runFingerprint`。
- MiroFish blocking topic-slice。
