# v4.2.0 MiroFish 资料需求与交付协议

状态：completed。
need level：`not_needed`。

## 当前判断

v4.2 启动包只讨论 Auto-Theater Lite / 凡阶战斗映射设计门禁。样本限定为 RebornG 既有 runtime-safe 或 synthetic/generic 凡阶战斗例子，不使用真实原著战役、命名 NPC、真实势力、hidden-adjacent、方源证据、L4/L5 锚点或正式 lore conclusion。

因此当前不需要 MiroFish export，也不需要 intake review。

## 升级条件

出现以下任一事项，need level 立即从 `not_needed` 升级为 `blocking`，并停止当前推进：

- 使用真实原著战斗、真实战役、真实地点毁灭或命名 NPC 战斗。
- 使用方源、尊者、天意、宿命、L4/L5、真实 era anchor 或隐藏因果证据。
- 将原著稀有蛊、仙蛊、仙蛊屋、杀招、传承或正式 lore conclusion 用作测试真相源。
- 需要把 MiroFish 基础包内容转成 knowledge-index body、runtime canon、DeepSeek visible context 或玩家 UI。

## 如果未来需要

按项目制度走：

`RebornG request -> MiroFish export -> RebornG intake review -> test matrix / rule draft`

任何 MiroFish 输出都只能进入：

- `candidate_pool`
- `fact_card_draft`
- `rule_draft`
- `test_sample`
- `deferred`

不得直接进入 runtime truth、DeepSeek authority、UI、save、canon 或 hidden body。
