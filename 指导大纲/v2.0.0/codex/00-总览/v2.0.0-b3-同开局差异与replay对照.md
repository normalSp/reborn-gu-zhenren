# v2.0.0-b3 同开局差异与 replay 对照

日期：2026-05-22
状态：completed locally
分支：`codex/v200-b3-same-start-replay-diff`

## 一句话

b3 不新增 `runFingerprint`，而是用既有 v25 `regionalEventLedger` 派生同开局 replay lanes，让同一低阶外缘开局能在公开压力、玩家选择和叙事表达上出现可测差异，同时保持稳定事实不漂移。

## 范围

本阶段只做三件事：

- 新增纯本地 helper `buildV200SameStartReplayDiff()`。
- 在现有 `账本` tab 内显示“同开局差异”审计卡，不新增世界面板 tab。
- 用 unit/e2e/live smoke/Player Advocate 验证：同开局差异可读，但不来自新持久随机字段。

## 不做

- 不新增 `runFingerprint`。
- 不新增 `regionalLifeState` / `areaLivingState`。
- 不新增 `identityRouteState` / `professionState`。
- 不新增 `SAVE_FORMAT_VERSION = 26`。
- 不新增 store action、migration/defaults 或新持久字段。
- 不新增 DeepSeek visible knowledge / RAG / ledger write authority。
- 不新增 BFF/backend、子代理、公测口径或 EdgeOne 部署。
- 不开放正式地点、阵营、奖励、NPC 生死、正式商队身份、正式通行、正式价格/库存。

| e2e | `tests/e2e/v200-regional-event-ledger.spec.ts` | 确认账本 sync 后同开局差异卡可见且仍标明不使用 `runFingerprint` |

## replay lane 设计

当前 b3 从公开区域事件派生 6 类 lane：

| lane | 事件来源 | 玩家可见意义 | 边界 |
|---|---|---|---|
| 解释身份 | `checkpoint_questioning` | 解释来路、降低嫌疑、找公开旁证 | 不给正式通行或身份 |
| 商队短工 | `caravan_contact` / `temporary_labor` | 低身份劳作、递话、许可压力 | 不给正式成员、工资、库存 |
| 临市补给 | `market_pressure` | 询价、压价、打听补给 | 不开价格表、库存、交易结算 |
| 遮蔽人情 | `shelter_debt` | 求助、人情债、守夜或脱身 | 不写长期债务字段或 NPC 命运 |
| 路途避险 | `road_conflict_pressure` | 绕路、等待、押运压力 | 不结算奖励、掉落、NPC 生死 |
| 远城门槛 | `gate_threshold` | 大城作为方向压力 | 不开放完整城市或正式身份系统 |

## 差异来源

允许：

- 已登记公开区域事件。
- pressure lane 覆盖差异。
- 玩家选择顺序与应对倾向。
- DeepSeek 对本地事实的叙事表达差异。

禁止：

- `runFingerprint`。
- 稳定事实随机变化。
- 正式地点、身份、阵营、奖励、NPC 生死随机变化。
- hidden/private 揭示。
- DeepSeek 自行裁决事实。

## 残余风险

| 等级 | 风险 | 处理 |
|---|---|---|
| P2 | live probe 出现 `青毛` 术语错字 | 进入 process-1 prompt/eval hardening |
| P2 | live probe 仍出现 `通行证`、`正式凭证` 等正式凭信词风险 | 进入 process-1 / rc T3 hardening |
| P3 | replay lane 目前不写持久随机种子，同开局差异强度需 T3 长测验证 | rc T3 验证；若不足再让用户单独决策 `runFingerprint` |
