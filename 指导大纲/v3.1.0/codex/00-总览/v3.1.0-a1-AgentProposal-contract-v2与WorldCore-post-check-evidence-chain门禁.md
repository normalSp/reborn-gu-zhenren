# v3.1.0-a1 AgentProposal contract v2 与 WorldCore post-check evidence chain 门禁

日期：2026-05-24
状态：completed
冻结模型：`v310_a1_agentproposal_contract_v2_worldcore_evidence_chain_v1`

## Contract v2 必填项

每个 AgentProposal 候选必须具备：

- `agentLayers`：仅允许 L2/L3 synthetic candidate。
- `proposalOnly=true`。
- `visibleContextLimited=true`。
- `visibilityClass=synthetic_redacted`。
- `worldCoreFinalAuthority=true`。
- `requiresWorldCorePostCheck=true`。
- `postCheckOutcome`：只能是 `approved_expression`、`rejected_violation`、`needs_user_decision`、`manual_review`、`future_gate_required`。
- `persistenceMode=transient_candidate_only`。
- `noSaveFieldAdded=true`。
- `evidenceRefs`：至少两个证据来源。

## WorldCore post-check evidence chain

WorldCore post-check 必须能输出：

- `approved_expression`：仅允许表达候选，不写事实。
- `rejected_violation`：阻断 P0/P1 越权。
- `needs_user_decision`：需要用户拍板。
- `manual_review`：需要人工复核，但未自动越权。
- `future_gate_required`：触发 F-310。

## 禁止混淆

- `manual_review` 不是批准。
- `future_gate_required` 不是实现授权。
- `approved_expression` 不是正式事实。
- `proposalOnly` 不是 runtime 接入。

## a1 结论

`v310_a1_agentproposal_contract_v2_worldcore_evidence_chain_v1` 成立。可进入 a2 gate hardening。
