## 默认节奏

1. startup 阶段只写启动包，不进入实现。
2. 用户批准 D-310 后，可用 `/goal` 连续完成 v3.1。
3. 触发 F-310 任一项，立即停止自动推进。
4. v3.1 完成后，下一步只建议进入 v3.2 rehearsal startup；不自动打开 F-300/F-310 runtime implementation。

## 不走的捷径

- 不把 b1 checker 伪装成 runtime agent。
- 不把 report archive 伪装成 backend/eval archive service。
- 不把 synthetic sample 伪装成真实原著事实。
- 不把 live metadata plan 伪装成 live DeepSeek 授权。
