# RebornG 知识库入口

日期：2026-05-20
状态：v1.1-process-2 建立骨架；非 runtime authority

## 定位

本目录存放 RebornG 项目知识索引。它服务专家团、MiroFish intake、测试矩阵、长线叙事漂移审查和后续 canon schema 工作。

本目录不是：

- runtime canon。
- DeepSeek visible context。
- 玩家可见内容源。
- 原著正文仓库。

所有条目必须遵守 `指导大纲/流程制度/全书知识库治理制度.md`。

## 使用规则

1. 新条目先进入知识索引，不直接进入 `src/canon`。
2. hidden/private 条目只能存 id、边界、source pointer 和使用限制，不存可见正文。
3. 任何条目进入 runtime 前，必须经过当前版本设计门禁、测试和用户决策。
4. 全书基础包不能整包导入；只能按版本目标、主题切片和 intake review 结果吸收。
