# 首批凡蛊与杀招样板清单

更新时间：2026-05-10  
对应数据：`src/canon/gu-expression-specs.json`、`src/canon/killer-move-expression-specs.json`

## 选型原则

v0.8.0 首批不追求一次覆盖所有流派，而是优先覆盖前中期凡战职责：

- 攻击、压制、破甲、击退。
- 防御、护持、定点守护。
- 控制、地形、伏击、追击。
- 侦察、鉴别、夜间警戒。
- 治疗、解毒、真元准备、后勤。
- 禁忌蛊样板，验证场景门槛和长期代价。

流派覆盖以青茅山、商路、商家城、三王山、王庭前中期常见需求为优先，后续再扩展星道、音道、信道、画道、云道等更高阶或更特殊流派。

## 首批蛊虫

攻击/压制：

- 月光蛊：直线切割，能切绳、压制远距目标。
- 月芒蛊：近距月华扩散，兼具眩目与照明。
- 小光蛊：增强光道蛊与照明，不做伤害按钮。
- 骨枪蛊：穿刺与钉住。
- 熊力蛊：近身破防。
- 蛮力天牛蛊：直线冲撞与击退。
- 水龙蛊：水道推进、浸湿、灭火。
- 火鸦蛊：追扰火源和环境风险。
- 雷翼蛊：短距换位和电击。
- 春雷蛊：打断、唤醒、触发机关。
- 岩枪蛊：地面突刺。
- 金龙蛊：弧线撕甲。

防御/护持：

- 石皮蛊：土道硬抗，减速，怕水道软化。
- 铜皮蛊：轻金属护身，引雷风险。
- 铁皮蛊：重防守点。
- 玉皮蛊：精细护体与轻抗毒。
- 白玉蛊：高消耗完整护体。
- 金钟蛊：定点护罩。
- 金缕衣蛊：护持友方。
- 金罡蛊：近战反震。
- 水幕天华蛊：水幕遮蔽与灭火。
- 月霓裳：月华护持。

控制/地形：

- 力气蛊：推拉和触发地形。
- 横冲直撞蛊：破障并暴露自身。
- 青丝蛊：束缚、拉扯。
- 种蛊：伏击前置布置。
- 裂地蛊：改变棋盘格。
- 幽影随行蛊：阴影换位和跟踪。
- 水蛛蛊：湿地蛛网控制。
- 破风蛊：清雾、推开飞行物。
- 爱生离：毒道剧情门槛。
- 毒液蛊：毒池和腐蚀格。

侦察/情报：

- 书虫：识别蛊、记录和证据。
- 侦察蛊：揭示伏击和危险格。
- 追踪蛊：追击链路。
- 巡夜蛊：夜间防伏击。
- 电眼蛊：读动作和机关细线。
- 察运蛊：读取风险倾向，不直接改运气。
- 鉴毒蛊：辨毒。
- 避瘴蛊：穿越毒瘴。

续航/辅助：

- 酒虫：真元质量和战前准备。
- 净水蛊：解毒、清洁格。
- 治愈蛊：近距治疗和稳定伤势。
- 金风送爽蛊：持续恢复、降温。
- 醒酒蛊：解除酒/迷乱。
- 辟谷蛊：后勤与长途。

特殊/禁忌样板：

- 血颅蛊：禁忌血道剧情，强场景门槛。
- 妇人心蛊：毒道炼蛊剧情，尸体/毒道门槛。
- 月影蛊：潜行和伏击。
- 驭虫蛊：环境虫群侦察与骚扰。
- 黑豕蛊、白豕蛊：长期体魄路线。
- 全力以赴蛊：力道爆发与出力后空档。

## 首批杀招

- 月刃连斩：光道连切，适合夜战和破轻遮蔽。
- 石皮护体：土道定身硬抗。
- 木灵缠绕：种蛊 + 青丝蛊的伏击控制。
- 草木生长：治疗、恢复和生成遮蔽。
- 青丝绞杀：室内/窄路束缚与拉扯。
- 水龙卷：水道与风道合势，推进、灭火、清烟。
- 金钟不破：定点护罩和守点。
- 巨灵变：力道体魄爆发，风险是体型暴露和疲劳。
- 酒虫精炼阵：战前真元质量准备。
- 食铁者：针对金属护甲、金道护罩和锁链。
- 吞噬强化：以战场残留换短时强化。
- 无形之手：偷道低阶样板，依赖未察觉状态。
- 影子行窃：暗光环境潜行窃取。
- 血颅续航：禁忌血道剧情门槛，不开放普通回血。
- 血滴子杀阵：高风险血道区域杀阵。

- 蛊仙阶段再扩展：宙道、宇道、梦道、天道、气道、星道、画道、音道。
