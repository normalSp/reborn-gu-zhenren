## runner 行为边界

runner 只做：

- 读取 `tests/evals/v230-agent-eval-farm/samples.json`。
- 校验 `agent_eval_farm_report_v230_a2` 合同。
- 检测 `agent_eval_taxonomy_v230_a1` failure families。
- 计算 P0/P1/P2、false negative、false positive、manual review、rescore stable。
- 写 report-only artifact。

runner 不做：

- 不读 `.env`。
- 不调用 live DeepSeek。
- 不改 runtime/source/UI/store/prompt/save。
- 不写正式存档。
- 不引入第三方依赖。
- 不启用 subagents。
- 不接 BFF/backend。
- 不做 MiroFish export/intake。
- 不做外部 SDK/framework PoC。
- 不开放正式地点、阵营、身份、奖励、NPC 生死或 canon promotion。
