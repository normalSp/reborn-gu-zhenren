# v0.16.0-b3 高阶与演武入口场景化

日期：2026-05-18
状态：已完成
类型：runtime/UI + tests/docs

## 目标

把高阶系统、演武/调试入口、宿命/传承/终局/商会等非低阶常驻入口移出底栏，统一放入 `世界` 工作台。

## 运行时变化

- 新增 `WorldHubPanel`。
- 底栏不再显示 `青茅 / 凡战 / 群像 / 大阵` 调试/演示按钮。
- `世界` 工作台包含：
  - `总览`
  - `宿命`
  - `传承`
  - `终局`
  - `商会`
  - `道场`
  - `小队`
  - `演武`
- `演武` 是开发/演示抽屉，明确不代表正式剧情战斗触发。

## 权威边界

- 高阶入口默认只读或场景化。
- 不开放正式地点、阵营、奖励、NPC 生死、宝黄天交易或结局裁决。
- 演武入口可以启动 UI/战斗演示，但不写正式剧情事实。
- 世界工作台不扩大 DeepSeek 权限。

## 测试证据

- `npm run test:e2e -- tests/e2e/v016-navigation-workspaces.spec.ts`：通过。
- `npm run test:e2e -- tests/e2e/v090-product-route-closure.spec.ts`：通过。
- 旧道场/宿命/传承入口在兼容回归批次中通过。
