## 修复过的回归点

- 旧 e2e 重复点击已激活的 `世界` / `蛊道` 底栏按钮会关闭侧栏，已改为直接切工作台标签。
- 青茅凡战资产检查从立即读取 `img.complete` 改成等待图片自然尺寸就绪，避免加载竞态。
- Playwright 对桌面/移动双 DOM 的严格定位改为 `:visible` 范围，避免隐藏副本造成 strict-mode 冲突。

## Player Advocate 结论

rc 60 轮通过，95% 下一步理解率，3 个 P2 体验观察：

- 演示入口移入 `世界/演武` 后，新玩家可能不会第一时间找到，但这是预期，因为演示不应是正式主入口。
- 少量玩家不立即知道杀招归入 `蛊道`，v0.17 战斗深化时继续观察。
- 成就归入 `角色` 后，少量玩家会先找 `记录`，后续如有需要再加轻提示。

没有 P0/P1 玩家阻断。
