# v0.16.0-a1 主导航与入口归并表

日期：2026-05-18
状态：第一版

## 一级入口冻结

| 一级入口 | 玩家心智 | 运行时可见形态 | v0.16 目标 |
|---|---|---|---|
| `地图` | 我在哪里，周围有什么 | 当前域可显示为 `青茅舆图` 等 | 保留为一级入口 |
| `行动` | 我现在能做什么，我想尝试什么 | 工作台，承接当前行动、自由目标、线索后续、路线/交易/社会机会 | b1 第一刀 |
| `角色` | 我是谁，我的状态和关系如何 | 玩家属性、空窍、人物、道痕、成就等 | b1/b3 后保持一级入口 |
| `蛊道` | 我的蛊、杀招、炼养用资源和蛊材 | 蛊虫、杀招、炼蛊、蛊材 | b2 第一刀 |
| `世界` | 世界局势、剧情锚点、高阶入口 | 宿命、终局、传承、商会、局势概览等受控入口 | b3 场景化 |
| `记录` | 已发生什么，哪些是候选和回流事实 | 事件日志、行动账本、可见事实回顾 | b1/b4 逐步收束 |

## 行动工作台内部建议

`行动` 工作台在 b1 中优先承接五类内容：

| 子区 | 内容 | 权威 |
|---|---|---|
| `当前行动` | 当前场景行动、青茅区域行动、本地可执行行动 | 现有 ActionPanel engines/store actions |
| `自由目标` | 玩家输入长期或短期目标、本地裁决预览和确认 | World Intent Engine + controlled living-world patch |
| `线索后续` | 调查、旁证、白家接触、遮掩痕迹等后续 | v0.11-v0.15 已有候选/正式前置 action gate |
| `路线/交易/社会机会` | 路线准备、市场窗口、灰色交易边界、社会反应 | candidate/read-only/controlled action split |
| `行动账本` | 本轮本地结算事实和回流文本依据 | local action ledger，不是 DeepSeek 权威 |

## 蛊道工作台内部建议

`蛊道` 工作台在 b2 中优先承接四类内容：

| 子区 | 内容 | 权威 |
|---|---|---|
| `蛊虫` | 已有蛊虫、养用提示、状态 | owned Gu/inventory 状态 |
| `杀招` | 已知杀招、候选杀招、战斗使用边界 | kill move 状态与战斗系统 |
| `炼蛊` | 炼蛊边界、候选蛊方、缺口提示 | refine/material registry；不私算成功 |
| `蛊材` | 已有蛊材、来源提示、消耗边界 | materialBag；不新增发放 |

## 高阶入口显示原则

`传承 / 宿命 / 终局 / 商会 / 宝黄天 / 仙蛊 / 蛊仙相关入口` 不能在一转青茅山开局常驻压迫玩家。

允许显示的场景：

1. 开发/演示抽屉。
2. 明确剧情或可见事实触发的受控入口。
3. 只读世界观说明，且明确“未开放/不可执行/候选”。
4. 后续版本用户批准的正式开放阶段。

禁止显示的暗示：

1. 玩家已拥有仙蛊、宿命蛊、十转、永生。
2. 玩家可在低阶阶段正式交易宝黄天。
3. 玩家可随手获得传承、结局裁决或高阶地点进入权。
4. DeepSeek 可直接补发高阶奖励。
