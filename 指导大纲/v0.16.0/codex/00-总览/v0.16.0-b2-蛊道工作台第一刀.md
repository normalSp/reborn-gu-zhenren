# v0.16.0-b2 蛊道工作台第一刀

日期：2026-05-18
状态：已完成
类型：runtime/UI + tests/docs

## 目标

把 `蛊虫 / 杀招 / 炼蛊 / 蛊材` 收束为 `蛊道` 工作台，降低底栏噪音，并让玩家知道这些都是“蛊师能力与资源”的同一类问题。

## 运行时变化

- 新增 `GuDaoPanel`。
- `GameScreen` 新增一级入口 `蛊道`。
- `蛊道` 工作台包含：
  - `蛊虫`：原 `GuInventoryPanel`。
  - `杀招`：原 `KillMovePanel`。
  - `炼蛊`：原 `RefinePanel`。
  - `蛊材`：原 `MaterialBagPanel`。
- 为 `RefinePanel` 和 `MaterialBagPanel` 增补稳定测试入口。

## 权威边界

- 蛊道工作台只归并入口，不改变拥有蛊、蛊材、残方、炼蛊失败代价或杀招来源。
- UI 不私算蛊方成功。
- UI 不消耗材料。
- UI 不发放蛊虫、蛊材、蛊方或奖励。
- 低阶阶段仍不开放正式市场库存、价格表、黑市交易或宝黄天交易。
