# v0.16.0-b1 行动与自由目标工作台第一刀

日期：2026-05-18
状态：已完成
类型：runtime/UI + tests/docs

## 目标

把底栏的 `行动` 与原 `自由目标` 合并为一个玩家主入口，减少底栏按钮数量，同时保留自由目标的本地裁决和写入边界。

## 运行时变化

- 新增 `ActionHubPanel`。
- `GameScreen` 的 `actions` 面板改为行动工作台。
- 行动工作台包含：
  - `当前行动`：原 `ActionPanel`。
  - `自由目标`：原 `FreeGoalPanel`。
- 底栏不再暴露 `自由目标` 独立按钮。

## 权威边界

- `ActionPanel` 继续执行既有本地行动。
- `FreeGoalPanel` 继续只通过既有 living-world patch / World Intent gate 写入目标和候选。
- UI 合并不授予任何奖励、地点、阵营、NPC 生死或路线成功。
- DeepSeek 权限没有变化。
