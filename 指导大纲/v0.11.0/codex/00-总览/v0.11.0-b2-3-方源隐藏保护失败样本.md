# v0.11.0-b2-3 方源隐藏保护失败样本

日期：2026-05-15
状态：已完成。
结论：玩家可以尝试打听/跟踪方源，但当前版本只会得到隐藏事实保护下的公开失败账本；不会泄露春秋蝉、回溯、重生或方源私有因果。

## 本刀目标

让“可尝试但不能泄密”的调查也留下工程化、可回归的活世界痕迹。

## 已实现

- `resolveQingmaoVisibleInvestigation()` 在 `targetRef = npc:fang_yuan` 且只命中隐藏事实时，生成：
  - `hiddenFactRefs.fang_yuan_private_causality_hidden_anchor`
  - `actionConsequences` 一条公开失败后果
  - `npcMemories` 一条谨慎记忆
- `npcMemories.privateRefId` 指向隐藏事实 ref，但 `publicSummary` 不包含隐藏正文。
- `resolveVisibleInvestigationAction()` 会通过 `living_world_engine` patch 写入 `npcMemories` 和 `actionConsequences`。
- e2e 摘要新增 `npcMemoryCount/npcMemoryIds`。

## 明确没有做

- 不让玩家知道春秋蝉、回溯、重生或方源私有因果。
- 不判定方源发现玩家。
- 不改变方源关系、敌意、位置、生命状态。
- 不改正史锚点。
- 不开放跟踪方源的完整行动链。
