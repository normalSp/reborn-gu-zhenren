## v0.11.0-a0 玩家可见旧债清理

状态：已完成第一刀，见 `v0.11.0-a0-玩家可见旧债清理.md`。

目标：

- 解决截图中“待 v0.8 深线补充”这类旧版本文案。
- 避免玩家在游戏里看到过期开发承诺。

已知候选：

- `src/canon/start-profiles.json`：白家寨一转保底蛊旧占位。
- `src/canon/faction-data.json`：白家寨长线留到 v0.8 的旧文案。

做法：

- 改成当前真实口径：当前提供身份、资源入口、青茅外线压力；专属保底蛊/深线奖励需后续区域持久化和原著事实审查后开放。
- 增加检查，禁止玩家可见 canon/UI 文案出现 `待 v0.x`、`留到 v0.x`。
- 不新增玩法，不改数值。

## v0.11.0-a3 自由意图裁决引擎

前置门禁：

- 必须先通过 `v0.11.0-世界意图裁决引擎-设计门禁.md`。
- 不得跳过现有统一行动协议、Qingmao region engine、combat/resource/refine/story/canon/store 的读写权审计。

目标：

- 建立 World Intent Engine 的最小版本。

首批意图：

- `obtain_item`
- `join_faction`
- `investigate`
- `travel`
- `long_term_goal`

用户已批准第一刀范围：

- 只做自由意图裁决入口，不做完整大世界。
- 首批意图只支持上述五类。
- DeepSeek 只生成候选解释/表达方式，本地引擎最终裁决可行性、门槛、阻断理由和回流字段。
- 极端意图样本必须进入测试。

首批极端样本：

- 我要拿九转蛊。
- 我要获得春秋蝉。
- 我要去宝黄天交易。
- 我要投靠白家。
- 我要跟踪方源。
- 我要逃离青茅山。
- 我要杀死白凝冰。

裁决输出：

- 可执行。
- 可执行但有代价。
- 需要前置条件。
- 只能传闻。
- 只能长期目标。
- 世界观硬阻断。
- 隐藏事实阻断。
- 重大 IF 偏离。

## v0.11.0-b1 原著事实抽取试点

目标：

- 不再靠记忆做青茅山锚点。

做法：

- 从本地原著文本抽取摘要事实卡。
- 只写摘要、章节/位置指针、设计解释，不复制大段原文。
- 分类：原著硬事实、玩家可见事实、隐藏事实、可 IF 偏离点。

不做：

- 不做完整青茅主线。
- 不把事实卡直接变成剧情成功条件。

已完成第一刀：

- 新增 `src/canon/qingmao-canon-fact-cards.json`。
- 首批 8 张事实卡覆盖：青茅/古月山寨、开窍/族学、灵泉元石根基隐藏事实、三寨格局、白凝冰公开天才压力、月光蛊本地特产、酒虫支持定位、方源核心因果隐藏保护。
- 事实卡只保存摘要、来源指针、关键词和设计边界，不复制原著正文。
- 隐藏事实必须 `runtimeExposure=hidden_ref_only`，不能有 `playerVisibleSummary`。
- 新增 `src/canon/qingmao-canon-fact-cards.test.ts`。

已完成 b1-2：

- 新增 `src/engine/v011-qingmao-fact-cards.test.ts`，并扩展 `v011-world-intent-engine.test.ts`。
