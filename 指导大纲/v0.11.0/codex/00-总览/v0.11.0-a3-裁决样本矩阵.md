日期：2026-05-15
状态：a3 第一版已建立；a3-2 UI/store、b2 可见范围调查、b2-2 白家压力、b2-3 方源隐藏保护失败账本、b2-4 调查后续提示候选、b2-5 白家接触窗口正式行动和 b2-6 逃离青茅山路线准备链已补样本，当前对应 `src/engine/v011-world-intent-engine.test.ts`、`src/engine/v011-qingmao-fact-cards.test.ts`、`src/engine/v011-qingmao-visible-investigation.test.ts`、`src/engine/v011-qingmao-investigation-followups.test.ts`、`src/engine/v011-qingmao-bai-contact-window.test.ts`、`src/engine/v011-qingmao-escape-route-prep.test.ts`、`src/store/slices/livingWorldSlice.test.ts` 和 `tests/e2e/v011-free-goal-panel.spec.ts`。

## 目的

本矩阵把自由意图裁决从“凭感觉判断”变成可回归、可扩展、可量化的测试资产。

当前矩阵已覆盖 a3 第一刀、a3-2 UI/store 第一刀、b2 可见范围调查第一刀、b2-2 白家接触压力样本、b2-3 方源隐藏保护失败账本、b2-4 调查结果后续提示候选、b2-5 白家接触窗口正式行动和 b2-6 逃离青茅山路线准备链。后续若继续扩大到势力归属、地点开放、NPC 生死或正史锚点，本矩阵必须继续扩展。

## 样本分类

| 分类 | 说明 |
|---|---|
| `golden` | 核心正向样本，必须长期稳定 |
| `boundary` | 世界观、奖励、地点、NPC 生死、隐藏事实硬边界 |
| `identity` | 同一目标在不同出身/势力下裁决不同 |
| `fuzzy` | 同义、口语、错别字、模糊目标 |
| `deepseek_override` | DeepSeek 越权候选必须被本地覆盖 |

## 当前断言重点

### 九转蛊

必须断言：

- `intentType = obtain_item`
- `targetRef = item:rank_nine_gu`
- `category = long_term_goal`
- `allowed = false`
- `suggestedPlayerGoal.status = deferred`
- `statePatchApplied = false`
- `route.forbiddenStateWrites` 包含 `inventory`

### 春秋蝉

必须断言：

- `category = world_rule_blocked`
- 不出现隐藏因果正文。
- 不进入背包、奖励、蛊方、传承。

### 宝黄天交易

必须断言：

- 凡人阶段 `category = world_rule_blocked`
- 理由包含凡人阶段不能正式交易。
- DeepSeek 不得生成价格、交易结果或奖励。

未来补充：

- 蛊仙阶段可裁为 `available_with_cost`，但仍必须路由到交易系统，不能由意图裁决直接结算。

### 投靠白家

必须断言：

- 古月/熊家身份不能一键换阵营。
- 白家身份可以转成本家行动。
- 散修/外来身份可以尝试接触，但不直接获得白家身份。

后续 a3-2/b2 补充：

- 触发势力压力。
- 写入 `playerGoals` 时保留身份风险。
- 不直接改 `currentFaction` 或声望。

### 跟踪方源

必须断言：

- 可以裁为可见范围内调查。
- 不泄露回溯、重生等隐藏事实。
- 有被警觉或反利用风险。

后续 b1/b2 补充：

- 依赖原著事实卡和隐藏事实引用。
- DeepSeek prompt 只能看到玩家已知事实。

### 逃离青茅山

必须断言：

- 不直接开放新地域。
- 不传送。
- 需要路线、补给、身份遮掩、追踪风险。

后续 a3-2/b2 补充：

- 确认为长期目标后写入 `livingWorldState.playerGoals`。
- 通过行动协议逐步获得路线线索。

### 杀死白凝冰

必须断言：

- 保持在首批五类意图内，当前归为 `long_term_goal`。
- `category = major_if_deviation`
- 不直接判定 NPC 死亡。
- 不改正史锚点。

后续 b1/b2 补充：

- 正史模式和 IF 模式分流。
- IF 模式也必须检查实力、情报、时机、代价和后续账本。

### DeepSeek 越权候选

必须断言：

- 即使 `source = deepseek_candidate`，本地 ruling 仍覆盖模型。
- DeepSeek contract 中 `localRulingAuthority = true`。
- forbidden outputs 包含 direct rewards。
- `statePatchApplied = false`。

## b2 后续必须新增样本

目标：b2 收束前至少 80 条。

### 原著事实卡样本

- 玩家不知道的隐藏事实不得出现在 visible explanation。
- 玩家已知事实可以进入 DeepSeek visible summary。
- 隐藏事实只能通过 `hiddenFactRefs` 引用。
- IF 偏离必须有 anchor ref。

### 青茅活世界样本

- 投靠白家后，古月怀疑压力上升，但不直接改势力归属。
- 跟踪方源失败后，写入 NPC 警觉记忆，但不泄露隐藏因果。
- 逃离青茅山准备路线后，区域行动后果写入 `actionConsequences`。
- 白家接触成功只开放候选行动，不给白家身份。

### DeepSeek eval 样本

- 模型生成“你得到春秋蝉”，本地覆盖为阻断。
- 模型生成“宝黄天交易成功获得仙材”，本地覆盖为凡人传闻。
- 模型生成“白凝冰已死”，本地覆盖为重大 IF 未决。
- 模型生成隐藏事实说明，过滤或降级。

## v0.11 rc 前目标

目标：至少 100 条样本，覆盖：

- 5 类首批意图。
- 8 类裁决分类。
- 青茅主要出身。
- canon / if 模式。

- DeepSeek 越权候选。
- UI 确认/取消/重复/读档。

rc 阻断条件：

- 硬边界样本不是 `100%`。

- 出现隐藏事实泄露。
