# v0.11.0-b1-2 事实卡读取与裁决引用桥

日期：2026-05-15
状态：已完成。

## 目的

b1 第一刀只建立了青茅事实卡。b1-2 的目标是把事实卡变成运行时代码可安全读取、可测试引用的底层能力，但仍然不进入正式剧情结算。

本阶段重点是三条线：

1. 事实卡只能通过只读 helper 读取，避免散落的 JSON 访问。
2. 玩家可见事实可以进入裁决上下文。
3. 隐藏事实只进入本地保护引用，不进入 `DeepSeek` 可见事实列表和玩家可见摘要。

## 新增源码

`src/engine/v011-qingmao-fact-cards.ts`

提供：

- `listQingmaoCanonFactCards()`：克隆读取全部事实卡。
- `getQingmaoCanonFactCard(id)`：按 ID 读取单张事实卡。
- `listVisibleQingmaoCanonFactCards()`：只返回非隐藏事实卡。
- `listHiddenQingmaoCanonFactCards()`：只返回隐藏事实卡。
- `buildQingmaoFactReferenceSet()`：根据自由意图目标/文本生成 `visibleFactCardIds` 与 `hiddenFactCardIds`。
- `buildQingmaoFactPromptContext()`：构造可喂给后续 prompt/UI 的安全上下文。
- `buildPlayerKnownFactFromQingmaoCard()`：把玩家可见事实卡转成 `PlayerKnownFact` 草案。
- `buildHiddenFactRefFromQingmaoCard()`：把隐藏事实卡转成 `HiddenFactRefState` 草案。

## 裁决引擎桥接

`src/engine/v011-world-intent-engine.ts`

新增输出字段：

- `factCardRefs.visibleFactCardIds`
- `factCardRefs.hiddenFactCardIds`
- `factCardRefs.sourcePointerRefs`

并更新 `deepSeekContract.visibleFactIds`：

- 会合并当前玩家已知事实 ID 与可见事实卡 ID。
- 不会加入隐藏事实卡 ID。

当前桥接样例：

| 意图/目标 | 可见事实卡 | 隐藏事实卡 |
|---|---|---|
| 投靠白家 | `qingmao_three_clans_layout`、`baijia_bai_ning_bing_public_talent` | 无 |
| 跟踪方源 | 无 | `fang_yuan_private_causality_hidden_anchor` |
| 逃离青茅山 | `qingmao_location_guyue_village`、`qingmao_three_clans_layout` | 无 |
| 调查灵泉 | 视文本补青茅/古月可见卡 | `guyue_spirit_spring_resource_basis` |
| 春秋蝉相关 | 无 | `fang_yuan_private_causality_hidden_anchor` |

## 隐藏事实防线

隐藏事实卡即使被错误地传入 `visibleFactCardIds`，`buildQingmaoFactPromptContext()` 也会把它重定向为：

- `guard: hidden`
- `runtimeExposure: hidden_ref_only`
- `revealPolicyId`
- `sourcePointerRefs`

不会输出：

- `summary`
- `playerVisibleSummary`
- 原著正文
- 回溯、春秋蝉等玩家角色不可见解释

## 新增测试

`src/engine/v011-qingmao-fact-cards.test.ts`

覆盖：

- helper 返回克隆对象，测试修改不会污染源数据。
- visible helper 不返回隐藏卡。
- prompt context 只使用玩家可见摘要。
- 隐藏卡被误放到 visible 请求时仍会被红线化。
- 自由意图目标能映射到 visible/hidden fact refs。
- 可见事实卡可以转成 `PlayerKnownFact`。
- 隐藏事实卡只能转成 `HiddenFactRefState`，没有 `summary` 字段。

## 非目标

本刀不做：

- 不写 `livingWorldState`。
- 不新增存档字段，不提升 `SAVE_FORMAT_VERSION`。
- 不把事实卡变成奖励、地点、NPC 生死或剧情成功条件。
- 不把隐藏事实正文交给 UI 或 DeepSeek。
- 不做完整青茅主线。
