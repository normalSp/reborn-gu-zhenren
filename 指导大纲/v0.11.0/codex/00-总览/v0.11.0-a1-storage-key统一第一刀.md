# v0.11.0-a1 storage key 统一第一刀

日期：2026-05-15
状态：已完成

## 目标

把分散在 API、store、slice、UI 和 e2e harness 里的 localStorage key 收束到统一 registry，后续做存档/设置/诊断治理时不再靠到处搜索硬编码字符串。

  - `saveSlotMetaKey()`
- 新增 `src/store/storageKeys.test.ts`
  - 锁定旧 key 字符串不变。
  - 锁定手动存档槽位前缀不变。
- 已迁移使用点：
  - `src/api/deepseek.ts`

  - `src/store/slices/originUnlockSlice.ts`

  - `src/components/game/SettingsDialog.tsx`

## 保持不变

- 主存档 key 仍为 `gu-zhenren-save`。
- 成就 key 仍为 `gu-zhenren-achievements` / `gu-zhenren-achievement-progress`。
- DeepSeek API key 仍为 `deepseek_api_key`。
- 手动槽位前缀仍为 `gu-zhenren-slot-` / `gu-zhenren-meta-`。
- 字号、声音、起源、新手引导和诊断日志 key 均保持原字符串。

## 边界

- 本刀只抽常量，不改 key，不做迁移，不删除旧数据，不改变 API key 存储策略。
- 下载文件名 `gu-zhenren-save-...json` 不是 localStorage key，保持在导出逻辑中。
