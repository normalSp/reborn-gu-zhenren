# v0.11.0 世界意图裁决引擎专项

日期：2026-05-15
状态：设计专项；预计在 `v0.11.0-a3` 实施第一刀；运行时代码开始前必须先通过 `v0.11.0-世界意图裁决引擎-设计门禁.md`。

## 问题

玩家希望像和 AI 跑团一样自由表达目标：

- 我要投靠白家。
- 我要跟踪方源。
- 我要逃离青茅山。
- 我要拿九转蛊。
- 我要去宝黄天交易。

如果完全交给 DeepSeek，会出现：

- 越阶奖励。
- 世界观错误。
- 隐藏事实泄露。
- 正史核心被随意改写。
- 存档无法量化。

如果只给固定按钮，又会失去自由感。

## 方案

建立本地 World Intent Engine。

流程：

1. 玩家输入自由目标。
2. DeepSeek 或本地 parser 生成 `IntentCandidate`。
3. 本地 engine 读取玩家状态、区域状态、canon、可见事实和隐藏事实。
4. 本地 engine 输出 `IntentRuling`。
5. DeepSeek 只根据 ruling 写解释和候选。
6. store 写入允许的状态变化。

## 数据草案

```ts
type IntentKind =
  | 'obtain_item'
  | 'join_faction'
  | 'investigate'
  | 'travel'
  | 'long_term_goal'
  | 'social'
  | 'attack_or_betray'
  | 'train_or_refine'
  | 'unknown';

type IntentRulingKind =
  | 'executable_now'
  | 'executable_with_cost'
  | 'requires_precondition'
  | 'rumor_only'
  | 'long_term_goal_only'
  | 'blocked_by_lore'
  | 'blocked_by_visibility'
  | 'major_if_deviation'
  | 'dangerous_attention';
```

## 首批样本

### 九转蛊

玩家目标：我要拿九转蛊。

裁决：

- 一转凡人不可直接获得。
- 只能作为长期目标、传闻调查或被骗风险。
- 不允许进入背包、奖励、蛊方、传承。

### 春秋蝉

玩家目标：我要获得春秋蝉。

裁决：

- 当前青茅低阶玩家不可获得。
- 春秋蝉属于高阶特殊事实，不能作为普通奖励。
- 可转为“听闻春秋/时间类传闻”或“时间道长期目标”。

### 宝黄天

玩家目标：我要去宝黄天交易。

裁决：

- 凡人不可正式交易。
- 可听闻宝黄天传说。
- 蛊仙期另行开放。

### 投靠白家

玩家目标：我要投靠白家。

裁决：

- 根据身份不同分支处理。
- 古月身份：高风险叛逃/接触。
- 白家身份：本家任务。
- 散修身份：求庇护。
- 商队身份：贸易关系接触。

### 跟踪方源

玩家目标：我要跟踪方源。

裁决：

- 不能泄露“方源重生”等隐藏事实。
- 只能做可见范围内的观察、跟踪、打听。
- 失败可能引起警觉或反利用。

### 逃离青茅山

玩家目标：我要逃离青茅山。

裁决：

- 检查路线、资源、时间、身份、追踪风险。
- 可以变成长线目标或行动候选。
- 不是立即地图传送。

### 杀死白凝冰

玩家目标：我要杀死白凝冰。

裁决：

- 正史模式：重大锚点保护，通常阻断或转为局部干扰。
- IF 模式：重大 IF 偏离，需要实力、情报、时机、代价和后续账本。
- 不允许 DeepSeek 直接判定成功。

## 验收

- 已通过 `v0.11.0-世界意图裁决引擎-设计门禁.md`。
- 所有首批样本有 focused tests。
- DeepSeek 给出越界文本时，本地 ruling 能覆盖。
- UI 能显示为什么不能做，以及当前能做什么。
- 不可行目标不会写入奖励、背包、地点、正史或 NPC 生死。
