## 风险与延期

延期到后续版本：

- 方源旁证调查正式行动：需要更多公开活动事实卡和隐藏事实门禁。
- 逃离青茅山后续地域开放：需要路线系统、补给系统、追踪风险和区域状态设计。
- 白家正式投靠：需要身份、投名状、势力压力、声望与正史/IF 边界。
- NPC/势力反应引擎深化：v0.11 只做最小压力和记忆样本。
- GitHub Actions、ADR 补档、只读子代理试点、Prompt Budget、Canon Schema：已进入 v0.12 候选池。
