# v0.11.0-b2-2 白家接触压力样本

日期：2026-05-15
状态：已完成。
结论：在 b2 可见范围调查第一刀基础上，补上最小“世界反应”：白家相关调查会写入 `factionPressure` 的接触机会，但仍不改变玩家势力归属。

## 本刀目标

让玩家调查白家后，世界不只记住“事实已知”，还产生一个低风险、可解释的势力压力/机会条目。

## 已实现

- `resolveQingmaoVisibleInvestigation()` 在命中 `baijia_bai_ning_bing_public_talent` 可见事实卡时，额外生成：
  - `factionId = baijia_zhai`
  - `pressureType = opportunity`
  - `delta = 5`
  - `visibility = player_visible`
  - `reason = 对白家寨的可见范围调查形成接触机会；这不是势力归属变化。`
- `resolveVisibleInvestigationAction()` 会通过 `applyLivingWorldPatch({ source: 'living_world_engine' })` 写入 `factionPressure`。
- `actionConsequences.effectRefs` 会包含该 `factionPressure` id。
- e2e 摘要新增 `factionPressureCount/factionPressureIds`，用于浏览器回归断言。

## 明确没有做

- 不改 `currentFaction`。
- 不改 `selectedStartProfileId`。
- 不给白家身份。
- 不开放白家正式投靠。
- 不给声望、奖励、蛊虫、材料、地点或正史锚点变化。
