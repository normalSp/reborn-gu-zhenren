# v0.11.0-process-1 模板集

日期：2026-05-15
状态：第一版已建立。
用途：提供轻量模板，避免流程变成口头约定。

## ADR-lite 模板

适用：

- 存档策略。
- AI/DeepSeek 权限。
- 活世界状态结构。
- 自由意图裁决边界。
- 战斗/经济/奖励路线。
- 视觉资产路线。
- 外部依赖或子代理策略。

模板：

```md
# ADR-XXX 标题

日期：
状态：拟议 / 已接受 / 已废弃 / 被替代

## 决策

我们选择什么？

## 备选方案

- 方案 A：
- 方案 B：
- 不做：

## 后果

正面：

-

负面：

-

## 回滚或修正条件

什么情况下这个决策需要被重审？

-
```
