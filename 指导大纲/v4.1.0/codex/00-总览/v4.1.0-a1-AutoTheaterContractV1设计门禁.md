# v4.1.0-a1 AutoTheaterContractV1 设计门禁

状态：completed。
冻结标识：`v410_a1_auto_theater_contract_v1`。
日期：2026-05-26。

## 目标

把 v4.0 的 Auto-Theater 概念统一成后续 v4.x 共用工程合同。该合同是 report-only / design-gate 产物，不是 runtime schema、save schema 或 UI 组件。

## AutoTheaterContractV1

最小结构：

```ts
type AutoTheaterContractV1 = {
  contractVersion: 'v410_auto_theater_contract_v1';
  mode: 'report_only' | 'design_gate';
  preparation: PreparationEnvelopeV1;
  theaterLayers: TheaterLayerV1[];
  stackFrames: KillerMoveStackFrameV1[];
  ledgerEntries: CombatLedgerEntryV1[];
  expressionAuthority: ExpressionAuthorityV1;
  evidenceChain: WorldCoreEvidenceChainV1;
  boundaryAssertions: V410BoundaryAssertions;
};
```

### PreparationEnvelopeV1

只描述准备，不结算胜负：

- actor/ref：synthetic 或未来 WorldCore 角色引用。
- intentCandidate：候选战斗意图。
- resourceBudget：资源预算或风险说明，不写真实消耗。
- retreatThreshold：撤退阈值候选。
- counterRisk：反制风险候选。
- source：`synthetic_sample` / `worldcore_report` / `future_gate_required`。

禁止：

- 命中、伤害、奖励、NPC 生死、地点破坏、阵营变化。
- DeepSeek 直接生成结算。
- UI 私有公式。

### TheaterLayerV1

允许层：

- `air`
- `ground`
- `underground`
- `water`
- `aperture`
- `formation`
- `domain`
- `house`
- `environment`

每个 layer 必须包含：

- `layerId`
- `layerType`
- `status`: `candidate` / `fact` / `pressure` / `rejected` / `needs_user_decision`
- `evidenceRefs`
- `forbiddenImplications`

`fact` 只能来自 WorldCore / Combat Core / canon gate。v4.1 样本默认不产生 `fact`。

### KillerMoveStackFrameV1

允许阶段：

- `prepare`
- `charge`
- `lock`
- `release_candidate`
- `counter_window`
- `interrupt_candidate`
- `backlash_candidate`
- `resolved_fact`
- `rejected`

v4.1 只能测试 candidate / rejected / needs_user_decision，不产生 resolved runtime fact。

### CombatLedgerEntryV1

ledger lane：

- `fact`
- `candidate`
- `rejected`
- `needs_user_decision`
- `pressure`

每条 entry 必须有：

- `entryId`
- `lane`
- `summary`
- `evidenceRefs`
- `authority`
- `visibleToPlayer`
- `redactionPolicy`

hidden/private body 不得进入 ledger body。

## 完成门槛

- a1 文档冻结以上字段草案。
- b1 checker 必须覆盖 preparation、theater、stack、ledger 四大结构。
- negative fixtures 必须覆盖候选伪装事实、缺 evidence、UI/DeepSeek 私有结算。

## 边界

本门禁不生成 TypeScript runtime 类型，不接 Zod，不写 store/save，不接 UI/prompt。
