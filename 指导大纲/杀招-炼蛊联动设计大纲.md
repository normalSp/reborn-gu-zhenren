# 杀招-炼蛊联动设计大纲

> 版本: Phase 2 纲要 | 状态: 设计蓝图 | 日期: 2026-05-04

## 一、概述

杀招是蛊虫的组合使用方式，炼蛊是蛊虫的获取方式。两者天然联动：炼蛊产出蛊虫→蛊虫组装为杀招→杀招在战斗中生效。本大纲设计从"获得蛊虫"到"拥有杀招"的完整链路。

---

## 二、自创杀招配方系统

### 2.1 核心规则

自创杀招 = 核心蛊(1-3只) + 辅助蛊(1-5只) + 流派匹配 + 转数约束

| 要素 | 规则 |
|------|------|
| 核心蛊 | 必须是自有的高转数蛊虫，决定杀招流派和主要效果 |
| 辅助蛊 | 必须与核心蛊同流派或互补流派，决定杀招范围/增幅/特效 |
| 转数约束 | 杀招总转数 = max(核心蛊转数)，不能超过蛊师当前境界+1 |
| 流派匹配 | 核心蛊+辅助蛊中至少60%属于同一流派 |
| 独特性 | 同一配方只能被一个蛊师掌握（先创先得，后创者必须改良配方） |

### 2.2 配方发现流程

```
玩家选择核心蛊 → 系统推荐匹配的辅助蛊候选(从背包中筛选)
→ 玩家搭配 → 系统计算成功率(道痕+天赋+蛊虫契合度)
→ 尝试创招
  ├─ 成功 → 获得新杀招(名称自定或随机生成) + 获得流派经验
  └─ 失败 → 消耗尝试蛊虫的部分寿命 + 获得"失败经验"→下次成功率高5%
```

### 2.3 成功概率公式

```
基础成功率 = 炼道道痕×5% + 流派道痕×3% + 核心蛊契合度×20%
天赋加成 = 杀招天才(+20%) | 对应流派大师(+15%)
蛊虫契合度 = 1 - (辅助蛊转数差/核心蛊转数)²
最终成功率 = min(85%, 基础成功率 + 天赋加成)
```

## 三、杀招完整获取路径

### 3.1 已实现 (Phase 1)

- **角色创建福利**: TimelineConfig killermove步骤(蛊仙开局可选1-2条)
- **LLM叙事使用**: playerStateJSON 注入了 killMoves → LLM 感知
- **战斗主动使用**: CombatOverlay 杀招按钮 + 二级选择面板

### 3.2 待实现 (Phase 2)

#### A. NPC传授

- 好感度≥80时，同流派 NPC 可传授独门杀招(含原著专属杀招)
- 传授消耗：门派贡献度 + 元石(转数×50)
- 冷却：每 NPC 每 10 回合至多传授 1 招
- 事件触发：`state_update.kill_move.learn: ["杀招名"]`

#### B. 事件/探索获取

- 秘境探索：特定秘境隐藏杀招残卷
- 古遗迹：发现古人遗留的完整杀招配方
- 战斗缴获：击败特定 NPC 概率掉落其常用杀招的残卷
- 商店：极高价格(转数×200元石)出售基础杀招卷轴

#### C. 自创

- 拥有核心蛊+辅助蛊→尝试创招(见第二节)
- 自创杀招威力 = 公式计算(通常不如NPC传授的强大,但有成长空间)

## 四、流派专属杀招树

### 4.1 设计理念

每个流派有阶梯式杀招树，从基础到传说分级：

```
基础杀招(level 1-2): 流派入门即会,无需特别获取
进阶杀招(level 3-5): 需要道痕积累 + 特定蛊虫组合
高阶杀招(level 6-7): 需要仙蛊核心 + 流派大师级
传说杀招(level 8): 仅限专属/NPC传授/古方发现
```

### 4.2 各流派杀招树示例

**力道杀招树**:
- Lv1-2: 铁骨强化、重击
- Lv3-5: 山崩地裂、力拔山河、巨灵变
- Lv6-7: 天地同寿(自创)、金刚怒目
- Lv8: *(NPC专属/古方)*

**魂道杀招树**:
- Lv1-2: 搜魂、定魂
- Lv3-5: 魂爆、分魂
- Lv6-7: 魂归来兮(自创)、百鬼夜行
- Lv8: 万我(方源专属)

## 五、杀招熟练度与进化

### 5.1 熟练度系统

| 熟练度 | 使用次数需求 | 效果 |
|--------|------------|------|
| 入门 | 1-3 | 倍率-10% |
| 熟练 | 4-10 | 标准倍率 |
| 精通 | 11-30 | 倍率+10%，冷却-1 |
| 大师 | 31-80 | 倍率+20%，额外特效解锁 |
| 宗师 | 81+ | 倍率+30%，可传授他人 |

### 5.2 杀招进化

- 满足进化条件(修为/道痕/流派等级达标)时，杀招可升级转数
- 进化后：基础效果增强 + 可能新增辅助效果
- 进化消耗：大量同流派材料 + 元石 + 仙元(高阶)

## 六、数值平衡参考

### 6.1 自创杀招威力公式

```
基础伤害 = 核心蛊转数 × 10
流派加成 = 流派道痕 × 2
辅助蛊加成 = Σ(辅助蛊转数 × 1.5)
杀招总伤害 = (基础伤害 + 流派加成 + 辅助蛊加成) × 熟练度系数
```

### 6.2 NPC传授杀招定价

| 杀招等级 | 门派贡献度 | 元石 | 好感度要求 |
|---------|-----------|------|-----------|
| 1-2 | 10 | 50 | 30 |
| 3-4 | 50 | 300 | 50 |
| 5-6 | 200 | 1500 | 70 |
| 7-8 | 1000 | 8000 | 85 |

## 八、与炼蛊系统的联动总结

| 联动点 | 描述 |
|--------|------|
| 蛊虫→杀招组装 | 炼蛊产出蛊虫是杀招组装的原料 |
| 材料→杀招材料 | 炼蛊材料可用于优化杀招（材料强化杀招威力） |
| 杀招→炼蛊加速 | 特定杀招可加速炼蛊过程（如百炼成钢） |
| 炼蛊失败→杀招反噬 | 炼蛊失败可能触发已装备杀招的自动反噬 |
| 流派专属 | 同流派炼蛊成功率与杀招威力正相关 |
