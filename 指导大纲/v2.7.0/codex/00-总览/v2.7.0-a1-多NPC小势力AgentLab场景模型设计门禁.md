## Agent 层级边界

| 层级 | v2.7 用法 | 禁止越界 |
|---|---|---|
| L1 | 路人、短工、搬运、巡逻等聚合反应，只能输出 public pressure / routine hint | 不写私有动机，不形成正式关系，不写存档 |
| L2 | 小队成员、次要 NPC、被帮助或受骗者，只能输出 candidate memory / reflection candidate | 不 canonize 记忆，不写正式关系数值，不决定生死 |
| L3 | 当前场景关键 NPC，只能输出 intent / expression candidate | 不授予通行、身份、奖励、封锁、招揽或战斗后果 |
| L5 | 天道/宿命压力抽象导演，只能输出 macro pressure | 不裁决玩家结局，不变更正史锚点，不扮演普通 NPC |
