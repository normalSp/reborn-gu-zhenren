# v2.7.0 前置授权包

日期：2026-05-24
状态：pending user approval
适用版本：`v2.7.0`
主线：`低阶多 NPC / 小势力 Agent Lab 样板扩展`

## 授权包目标

本授权包用于把 v2.7 的审批前置化。用户批准后，Codex 可以在 `/goal` 中连续完成 v2.7 授权包内阶段，不再每个小版本重复等待常规审批。

本授权包建议批准一项有限工程动作：自有、零依赖、dry-run、report-only 的多 NPC / 小势力 Agent Lab runner 与 synthetic fixtures。它不读取真实 hidden/private body，不接 DeepSeek，不写 runtime/store/save/canon，不接外部框架，不启用 subagents。

## 编号说明

历史 `D-270` 已用于 v2.5 启动前决策。v2.7 采用 `D-271` / `F-271`，避免同号复用。

## 例外停机清单

触发以下任一条件，立即停止 `/goal` 自动推进：

- 需要任何 F-271 事项。
- runner 必须读取真实 hidden/private body、知识库正文、MiroFish raw output、原著正文、prompt body、玩家正式存档或 `.env` 才成立。
- runner/report 需要写 runtime/source/UI/store/save/canon 或 DeepSeek context。
- 需要引入外部依赖、框架、服务、队列、数据库、云存储或 subagent。
- CI/测试连续失败且需要改变方案。
- 文档或 runner 发现 hidden leak、formal authority drift、WorldCore bypass 等 P0/P1 且不能局部修复。
- 用户中途提出新方向或更高优先级。
