## 第一刀边界

v1.6 没有进入 runtime：

- 只读检查 `指导大纲/vMiroFish/基础包/`、`指导大纲/知识库/蛊真人/`、`指导大纲/vMiroFish/intake-reviews/`、当前版本测试矩阵。
- 输出报告，不自动修改知识库、不生成 runtime canon。
- 新脚本不进入 CI hard gate。

## 进入 runtime 前硬门

后续若出现 runtime 需求，必须重新经过：

1. 当前阶段设计门禁。
2. MiroFish topic-slice intake review。
3. 用户批准 runtime/canon/DeepSeek 可见边界。
4. focused tests + Player Advocate / drift gate。
5. `SAVE_FORMAT_VERSION` 影响评估。

## 当前结论

v1.6.0 本地开发已完成。它提供内容工厂地基，不提供内容晋升本身。下一步应开 v1.7 专家团启动会，决定第一个受控 topic-slice 或区域活世界测试目标。
