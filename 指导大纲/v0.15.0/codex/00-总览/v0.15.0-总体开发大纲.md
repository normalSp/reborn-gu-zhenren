# v0.15.0 总体开发大纲

日期：2026-05-18
状态：文档启动已建立
主题：`低阶蛊师经济、补给、炼养用深循环`

## 目标

让玩家在低阶蛊师阶段的长期行动受到真实资源压力约束：蛊虫要养，炼蛊会失败，残方不等于完整蛊方，补给和元石会影响路线，市场/商队只是机会而不是无限商店。

这不是全量经济系统，也不是开放完整交易所，而是先把“活世界”里的低阶生存成本接到已有路线、NPC/势力反应和自由目标系统上。

## 版本价值

v0.11 让玩家能表达目标。
v0.12 让青茅有正史和 IF 边界。
v0.13 让 NPC/势力开始记住玩家。
v0.14 让玩家的目标能承接到路线。
v0.15 要让路线、目标和行动受到蛊师经济与炼养用压力约束。

如果没有 v0.15，后续南疆人生会变成“选择路线 -> 文本推进”；有了 v0.15，玩家必须考虑怎么筹资源、怎么维护蛊虫、怎么承担失败、怎么避免被市场和势力反噬。

## 核心交付

1. 低阶经济/炼养用设计门禁。
2. 资源、蛊材、食料、残方、炼蛊失败、补给、市场窗口字段表。
3. MiroFish 三包的 RebornG-owned 候选规则池。
4. 补给准备或喂养缺口正式行动样板。
5. 炼蛊/残方/失败代价可读闭环。
6. 商队/市场窗口候选，不开放完整坊市或商家城。
7. 反刷、灰色交易、委托边界测试。
8. Player Advocate 20/60 轮走查。
9. rc 质量收束。

## 设计边界

### 可做

- 低阶资源、补给、食料、蛊材、残方、失败代价的 rule draft。
- 读取已有 `materialBag`、蛊虫、残方、路线缺口、行动账本、NPC/势力压力。
- 使用现有 v22 字段记录低风险行动结果。
- 显示“缺什么、为什么缺、能做什么、为什么不能直接交易/炼成/拿奖励”。
- 对奖励、材料、炼蛊和市场建立反刷规则与测试。

### 不做

- DeepSeek 直接发材料、蛊虫、蛊方、元石或交易结果。
- UI 私算价格、成功率、掉落、材料或炼蛊结果。
- 完整坊市、完整商家城、正式黑市或委托代售。
- 稳定套利、无限刷钱、无限刷材料。
- 高阶蛊虫、仙材、宝黄天、福地或蛊仙经济。
- 未批准的 save format bump。

## 完成标志

- 玩家能看懂低阶行动为什么受资源、补给、喂养、残方和市场窗口约束。
- 至少一个补给/喂养/资源缺口能从目标进入正式本地行动样板。
- 至少一个炼蛊/残方/失败代价链条可读、可测、不可越权。
- 商队/市场窗口能显示机会和风险，但不变成无限商店。
- 反刷规则覆盖同场景重复刷、奖励上限、DeepSeek 越权、黑市/委托延期边界。
- 不新增隐藏事实泄漏、正式地点/阵营/奖励/NPC 生死越权。
- v0.15 rc 质量门和 Player Advocate 60 轮通过。

## 风险

- 如果过早开放市场，会破坏低阶资源稀缺。
- 如果过早开放黑市/委托，会引入套利、诈骗、势力关注和任务奖励网络。
- 如果直接做完整二转/三转经济，范围会失控。
- 如果新增持久字段但没有迁移，旧档风险会上升。
- 如果 DeepSeek 写入奖励或价格，会破坏本地引擎权威。
