# v0.15.0-b4 灰色交易/委托边界样本

日期：2026-05-18
状态：已完成并推送，远端 CI 通过

## 定位

`v0.15.0-b4` 把已通过 intake 的黑市/委托边界包降级吸收为 RebornG-owned 风险边界样本：

`商队/市场窗口候选 -> 试探灰色交易/委托边界 -> 写入既有 livingWorldState 字段 -> 回流自由目标面板`

本阶段不是正式黑市系统，不是正式委托代售系统，也不是灰色交易库存或价格系统。它只让玩家知道：黑市传闻、委托代售、假货、陷阱、势力关注和稳定套利都属于高风险后续系统，当前只能作为延期边界和测试样本。

## 边界

明确不做：

- 不开放正式黑市。
- 不开放正式委托、代售、分成或收益。
- 不开放灰色交易库存、价格表或商品列表。
- 不判定买入、卖出、交易成功、被骗、被抓或反杀。
- 不写 `materialBag`、库存、货币、蛊虫、蛊方或奖励。
- 不写正式势力压力、通缉、声望、身份洗白、阵营转移或正式任务。
- 不改变地点、路线进入、NPC 生死或抓捕结论。
- 不新增持久字段，不提升 `SAVE_FORMAT_VERSION`。
- 不扩张 DeepSeek 权限。
