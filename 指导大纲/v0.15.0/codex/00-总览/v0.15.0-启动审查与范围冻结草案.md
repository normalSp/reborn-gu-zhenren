## 范围冻结

v0.15 主线冻结为：

`低阶蛊师经济、补给、炼养用深循环`

核心范围：

- 低阶蛊材、食料、残方、炼蛊失败代价。
- 补给准备、喂养缺口、市场窗口、商队递话。
- 反刷、奖励边界、DeepSeek 越权阻断。
- 玩家可读的资源压力和下一步提示。

## 明确非目标

本版本默认不做：

- 完整南疆开放世界。
- 完整坊市、商家城、黑市、委托代售系统。
- 正式商品价格表或动态经济模拟。
- 二转/三转完整成长曲线。
- 正式阵营迁移、通缉、招揽、任务网络或奖励网络。
- NPC 生死、捕获、追杀结论。
- 新 save fields 或 save format bump。
- DeepSeek reward/material/recipe/location/faction authority。

## Stop Gates

遇到以下任一情况必须停下来让用户决策：

- 新增持久字段或提升 `SAVE_FORMAT_VERSION`。
- 新增正式奖励、材料发放、元石收益、蛊虫、蛊方或稳定掉落。
- 正式开放市场购买、坊市、黑市、委托代售、商家城交易。
- 建立正式价格表、库存表或动态经济系统。
- 改变地点、阵营、声望、通缉、NPC 生死或正史锚点。
- 给 DeepSeek 新增经济、奖励、材料、炼蛊、市场或任务写入权。
- 引入后端/BFF、外部 runtime 依赖、EdgeOne 自动部署或 branch protection。
