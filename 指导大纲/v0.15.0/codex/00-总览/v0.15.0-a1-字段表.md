日期：2026-05-18

## 候选规则字段

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | string | RebornG-owned 稳定 id |
| `category` | enum | `material_candidate` / `feeding_requirement` / `refinement_fragment` 等 |
| `summary` | string | RebornG 重写摘要，不复制 MiroFish 原文 |
| `sourceRefs` | string[] | 指向 MiroFish sourcePointerIds 或项目 fact/card/rule ids |
| `visibility` | enum | `public` / `limited_public` / `hidden_ref_only` |
| `allowedUse` | string[] | `candidate_pool` / `rule_draft` / `test_sample` / `deferred` |
| `blockedUse` | string[] | 禁止 runtime truth、DeepSeek authority、formal reward 等 |
| `authorityOwner` | enum | `canon` / `engine` / `store_action` / `ui_readonly` |
| `rewardPolicy` | enum | `none` / `candidate_only` / `local_engine_capped` |
| `antiFarmPolicy` | string | 反刷策略摘要 |
| `saveImpact` | enum | `none` / `uses_existing_v22` / `requires_save_decision` |

## Store 写入口径

默认优先：

- 只读 preview。
- 写现有 v22 `livingWorldState` 的 `knownFacts`、`actionConsequences`、`factionPressure`、`npcMemories`、`localActionLedger`。
- 使用既有 `materialBag` 只在明确通过本地 engine 和测试后。

默认不新增：

- `economyState`。
- `marketState`。
- `feedingSchedule`。
- `refinementHistory`。
- `tradeLedger`。

若必须新增，触发 save format 决策。

## DeepSeek 可见字段

DeepSeek 可见：

- prompt-safe 已知事实 id。
- 候选行动摘要。
- 可见风险。
- 缺口解释。

DeepSeek 不可见：

- hidden fact body。
- MiroFish hidden/ref 原始内容。
- 正式价格表。
- 可执行奖励表。
- 未注册材料/蛊方/蛊虫。
- 本地 engine 私有成功率或反刷阈值。
