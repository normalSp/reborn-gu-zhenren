# v0.15.0-b1 补给/喂养缺口行动样板

日期：2026-05-18
状态：本地完成，待提交/推送/远端 CI

## 定位

`v0.15.0-b1` 把 a2 低阶经济规则池里的路线补给、落脚遮掩、酒虫食料压力，升级成一个玩家可执行的本地前置行动：

`离开青茅山目标 -> 路线准备/路线承接上下文 -> 整理补给与喂养缺口 -> 写入既有 livingWorldState 字段 -> 回流自由目标面板`

本阶段不是材料系统、市场系统或喂养库存系统。它只把玩家下一步真正缺什么讲清楚，并留下可测试、可回滚的行动账本。

## 阻断条件

必须先满足：

- 有离开青茅山长期目标，或其 rationale 明确是逃离青茅山。
- 已有路线准备、遮掩痕迹或山路候选承接上下文。
- a2 规则池能提供补给和食料规则草案。

缺少条件时只返回阻断结果，不写 store。

## 边界

明确不做：

- 不发材料。
- 不扣或发放元石。
- 不开放完整市场、黑市或委托交易。
- 不写 `materialBag`、库存、正式价格、商店库存或喂养库存。
- 不判定路线进入、逃离成功、地点解锁或区域解锁。
- 不改变阵营、声望、通缉、正式任务或 NPC 生死。
- 不新增持久字段，不提升 `SAVE_FORMAT_VERSION`。
- 不扩张 DeepSeek 权限。

## 玩家可见反馈

`FreeGoalPanel` 新增 `补给/喂养缺口` 区块。玩家在完成逃离目标、路线准备/承接后，可以点击 `整理缺口`。

面板文案强制显示：

- 不发材料。
- 不扣元石。
- 不开放市场。
- 不判定离山成功。
