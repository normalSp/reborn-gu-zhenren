# v3.0.0-b3 同开局差异、NPC 意图与势力压力复核

日期：2026-05-24
状态：completed
冻结模型：`v300_b3_same_start_npc_intent_faction_pressure_review_v1`

## 复核目标

v3.0 不是为了让同开局随机乱跳，而是让未来 L2/L3 agent 在 WorldCore 约束下提供可见、可审计、可回滚的候选差异。

## 同开局差异边界

允许：

- 语气、短期意图、注意力焦点、可解释的压力候选。
- 不改变事实的表达差异。
- 进入报告的候选差异。

禁止：

- 候选直接写事实。
- 候选直接写存档。
- 以随机数替代世界因果。
- 未经审批加入 `runFingerprint`。

## NPC 意图边界

L2/L3 可以提议：

- 低阶 NPC 的短期谨慎、怀疑、试探、避险。
- 小势力边缘人物的压力与行动候选。
- 小队成员的表达与提醒候选。

L2/L3 不能决定：

- NPC 生死、背叛、正式关系、招揽、通缉、封锁。
- 原著关键人物动机。
- 天道/宿命判断。

## 势力压力边界

未来可作为候选：

- 小势力注意力变化。
- 临时盘查压力。
- 劳役、交易窗口、路途压力的表达候选。

仍需 future gate：

- 正式势力状态。
- 正式敌对/结盟。
- 正式地点、奖励、身份。

## b3 结论

`v300_b3_same_start_npc_intent_faction_pressure_review_v1` 成立。v3.0 保留同开局差异的工程方向，但本版本不写 runtime。
