# v2.8.0-a2 agent capability classification matrix 门禁

日期：2026-05-24
状态：completed
冻结版本：`v280_a2_agent_capability_classification_matrix_v1`

## 分类

| classification | 含义 | 是否可进 v3.0 设计讨论 |
|---|---|---|
| `admissible_candidate` | 可讨论进入 v3.0 小区域试点，但仍只输出候选 | 是 |
| `lab_only` | 继续留在 Agent Lab / report-only / local-rule 研究 | 否 |
| `rejected` | 当前路线拒绝，除非未来单独改路线 | 否 |

## Capability Matrix

| 能力 | 默认分类 | 理由 |
|---|---|---|
| L2 event reflection candidate | `admissible_candidate` | 可增加 NPC 连续性；必须 TTL + audit + WorldCore post-check |
| L2 squad long-goal candidate | `admissible_candidate` | 可提升队友感；不得写关系数值或 save |
| L2 social follow-up candidate | `admissible_candidate` | 可让 NPC 记得公开压力；不得转正式关系 |
| L3 current-scene expression | `admissible_candidate` | 可增强同开局差异度；只改表达，不写 outcome |
| L3 current-scene intent | `admissible_candidate` | 可让关键 NPC 更像人；必须由 WorldCore 裁决 |
| L1 routine behavior | `lab_only` | 用行为树/效用规则更稳，不需 live LLM |
| L2 small-faction pressure | `lab_only` | 未有正式社会状态前，不能进入 runtime |
| self-learning candidate patch | `lab_only` | 可研究，不得自改项目 |
| L5 macro pressure report | `lab_only` | 可研究天道/宿命压力，不接 runtime |
| L4 original key character live | `rejected` | 原著关键人物必须 future gate |
| L5 Fate/Heaven verdict | `rejected` | 不得裁决玩家结局或正史锚点 |
| Direct save/runtime/canon write | `rejected` | WorldCore 之外不可写 |
| DeepSeek visible lore/RAG | `rejected` | 需要未来单独授权 |
| backend/external framework/subagent shortcut | `rejected` | F-280 future gate |

## v3.0 设计输入

v3.0 只可考虑：

- L2 private memory / reflection / long-goal candidates。
- L3 current-scene expression / intent candidates。
- 一个小区域、少量 NPC、强 WorldCore post-check。

v3.0 不可默认考虑：

- L4/L5 runtime。
- self-learning runtime。
- 正式 faction/social state write。
- external framework PoC。
