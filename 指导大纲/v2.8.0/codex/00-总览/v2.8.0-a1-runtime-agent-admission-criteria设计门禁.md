# v2.8.0-a1 runtime agent admission criteria 设计门禁

日期：2026-05-24
状态：completed
冻结版本：`v280_a1_runtime_agent_admission_criteria_v1`

## 定位

v2.8-a1 定义未来 v3.0 小区域 runtime agent 试点的准入标准。它只回答“哪些能力有资格进入 v3.0 设计讨论”，不批准 agent 进入 runtime。

## 总原则

- WorldCore / local core 拥有最终事实、位置、奖励、数值、战斗、NPC 生死、正史锚点和存档写入权。
- Agent 只能输出 `AgentProposal`。
- `AgentProposal` 必须可拒绝、可撤销、可审计、有 TTL、有来源 refs、有 WorldCore post-check。
- DeepSeek 只可作为未来表达/候选来源，不可写事实、奖励、地点、身份、结局或存档。
- v2.8 不调用 live DeepSeek。

## L1 准入标准

结论：`lab_only` 或 local-rule-only。

理由：

- L1 是路人、商贩、巡逻、兽群、队列等低价值高频行为。
- 最适合行为树、效用规则、局部随机表，不值得接 live LLM。
- 若未来进入 runtime，也应作为 local deterministic routine，而不是 runtime agent。

## L2 准入标准

结论：可进入 `admissible_candidate`，但只限候选。

允许：

- 事件后反思候选。
- 小队成员长期目标候选。
- 私有记忆候选。
- 社会后续问题候选。

必须满足：

- `authority = proposal_only`。
- `requiresWorldCorePostCheck = true`。
- 不写 save/runtime/canon/prompt/knowledge-index body。
- 不形成正式关系数值、正式身份、正式势力、通缉/招揽/封锁。
- 有 TTL 和 audit refs。

## L3 准入标准

结论：可进入 `admissible_candidate`，但只限当前场景表达和意图。

允许：

- 当前场景关键 NPC 语言表达候选。
- 当前场景关键 NPC 意图候选。
- 对可见条件、可见压力、下一步提示的表达变化。

不允许：

- 直接写 outcome。
- 直接决定交易、奖励、地点、身份、战斗结果、NPC 生死。
- 解释原著关键人物真实动机或隐藏事实。

## L4 准入标准

结论：`rejected` / `future_gate_required`。

L4 原著关键人物和大势力核心需要 source pointer、私有记忆、强审计和用户决策。v2.8 不允许 L4 live agent 或 L4 runtime candidate。

## L5 准入标准

结论：`lab_only` 或 `rejected`。

- report-only macro pressure 可以作为长期路线研究材料。
- 天道/宿命不得裁决玩家结局。
- 不得改变正史锚点。
- 不得写入 runtime 或 DeepSeek visible context。

## Self-learning 准入标准

结论：`lab_only`。

允许生成候选 patch summary 或 manual review note。禁止直接改 canon/runtime/save/prompt/knowledge-index body/expert skill/lore skill。

## WorldCore post-check 必选项

每个可准入 L2/L3 proposal 至少检查：

- 是否只含候选/表达/意图。
- 是否未写事实、奖励、地点、身份、NPC 生死。
- 是否未暴露 hidden/private body、prompt body、原著正文或 MiroFish raw output。
- 是否未扩大 DeepSeek authority。
- 是否可撤销、有 TTL、有 audit refs。

## 直接拒绝项

- runtime/save/canon/prompt/knowledge-index body write。
- DeepSeek visible lore/RAG。
- backend/BFF/service/job queue/cloud save。
- external framework PoC/dependency/subagents。
- MiroFish export/intake 或真实原著事实吸收。
- L4 live / L5 verdict。
- formal location/faction/identity/reward/NPC life-death。
