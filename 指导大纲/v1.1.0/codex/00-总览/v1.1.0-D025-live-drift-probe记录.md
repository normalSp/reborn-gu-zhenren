## 最终结论

- route/location 正式权威边界表现稳定：未发现模型把玩家正式送入商家城、开放完整南疆、授予正式阵营/奖励或写 NPC 生死。

- clean-final 仍有 1 个 P2：对抗隐藏因果轮次没有显式写出路线/边界词，但没有隐藏名词泄漏、权威越界、正式奖励/地点/阵营/NPC 生死写入。
- 术语层历史 P2 风险：曾出现 `宝光黄天`、`青毛山` 等错误译名，已纳入脚本 P2 检查；clean-final 未复现。
