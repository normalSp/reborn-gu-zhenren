## 字段表

`routeLocationState` 是 v1.1 的唯一新增持久路线/地点对象。它只表达粗粒度范围，不是完整地图、地点解锁或剧情结论。

| 字段 | 类型 | 默认 | allowlist / 约束 | 说明 |
|---|---|---|---|---|
| `status` | string | `not_started` | `not_started`、`preparing_departure`、`route_in_progress`、`outer_edge_projection`、`blocked` | 路线阶段，不代表成功、奖励或阵营结果 |
| `routeId` | string/null | null | null、`southern_border_low_rank_route`、`qingmao_exit_route` | 当前路线候选，不代表抵达 |
| `locationScopeId` | string | `qingmao_mountain` | `qingmao_mountain`、`qingmao_exit_path`、`southern_border_outer_edge`、`unknown_conservative` | 地点范围，不是正式地点解锁 |
| `regionScopeId` | string | `qingmao` | `qingmao`、`southern_border_outer_edge`、`unknown_conservative` | 区域范围，不是完整区域开放 |
| `authority` | string | `start_profile` | `start_profile`、`living_world_engine`、`route_location_engine` | DeepSeek/UI/narrative text 不可写 |
| `evidenceLedgerEntryIds` | string[] | [] | 只存 id，不存正文 | 迁移和更新依据 |
| `sourceRefs` | string[] | [] | 只存 RebornG/MiroFish/知识库安全指针 | 不是 runtime 权威本体 |
| `lastUpdatedAtTurn` | number/null | null | 非负整数或 null | checkpoint 与 drift audit 用 |
| `migrationNote` | string/undefined | undefined | 仅内部说明 | 异常旧档或保守回落原因 |

## 迁移矩阵

| 旧档识别 | v23 结果 | 证据 | 禁止推断 |
|---|---|---|---|
| 无离山目标、无路线事实、无相关账本 | `not_started` / `qingmao_mountain` / `qingmao` | save default | 不推断玩家已离开 |
| 有 `region:outside_qingmao` 目标或离山 rationale | `preparing_departure` / `qingmao_mountain` / `qingmao` | goal id | 不承认路线准备已完成 |
| 有 `qingmao_escape_route_preparation_baseline` 或路线准备 consequence | `preparing_departure` + `southern_border_low_rank_route` | fact/consequence id | 不承认抵达南疆 |
| 有 v0.18 门槛/压力事实 | `route_in_progress` / `qingmao_exit_path` / `qingmao` | v018 fact id | 不写完整南疆 |
| 有 v0.18 candidate continuation 或 v1.0 continuity acceptance | `outer_edge_projection` / `southern_border_outer_edge` / `southern_border_outer_edge` | v018/v100 fact id | 不开放商家城或正式地点 |
| 字段类型坏、id 越界、证据冲突 | `blocked` 或 `unknown_conservative` | migrationNote | 不黑屏、不误解锁 |

迁移必须同时满足：

- 不创建 `route_entered`、`currentRoute`、`currentRegion`、`location_unlock` 等旧式或越权字段。
- 不写奖励、阵营、NPC 生死、商家城正式进入。
- 不把 hidden/private source pointer 展示给玩家或 DeepSeek。
