## 范围冻结建议

v1.1 范围冻结为：

`route/location/save-format 设计门禁 -> MiroFish intake -> 字段表/迁移矩阵 -> 小范围正式路线状态 -> 区域状态可读 UI -> 前期账本压力回流`

并在 runtime 前插入：

`长线叙事漂移门禁 -> 全书知识库治理门禁 -> 历史索引与入口压缩 -> skill 同步 -> 流程减法审查`

冻结后不纳入：

- 完整南疆、完整商家城、完整五域。
- 蛊仙期、宝黄天正式交易、尊者线、任意时代开局。
- 正式阵营转移、奖励、NPC 生死。
- 完整经济库存/市场系统。
- BFF/backend、云存档、账号系统。
- 自动部署或公开发布承诺。
- live DeepSeek drift probe 默认进入 CI 或未经批准执行。
- 全书知识库直接进入 runtime canon、DeepSeek visible context 或玩家可见隐藏事实。
