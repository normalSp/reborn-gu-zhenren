## Future Sample Pool

- 玩家要求改名换身份混入白家或商队。
- 玩家主动暴露青茅秘密换取庇护。
- 玩家长期逃避追踪导致区域势力误判。
- 玩家旧档已有多个逃离目标，迁移后如何聚合。
- 玩家在路线中发生战斗失败，是否影响地点状态。
- 玩家在外缘投影状态下长期反复尝试进入商家城，是否触发新测试样本而不是直接开放地点。
- 外部参考映射样本：ReAct 链路断裂时，玩家自由意图是否被 DeepSeek 直接写成事实。
- 外部参考映射样本：Reflexion 回流失败时，Player Advocate 困惑是否没有进入测试矩阵。
- MiroFish intake 样本：`human_review` 与 `hidden_ref_only` 项是否被错误展示给 UI 或 DeepSeek。
- 长线叙事样本：玩家 150 回合后仍围绕同一逃离路线推进，DeepSeek 是否开始空转或改写地点。
- 知识库样本：全书基础包中同一人物/地点/事件多条来源冲突时，如何进入 `deferred` 而不是 runtime。
- 流程减法样本：旧流程文件仍被 skill 引用时，不能只因“过长”就标废弃。
