# v3.8.0-a1 proposal graph stability model设计门禁

日期：2026-05-25
状态：completed
冻结版本：`v380_a1_long_horizon_proposal_graph_stability_model_v1`

## 目标

在 v3.7 proposal graph 上增加长期稳定性模型，用于检查：

- 多小势力压力源是否仍为公开压力候选。
- candidate/rejected/expired/needs_user_decision 是否保持 lane 状态，不污染事实链。
- same-start variation 是否来自 deterministic proposal lane，而非存档种子或隐藏随机指纹。
- WorldCore 是否仍是最终裁决者。

## 模型边界

`v380` 模型只允许读取已有 projection 输入：

- `turn`
- `previewLane`
- `regionalEventLedger`
- `routeLocationState`
- `livingWorldState`
- `localActionLedger`

禁止读取或写入：

- 新 store slice。
- 新 save field。
- `runFingerprint`。
- DeepSeek prompt/context。
- MiroFish base package 正文。
- runtime canon / knowledge-index body。
- backend/BFF/job queue。

## 冻结接口

实现接口：

- `src/engine/v380-transient-agent-proposal-graph-stability.ts`
- `buildV380TransientAgentProposalGraphStability(input, options)`
- `runV380DeterministicProbe(roundCount = 150)`

冻结常量：

- `V380_PROPOSAL_GRAPH_STABILITY_MODEL_VERSION = v380_a1_long_horizon_proposal_graph_stability_model_v1`
- `V380_MULTI_PRESSURE_COPY_GATE_VERSION = v380_a2_multi_pressure_same_start_copy_gate_v1`
- `V380_TRANSIENT_PROPOSAL_GRAPH_STABILITY_VERSION = v380_b1_transient_agent_proposal_graph_stability_v1`
- `V380_DETERMINISTIC_ROUNDS = 150`

## 必须输出

- inherited v3.7 graph evidence。
- pressure stability lanes。
- multi-pressure summary。
- memory contamination audit。
- deterministic probe result。
- Player Advocate metadata。
- old-save/no-save/rollback evidence。
- boundary lines。

## 通过标准

- 所有 lane 均为 synthetic/generic。
- lane 必须显式标注候选不是事实。
- `factsCommitted/saveWritten/promptWritten/canonPromoted/formalStandingWritten/npcLifeDeathWritten/rewardGranted` 均为 `false`。
- deterministic gate 至少 150 轮，acceptedForGate 必须为 `true`。
- unique stability signatures 至少 12。
- 不新增 save field，不 bump `SAVE_FORMAT_VERSION = 25`。

## 结论

a1 通过。可进入 a2 copy gate。
