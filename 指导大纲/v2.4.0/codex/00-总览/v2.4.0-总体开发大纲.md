## 设计原则

- WorldCore 保持最终裁决。
- BFF 是基础设施，不是世界裁判。
- Agent 输出候选，不写事实。
- DeepSeek 输出叙事和候选，不写 ledger/canon/save。
- private canon 不进入 DeepSeek visible context。
- self-learning 只产候选，不自裁决。
- 外部 framework 只通过 RebornG-owned adapter 输出 `AgentProposal` / report。

## 技术边界

当前阶段不实现：

- API proxy。
- private canon service。
- job queue。
- cloud save。
- external framework adapter runtime。
- live eval service。

v2.4 若后续进入 b1-b3，也优先做 report-only / schema-only / dry-run-only。
