## rc 硬边界

本 rc 不授权：

- runtime/source/UI/store/prompt/save 变更。
- 新 save fields、`SAVE_FORMAT_VERSION` bump、`runFingerprint`。
- backend/BFF/API proxy/private canon service/eval archive service/job queue/cloud save implementation。
- runner、service、artifact、migration、历史 report 回填。
- live DeepSeek、DeepSeek prompt/context/model/authority 变更。
- subagents。
- external framework PoC、dependency、vendored subset、read-only scan、patch artifact、项目读写、命令执行、git 操作。
- MiroFish export/intake。
- 知识库正文、runtime canon、DeepSeek visible lore/RAG。
- 真实 hidden/private body、原著正文、MiroFish raw output、prompt body archival。
- 正式地点、阵营、身份、奖励、NPC 生死或 canon promotion。
- EdgeOne 自动部署。
