## Trace Schema

v0.17 不新增正式 trace 存档字段，只读取既有 `BattleResolutionStep`：

- `round`：回合。
- `kind`：行动类型，例如 `gu_use`、`move`、`counter`、`formation`。
- `actorId/targetIds`：行动者与目标。
- `sourceName`：蛊虫、杀招或战术来源。
- `message`：玩家可读解释。
- `tags`：用于测试和 UI 归类。

## 回流 Schema

战后回流仍以既有 `BattleOutcomeSummary` 为准：

- `result/winner/roundsTaken`：胜负与回合。
- `hpDelta/essenceDelta`：本地损耗。
- `steps`：最后若干可读轨迹。
- `beastLoot`：本轮 v0.17 禁用，不设置 `enemySpecIds`。
