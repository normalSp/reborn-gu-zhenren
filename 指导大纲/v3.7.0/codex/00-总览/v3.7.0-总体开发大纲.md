# v3.7.0 总体开发大纲

日期：2026-05-25
状态：approved；pending `/goal` implementation

## 目标

在不开放完整 runtime agent society 的前提下，让 `意图` tab 从 v3.6 的 synthetic L2/L3 多 lane 进一步进入 multi-NPC / small-faction proposal graph 复核。

核心不是让 NPC “真实存在”，而是让多个候选视角围绕同一公开事件形成不同压力，并证明这些压力仍由 WorldCore 管住。

## 默认实现形态

用户已批准 D-370；后续 runtime 第一刀建议新增：

- `src/engine/v370-transient-agent-proposal-graph.ts`
- `buildV370TransientAgentProposalGraph(input)`
- `runV370DeterministicProbe(120)`
- `V370_TRANSIENT_AGENT_PROPOSAL_GRAPH_VERSION`

UI 仍复用世界面板 `意图` tab，不新建大系统入口。

## 非目标

v3.7 不是完整 NPC 社会、不是持久记忆、不是真实势力系统、不是 live DeepSeek agent，也不是后端或外部框架专项。
