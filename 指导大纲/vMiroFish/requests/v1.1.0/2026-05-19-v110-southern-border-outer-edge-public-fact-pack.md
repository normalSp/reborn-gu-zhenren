日期：2026-05-19
请求方：RebornG Expert Council
目标版本：v1.1.0
优先级：blocking before b2/b4 public region UI

## 目的

请为 RebornG v1.1 提供“南疆早期外缘公开事实”候选资料包。

本包用于支持玩家离开青茅山后，在不开放完整南疆/完整商家城的情况下，看到可信的区域外缘信息。

## 范围

重点关注：

- 山路、商队、散修、低阶城外缘、资源点、危险区域。
- 低阶蛊师在南疆外缘可能看到的公开压力。
- 可用于 UI 的公开事实摘要。
- 不应玩家可见的隐藏事实或重大正史因果。

## 输出要求

每项建议包含：

- `itemId`
- `category`
- `visibility`
- `publicSummary`
- `sourcePointers`
- `playerVisibleBoundary`
- `hiddenBoundary`
- `riskTags`
- `recommendedRebornGUse`

## 禁止

- 不要包含原文正文或长引用。
- 不要输出完整商家城/完整南疆开放承诺。
- 不要输出正式奖励、阵营、NPC 生死或高阶传承结论。
- 不要把 MiroFish 候选内容写成 canon truth。

## 交付后 RebornG 的处理方式

RebornG 会做 intake review，只把合格内容改写为 public fact draft、rule draft 或 test sample。
