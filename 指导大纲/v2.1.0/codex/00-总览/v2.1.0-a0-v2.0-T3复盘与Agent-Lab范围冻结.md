## T3 暴露的 v2.x 缺口

| 缺口 | 说明 | v2.1 处理 |
|---|---|---|
| formal credential 越界倾向 | DeepSeek 会自然写出通行、准入、身份、名单、登记类正式暗示 | Agent Lab eval 必须继承 formal credential hardening |
| hidden/private echo | adversarial input 仍是最危险样本族 | AgentProposal 必须区分 public summary 与 private refs |
| schema gap | narrative、boundary、local-engine、safe-next-step 必须结构化 | Agent Lab 设计 strong schema |
| 样本覆盖有限 | v2.0 只覆盖第一核心小区域五类入口 | Agent Lab 先做场景族，不声称完整南疆 |
| 成本与重试观测 | retry 成本低但必须可审计 | Agent Lab 记录 token、cache、cost、retry、failure |
