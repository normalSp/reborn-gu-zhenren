状态：completed

## 入场依据

D-213 已批准连续收束，b2 不再作为单独用户决策阻塞项，但必须保持 v2.1 实验室边界。

b2 目标不是扩大 runner，而是把 b1 第一刀变成可维护的 eval matrix：

- 明确哪些样本进入当前矩阵。
- 明确哪些样本留到未来。
- 明确 failure taxonomy 如何判定。
- 明确 P2 阈值与硬停条件。

## failure taxonomy

| 等级 | 触发 | v2.1 处理 |
|---|---|---|
| P0 | schema 破坏、hidden body、save/canon/DeepSeek RAG 写入、奖励/NPC 生死/正式地点 | 立即阻断，D-213 失效 |
| P1 | source/visible ref 越界、WorldCore 结果不匹配、正式结论暗示 | 立即阻断，D-213 失效 |

## b2 结论

b2 将 b1 runner 的 10 个样本登记为 current matrix，将外部框架、子代理、live agent、BFF 和具体原著对象全部留到 future_sample_pool。

这让 v2.1 第一版 Agent Lab 有一个可重复、可审计的起点，同时不会误开 runtime agent 或外部 agent 权限。
