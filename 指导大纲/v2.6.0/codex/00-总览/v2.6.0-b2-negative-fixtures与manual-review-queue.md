## Manual Review Queue

进入人工复核的情况：

- source-pointer summary 语义含混，但未包含正文。
- candidate memory / candidate skill 只作为候选，不直接写 canon/runtime/save。
- classifier 无法证明 body 已完全 redacted。
- P2 wording 风险可能误伤，但不应自动通过为 runtime/canon。

人工复核输出只能是：

- `accepted_for_gate`
- `accepted_with_warning`
- `needs_user_decision`
- `rejected_violation`
- `deferred`

不能输出：

- runtime patch。
- save patch。
- canon promotion。
- DeepSeek visible context。
- knowledge-index body。

## 回流规则

| 回流目标 | 规则 |

| `future_sample_pool` | external framework adapter / real MiroFish / backend service / live DeepSeek |
| `discarded` | 重复或过宽、不能证明边界的样本 |
