## 测试目标

v1.0 的测试目标不是单纯“功能不报错”，而是验证：

1. 玩家能看懂自己在蛊真人风格世界里能做什么、不能做什么。
2. 自由意图不会绕过本地裁决。
3. DeepSeek 不会获得正式奖励、地点、阵营、NPC 生死、隐藏事实权限。
4. 低阶蛊师 life loop 能形成可理解循环。
5. 公开版不泄露 hidden/private canon，不承诺未开放内容。

## Player Advocate 分层

rc 150 轮建议分配：

- 30 轮新档常规路线。
- 20 轮旧档回流。
- 20 轮移动端/窄屏。
- 30 轮自由意图与极端意图。
- 20 轮低阶蛊师 life loop。
- 20 轮公开文案/截图理解。
- 10 轮随机混合压力样本。

通过标准建议：

- 下一步理解率 >= 90%。
- P0/P1 玩家阻塞为 0。
- 隐藏事实泄露为 0。
- 正式奖励/地点/阵营/NPC 生死越权为 0。
- UI 关键入口遮挡/误导为 0。
