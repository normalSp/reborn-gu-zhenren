# v1.0.0-a1 接受标准与权威冻结

日期：2026-05-19
状态：已完成第一版

## 目标

冻结 v1.0 作为“活世界早期正式版”到底承诺什么、不承诺什么，以及每类事实由谁负责写入。

## v1.0 对玩家的承诺

v1.0 可以承诺：

1. 青茅山到南疆早期路线有连续体验与下一步提示。
2. 玩家自由目标会被本地系统裁决，能看到前置条件、拒绝、延期、风险或降级理由。
3. 低阶蛊师的修行、补给、炼养用、战斗、交易/委托和社会压力有可读闭环。
4. NPC/势力/路线压力会记住玩家行为，并以候选行动、公开摘要、压力提示或风险说明回流。
5. DeepSeek 参与叙事，但不能替本地系统决定奖励、地点、阵营、NPC 生死、正史事实或隐藏事实。
6. 公开素材只描述 v1.0 已开放或可安全暗示的低阶范围。

v1.0 不能承诺：

1. 完整蛊界、完整五域、完整南疆或完整商家城。
2. 蛊仙期、仙蛊、仙材、宝黄天正式交易、尊者线、宿命大战。
3. 任意时代开局、任意人物附身、盗天魔尊时代开局。
4. 玩家任意目标都可成功。
5. 正式加入强势阵营、决定关键 NPC 生死或绕开正史核心因果。
6. 完整后端、云存档、自动部署或公开运营级观测。

## 发布接受标准

| 维度 | v1.0 接受标准 |
|---|---|
| 路线连续性 | 玩家能从青茅后续压力进入南疆早期候选路线，能理解为何能走或不能走 |
| 自由意图 | 正常、短期、长期、极端目标均有本地裁决与测试样本 |
| 低阶闭环 | 修行/补给/蛊虫/杀招/战斗/交易/委托至少形成释出版可读循环 |
| NPC/势力 | 行动影响能回流为记忆、压力、公开摘要、候选行动或风险 |
| UI 减法 | 正式入口不暴露大量开发、高阶、未开放按钮 |
| 世界观 | 隐藏事实、正史核心因果、高阶规则不泄露、不越权 |
| DeepSeek | 仅生成叙事/候选/线索/传闻/压力，不写正式事实 |
| 安全 | 不公开共享 API key、不下发完整 private/hidden canon、不自动发布 |
| 测试 | rc 150 轮 Player Advocate，自动化质量门、长测、旧档、移动端、公开 copy 检查通过 |

## 权威冻结表

| 内容 | 权威来源 | v1.0 写入权限 |
|---|---|---|
| 正式奖励、材料、蛊虫、完整蛊方 | local canon/engine | DeepSeek/UI/MiroFish 不可写 |
| 正式地点/路线状态 | local engine/store，经用户批准的 save protocol | 新持久字段需单独停手 |
| NPC 生死/抓捕/关键命运 | local canon/engine + 用户决策 | v1.0 不自动开放 |
| 阵营身份/正式加入/通缉/招揽 | local social/faction engine + 用户决策 | v1.0 仅展示前置条件和压力，除非另批 |
| 隐藏事实 | local hidden refs/private canon | 不进入 UI/DeepSeek visible context |
| 正史硬事实 | fact cards/canon anchors/engine | MiroFish 只候选，需 RebornG 重写 |
| IF 偏离代价 | local IF matrix/world intent engine | DeepSeek 只能解释，不裁决 |
| 叙事文本 | DeepSeek + local context | 不得写正式数值事实 |
| 公开文案/截图 caption | RebornG-owned copy boundary + 用户最终批准 | MiroFish 仅候选，不能直接发布 |

## v1.0 的一句话外部口径

安全口径：

`RebornG v1.0 是一个以青茅山到南疆早期路线为起点的 AI 驱动文字 RPG 早期正式版。玩家可以在低阶蛊师阶段处理修行、生存、路线、人际和势力压力；自由目标会被本地世界规则裁决，而不是由 AI 随口满足。`

禁止口径：

- “完整开放蛊真人世界”。
- “任意时代任意身份开局”。
- “任何目标都可以达成”。
- “完整南疆/五域/商家城/蛊仙期”。
- “可获得仙蛊/尊者传承/宝黄天交易”。

## a1 结论

a1 不改 runtime。它的作用是把 v1.0 的承诺压实：后面所有 b1-b4 实现都必须服务这个接受标准，不得临时扩张成完整蛊界。
