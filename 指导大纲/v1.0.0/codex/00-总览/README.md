日期：2026-05-19

主题：`活世界早期正式版`

## 定位

`v1.0.0` 不是“完整蛊界”终局版本，而是 RebornG 的第一个可公开验收的活世界早期正式版。

它要证明四件事：

1. 玩家能从青茅山走向南疆早期路线，体验到连续的人生承接。
2. 玩家自由意图会被本地系统解析、裁决、降级、拒绝或转成前置条件，而不是由 DeepSeek 随口满足。
3. NPC、势力、经济、修行、炼养用、战斗、路线压力之间有可读回流，玩家能看懂自己影响了什么。
4. 公开版不泄露隐藏事实、不承诺未开放内容、不暴露高风险密钥或私有 canon。

## 明确非目标

除非用户另行批准，v1.0 不做：

- 完整蛊界、完整南疆、完整商家城、五域大地图。
- 蛊仙期、仙蛊、仙材、宝黄天正式交易、尊者线、宿命大战、梦境大体系。
- 任意时代开局、任意人物附身、盗天魔尊时代等大时代模式。
- 正式 BFF/backend、云存档、账号体系、自动 EdgeOne 部署。
- DeepSeek 权限扩大到写正式奖励、地点、阵营、NPC 生死、结局。
- MiroFish 输出直接进入 runtime、canon 真相源、DeepSeek context 或公开文案。
- 大规模新图生成或替换全部美术风格。
