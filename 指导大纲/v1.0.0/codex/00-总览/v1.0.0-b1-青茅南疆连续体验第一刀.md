## 目标

b1 的目标不是正式开放南疆，也不是写入路线/地点持久字段，而是把 v0.11 至 v0.18 已有的离山目标、路线准备、候选承接和压力回流合并成一个玩家可见的 v1.0 连续体验验收点。

玩家应该能看懂：

1. 自己已经在准备离开青茅山。
2. 系统知道存在南疆早期外缘路线候选。
3. 补给、身份、担保、追索、残余势力压力会影响后续。
4. 当前仍然没有正式进入新地点、转阵营、拿奖励或决定 NPC 生死。
