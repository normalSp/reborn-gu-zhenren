# v1.0.0-b2 低阶蛊师 life loop 释出版闭环

日期：2026-05-19
状态：runtime 第一刀已实现，本地 full gate 已通过，待 commit / push / GitHub Actions

## 目标

b2 的目标不是做正式经济系统、库存系统或商店系统，而是把 v0.15 的补给、喂养、炼养用、市场窗口，和 v0.18/v1.0-b1 的路线连续体验收束成一个玩家可理解的低阶蛊师生活闭环。

玩家应该能看懂：

1. 自己不是“离开青茅山就成功进入新地图”，而是进入低阶路线前的生活压力闭环。
2. 修行、补给、蛊虫维护、炼养用、交易窗口、路线压力互相牵动。
3. 当前仍然不发材料、不扣元石、不写价格表、不开放正式交易、不判定炼蛊成功、不写路线/地点状态。
4. 下一步应进入自由意图与极端意图收束、公开发布边界和 rc 长测，而不是继续扩完整南疆。
