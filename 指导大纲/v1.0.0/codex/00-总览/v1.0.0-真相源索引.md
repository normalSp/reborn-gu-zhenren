## v1.0 运行时权威边界

DeepSeek 仍只拥有：

- 叙事。
- 候选。
- 线索。
- 传闻。
- 压力。
- 请求/解释。

本地 canon/engine/store 仍拥有：

- 正式奖励。
- 正式地点。
- 正式阵营。
- NPC 生死/抓捕/命运。
- 正史硬事实。
- 隐藏事实保护。
- 战斗结算。
- 存档字段与迁移。

MiroFish 只拥有：

- quote-redacted 候选材料。
- source pointer。
- review status。
- 可转写的 boundary/test sample。
