## Player Advocate 发现

本轮无 P0/P1 阻断。后续可进入 v1.x/v2.0 的增长项：

- 正式 route/location/currentRegion 持久字段需要单独设计门禁与 `SAVE_FORMAT_VERSION = 23` 决策。
- 正式库存、材料、价格、市场、商队/黑市/委托收益需要独立经济专项。
- live DeepSeek 文风/成本/缓存评估需要用户批准后单独 probe，不能混入 rc。
- 任意时代、任意人物附身、蛊仙期、宝黄天正式交易、尊者线属于长期愿景，不进入 v1.0。
