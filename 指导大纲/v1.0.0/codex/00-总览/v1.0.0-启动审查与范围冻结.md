# v1.0.0 启动审查与范围冻结

日期：2026-05-19
状态：草案，等待用户审阅/批准

- `v0.11.0`：活世界地基与自由意图闸门。
- `v0.12.0`：青茅山正史锚点与 IF 框架。
- `v0.13.0`：NPC 与势力反应系统。
- `v0.14.0`：青茅后续路线承接。
- `v0.15.0`：低阶蛊师经济、补给、炼养用深循环。
- `v0.16.0`：系统收束、UI 减法与权威归并。

- `v0.18.0`：南疆路线与多区域承接。
- `v0.19.0`：内容生产、长测与 v1.0 发布工具。

当前关键事实：

- 包版本仍为 `0.9.0`。
- 存档版本仍为 `SAVE_FORMAT_VERSION = 22`。
- DeepSeek 运行模型为 `deepseek-v4-flash`。
- v0.19 未新增存档字段、正式 route/location、正式奖励/阵营/NPC 生死、BFF/backend、自动部署、公开承诺或 DeepSeek 权限。

## 范围冻结建议

v1.0 范围冻结为：

`青茅山 + 南疆早期路线承接 + 低阶蛊师 life loop + 自由意图裁决 + NPC/势力/路线回流 + 公开安全与长测收束`

冻结后不纳入：

- 完整南疆、完整商家城、完整五域。
- 蛊仙期和尊者线。
- 任意时代开局。
- 完整自由行动大世界。
- 正式云存档、账号、BFF/backend。
- 自动部署。
- 未经用户批准的公开发布承诺。

## 专家团审查

| 专家角色 | 结论 |
|---|---|
| 项目总策划 | v1.0 必须证明“活世界早期正式版”，不能继续只铺地基 |
| 蛊真人世界观审校 | 公开版必须保护隐藏事实和正史核心因果，MiroFish 仍是候选材料 |
| 玩法/数值架构师 | 低阶蛊师闭环要收束为可玩循环，不扩充全部蛊虫/杀招 |
| 前端创意导演 | UI 继续做减法，避免开发入口和未开放高阶入口干扰正式体验 |
| Motion/Technical Artist | v1.0 只整理 hero/截图/短录屏候选，不大规模重皮 |
| DeepSeek/模型调优专家 | 不切模型，不扩大权限；v1.0 可做固定样本 live probe，但需用户批准 |
| QA/发布稳定专家 | rc 建议 150 轮 Player Advocate，并加入旧档、移动端、公开 copy 和 EdgeOne smoke |
| Codex 操作安全官 | 每小版本更新文档、测试、仪表盘、Git；不使用自动部署或可写子代理 |

## 当前需要用户决策

详见 `v1.0.0-需求决策池.md`。启动前最重要的决策是：

1. 是否批准 v1.0 定位为“活世界早期正式版”，而非完整蛊界。
2. 是否允许 v1.0 在 b1 前进行 route/location 持久字段设计门禁；若真正需要新增字段，再单独停手批准 `SAVE_FORMAT_VERSION = 23`。
3. 是否批准 v1.0 先不做 BFF/backend，只做发布威胁模型；若公开版必须保护共享 API key，则后续另开 BFF 决策。
4. 是否批准一次性请求 v1.0 三个 MiroFish 包。
5. 是否批准 v1.0 rc Player Advocate 提升到 150 轮以上。

## MiroFish 判定

v1.0 涉及 canon/IF/路线/NPC/势力/公开文案，因此需要 MiroFish，但不应让 MiroFish 变成 runtime 权威。

建议请求：

1. `v100_qingmao_southern_border_continuity_pack`：blocking before b1 runtime absorption。
2. `v100_low_rank_life_loop_release_boundary_pack`：preferred before b2/b3 final wording。
3. `v100_public_release_copy_art_boundary_pack`：blocking before public copy/art approval。

请求文件已写入 `指导大纲/vMiroFish/requests/v1.0.0/`。

## 启动通过标准

v1.0 启动审查通过需要：

- 用户批准 D-001 至 D-010 中的必要项。
- MiroFish 请求已转交或用户确认先用现有材料启动 a0/a1/a2。

## 当前建议

批准 v1.0 启动，但按“先文档/威胁模型/接受标准，再 runtime b1”的顺序推进。这样不会把远期愿景一次性压进 v1.0，也能保证 v1.0 真正像一个可以公开验收的版本。
