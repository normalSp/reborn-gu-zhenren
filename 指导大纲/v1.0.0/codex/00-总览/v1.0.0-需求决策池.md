## 自动推进边界

若用户批准 D-001 至 D-007，Codex 可自动推进：

- a0/a1/a2 文档、威胁模型、接受标准。
- MiroFish request 文件与 intake review。
- v1.0 测试矩阵、真相源索引、Git 计划。
- 不改 runtime 的流程/文档/检查清单。

仍必须停手：

- `SAVE_FORMAT_VERSION = 23`。
- route/location/currentRegion 持久字段。
- 正式地点开放、正式阵营转移、正式奖励、NPC 生死/抓捕。
- DeepSeek 权限扩大。
- BFF/backend 实施。
- live DeepSeek probe 样本执行。
- 对外发布承诺。
- 大规模新图生成或正式替换 hero。
