# v3.4.0 前置授权包

日期：2026-05-24
状态：proposed；pending user decision

## 授权包说明

v3.4 需要先由用户选择路线。专家团推荐路线 2，但当前尚未获批。

批准本包中的 route 2 时，只表示打开一个非常窄的 runtime implementation 口子：最小 `transient proposal-only` L2/L3 第一刀。不表示打开 save、live DeepSeek、MiroFish、backend、external framework、L4/L5、正式地点/阵营/奖励/NPC 生死。

## 路线选择

| 路线 | 内容 | 专家建议 |
|---|---|---|
| Route 1 | 继续 report-only hardening | 可选；最安全但边际收益下降 |
| Route 2 | 单独打开最小 transient proposal-only runtime first cut | 推荐 |
| Route 3 | 继续 defer 到 v3.5+ | 不推荐，除非用户希望继续零 runtime 风险 |
