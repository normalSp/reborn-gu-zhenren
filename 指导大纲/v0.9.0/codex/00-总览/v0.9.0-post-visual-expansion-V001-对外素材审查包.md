# post-v0.9 V-001 对外素材审查包

日期：2026-05-13
状态：完成第一轮审查。用户已在 2026-05-13 允许制作一组 v0.9.0 新功能对外截图/短录屏素材；本文把当前素材分成“可用于 v0.9.0 公告候选 / 仅内部审查 / 禁用”三类。

## 审查结论

当前可用于 v0.9.0 公告候选的第一组素材应是“重新捕获的青茅凡战 UI 截图或短录屏”，而不是直接把单张底图或具体人物战斗图当宣传图。

受控捕获已经完成，记录见 `v0.9.0-post-visual-expansion-V001-素材捕获记录.md`，产物位于 `artifacts/v0.9.0/post-visual-expansion/V001/`。本轮允许把其中的桌面总览、月光执行、白玉护体、禁忌失败和短录屏作为 v0.9.0 公告候选素材；移动端 reduced-motion 截图仍偏技术证明，不建议做主视觉。

原因：

- b3/b3-5 的强项是“可读游戏 UI + 关键节点演出”，需要通过截图或短录屏展示。
- C-035 泛用底图是运行时氛围层，不是完整宣传图。
- 方源对白凝冰、青书对白凝冰等 bitmap 图是具体人物战斗图，不能冒充 generic 演武背景。
- 当前已新增 `npm run capture:qingmao:v001` 作为受控捕获入口，避免继续依赖 `test-results` 中易被覆盖的 Playwright 附件。

## 推荐对外素材包结构

第一轮对外素材不要超过 4 件：

1. 桌面青茅凡战全景截图。
2. 月光蛊执行短截图或 5-8 秒片段。
3. 白玉蛊护体或禁忌失败二选一，优先展示“可读失败原因”。
4. 移动端 reduced-motion 截图，作为技术可信度证据，不做主视觉。

暂不建议：

- 直接拿 C-035 SVG 底图做宣传主图。
- 直接拿具体人物 bitmap 战斗图做当前版本主视觉。
- 一次性发很多蛊虫图，避免让玩家误以为 v0.9 已完成完整蛊虫资产库。
