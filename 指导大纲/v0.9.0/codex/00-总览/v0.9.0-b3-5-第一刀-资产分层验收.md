# v0.9.0-b3-5 第一刀：视觉资产 manifest 分层验收

日期：2026-05-13

## 结论

`v0.9.0-b3-5` 第一刀已完成。本轮只做 C-034：视觉资产 manifest 分层，不生成新图、不改战斗规则、不做全局重皮。

现在青茅山视觉资产被明确分成：

- `active`：可进入当前通用运行时 UI 的安全资产。
- `candidate`：未来可被审查后进入通用运行时的候选资产。
- `review-only`：仅作美术/剧情节点参考，不能进入当前通用战场背景。
- `blocked`：当前阶段禁止进入运行时。
