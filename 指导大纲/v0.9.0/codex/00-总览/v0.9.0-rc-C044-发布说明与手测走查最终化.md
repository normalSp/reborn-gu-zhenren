# v0.9.0-rc C-044 发布说明与手测走查最终化

日期：2026-05-13

## 目标

把 rc 发布前最容易混淆的两类文档收束清楚：

- 对外口径：玩家能体验什么、系统边界是什么、哪些内容暂不承诺。
- 内部走查：发布前要用哪些入口、存档、视口和命令确认体验没有回退。

## 本轮改动

- `v0.9.0-rc-发布说明草案.md` 更新为候选公告底稿。
- 增加推荐体验入口：古月开局、三王山小队战、宝黄天经济边界、福地/洞天边界、青茅凡战。
- 明确世界观边界：DeepSeek 不写正式数值/奖励/归属；凡人不正式交易宝黄天；青茅凡战不暗示仙蛊、永生、十转、宿命蛊归属；酒虫不作为普通攻击按钮。
- `v0.9.0-rc-手测走查清单.md` 增加自动化门禁、青茅底图/入场演出检查、无 API key 场景和手测结论模板。
- `v0.9.0-需求决策池.md` 增加 C-044 与后续 C-045。

- 不在 C-044 中切换 DeepSeek 正式叙事模型；该事项已在后续 C-049 专项中完成。
