## 世界观与美术边界

- 白玉蛊只表现凡人护体，不表现仙体、永生或绝对免伤。
- 禁忌门槛只表现失败、反制、窍壁压力或风险，不表现未开放禁术奖励。
- 酒虫仍只作为战前真元质变/场景支持，不进入普通攻击行动卡。
- 春秋蝉资产继续留在 manifest 的 `blocked` 状态，不进入 b3 凡战 UI。
- UI 与 GSAP 仍只消费 `BattleResolutionStep`、canon 资产和 battlefield 状态，不私算伤害、命中、消耗、状态、胜负或奖励。

## 源码证据

- `triggerQingmaoForbiddenThresholdFailure()`：e2e harness 专用失败触发，选择月光蛊对非直线格 `c1_0` 发起动作，由本地 battlefield 校验产生 `target_out_of_range` 失败轨迹。
- `attachBattlefieldScreenshot()`：Playwright 将白玉护体与禁忌门槛截图作为测试附件沉淀，避免在 UI 仍迭代时引入脆弱 snapshot baseline。
- `check-qingmao-visual-assets.mjs`：扫描 `src/canon/qingmao-visual-assets.json`，验证 active/candidate 的 `/rebrng/` 文件存在、ID 不重复、必需 active 资产齐全、春秋蝉不在 active。

- `npm run build` 的 chunk warning 仍是历史体积风险，b3-4 不处理代码分包。
