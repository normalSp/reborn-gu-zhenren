## 捕获命令

```powershell
npm run capture:qingmao:v001
```

脚本：`scripts/capture-qingmao-v001-materials.mjs`
输出目录：`artifacts/v0.9.0/post-visual-expansion/V001/`

脚本行为：

- 自动启动或复用 `http://127.0.0.1:5173` 的 Vite 页面。
- 通过 e2e harness 进入青茅凡战样板。
- 捕获桌面总览、月光蛊执行、白玉蛊护体、禁忌门槛失败、移动端 reduced-motion 和一段 v0.9.0 功能短录屏。
- 生成 `manifest.json`，记录文件大小、sha256、版本号和世界观边界。
- 若浏览器 console 出现严重错误，脚本会失败。

- 桌面截图已经能表达“可读游戏 UI + 关键节点演出”的方向，适合作为小规模公告主图候选。
