# Codex 插件使用清单

## 当前优先使用

| 插件 | 使用场景 | 边界 |
|---|---|---|
| Browser | 本地网页打开、截图、交互验证 | 不替代 Playwright 回归。 |
| Build Web Apps | 前端 UI、组件、可用性建议 | 不越过 RebornG 视觉边界。 |
| Game Studio | 浏览器游戏架构、2D/3D/动效/playtest | 只用于游戏体验，不替代世界观审校。 |
| GitHub | 后续同步 issue、PR、CI 状态 | 第一阶段不强依赖。 |
| CodeRabbit | PR 代码审查 | 只作为补充审查，不替代本地测试。 |
| Plugin Eval | 验证 skill 质量 | 优先用于 `reborn-expert-council` 和核心 skill。 |
| Sentry | 公布后异常监控 | 发布前再接入。 |
