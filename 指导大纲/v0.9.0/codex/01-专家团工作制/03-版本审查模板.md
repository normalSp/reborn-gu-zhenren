## 必填口径

- 阶段状态必须与 `00-总览/README.md`、阶段跟踪、路线图、上下文交接一致。
- 如果 `npm test`、`npm run build` 或 e2e 未跑，必须写明原因。
- 如果 `npx tsc --noEmit` 仍有历史类型债，不作为本轮阻断，但必须进风险池。
