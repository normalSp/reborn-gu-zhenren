# v0.10.0 小版本执行路线图

日期：2026-05-15
状态：`v0.10.0` 已完成并作为本地开发里程碑锁定；D-006 EdgeOne 公测准备不纳入本轮。

## 小版本路线

| 小版本 | 目标 | 交付 | 验收重点 |
|---|---|---|---|
| `v0.10.0-a1` | 范围冻结与真相源建模 | 已完成第一刀：`qingmao-region-board.json`、三寨身份压力表、线索/行动槽、存档迁移判断 | 本阶段不提升 `SAVE_FORMAT_VERSION`；非古月出身不被改写 |
| `v0.10.0-a2` | 区域线索池与行动入口 | 已完成第一刀：`v010-qingmao-region-engine.ts` 按身份筛线索、生成行动候选、验证 AI 候选越界 | DeepSeek 只写候选；本地解释 AP、地点、风险、阻断原因 |
| `v0.10.0-b1` | 青茅山首批正式行动闭环 | 已完成最小闭环：族学/道场和山道巡查已走本地闭环并接入 store/UI；三寨委托因持久化状态暂缓 | 行动账本、回流文本、本地结算一致；无重复扣 AP |

| `v0.10.0-b4` | 青茅山视觉与可读性扩展 | 已完成第一刀：3 个场景变体、3 张候选 SVG 氛围图、ActionPanel 可读性卡、截图构图/短录屏第一步 | 不全局重皮；不暗示高阶事实；candidate 资产不替换 active 底图；reduced motion 有静态信息 |
| `v0.10.0-rc` | 发布候选质量收束 | 已完成并锁定：全量测试、tsc、build、资源扫描、青茅资产扫描、关键 e2e、长测、生产预览黑屏烟测、风险池；锁定前补充青茅战场棋盘/行动卡可读性回归 | `v0.10.0` 完成；D-006 EdgeOne 公测仍不纳入 |

## 专家团建议顺序

1. `v0.10.0` 已完成；下一步进入 `v0.11.0` 范围讨论与冻结。
2. 若后续需要保存区域压力、区域线索历史、委托进度或经济奖励，先暂停并重新决策 `SAVE_FORMAT_VERSION`。
3. `v0.11.0-a1` 已批准为首个候选专项，但仍需在 v0.11 范围讨论后按阶段启动，不抢跑架构/存档治理。

## 阻断条件

- 新增持久化区域状态前，必须重新确认并提升 `SAVE_FORMAT_VERSION`。
- 低阶资源循环没有经济上限时，不开放可重复收益入口。
- 世界观审校未通过时，不把 DeepSeek 生成的蛊、材料、地点、任务或奖励写入 canon。

## 固定验收命令

每个实现阶段至少根据改动范围选择：

- `npx tsc --noEmit --pretty false`
- `npm test`
- `npm run build`
- `npm run check:runtime-assets`
- `npm run check:qingmao-assets`
- 相关 Playwright e2e

如后续用户重新批准 EdgeOne 公测准备，再另开 D-006 复议或独立发布阶段。
