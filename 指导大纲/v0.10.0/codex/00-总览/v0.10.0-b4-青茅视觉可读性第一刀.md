# v0.10.0-b4 青茅视觉与可读性第一刀

日期：2026-05-15
状态：第一刀完成；不新增存档字段，不提升 `SAVE_FORMAT_VERSION`。

## 目标

把 `v0.10.0` 已经跑通的青茅区域行动、战斗候选和炼养用资源，整理成玩家能看懂、也能服务截图/短录屏的三个场景变体：

- `族学演武坪`：服务族学/道场磨练、族学切磋候选、5x3 凡战入口。
- `前山巡查路`：服务山道巡查、伏击风险、野外行动与凡战候选分层。
- `月草坡与酒坊支线`：服务月华草、美酒、白玉蛊缺口、喂养和残方门槛展示。

## 运行时事实

- 新增 `src/canon/qingmao-scene-variants.json`，作为 b4 场景变体、构图和短录屏脚本真相源。
- 新增 `src/engine/v010-qingmao-scene-variants.ts`，把区域行动、资源入口和战斗 readiness 投影成只读 UI 卡片。
- `ActionPanel` 新增“青茅场景变体”区块，显示：
  - 当前变体状态：可演示、候选可读、当前阻断或构图参考。
  - 关联的运行时入口。
  - 截图构图第一要点。
  - 短录屏第一步。
  - 禁止暗示的世界观/奖励边界。
- `src/canon/qingmao-visual-assets.json` 新增 3 个 `candidate + generic_candidate` 氛围资产。
- `public/rebrng/scenes/s0-qingmao/` 新增 3 张候选 SVG 氛围图：
  - `qingmao-clan-school-courtyard-variant.svg`
  - `qingmao-mountain-patrol-ridge-variant.svg`
  - `qingmao-resource-grove-variant.svg`

## 边界

- 本阶段是可读性与素材规划，不是全局重皮。
- 新增资产是候选氛围图，不替换 b3 凡战 active 泛用底图。
- 不新增存档字段；`SAVE_FORMAT_VERSION` 继续为 `21`。
- 不开放新奖励、蛊虫、蛊方、材料来源、声望、地点或三寨压力状态。
- 白玉蛊仍只显示缺口，不开放稳定 `碎玉片` 来源。
- DeepSeek 不能通过这些场景变体写入正式奖励或原著硬事实。
