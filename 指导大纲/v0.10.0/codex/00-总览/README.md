## v0.10.0 推荐主题

专家团建议把 v0.10.0 定为：

`青茅山三寨区域主线可玩化`

## 用户决策结果

用户已在 2026-05-13 批准以下事项：

1. D-001：v0.10 主线锁定为“青茅山三寨区域主线可玩化”。
2. D-002：采用“小内容包高质量”策略，先做 12-18 个凡蛊/资源/战斗表达的可玩闭环，不一次性铺 100 多种蛊虫。
3. D-003：采用“canon-near + IF 支线”口径，保留原著主线硬边界，允许玩家以不同出身参与青茅山压力线，但不改写重大原著结果。
4. D-004：视觉继续以青茅山为主，做区域场景变体、截图构图和短录屏脚本，不做全局重皮。
5. D-005：不再评估其他模型，只优化 `deepseek-v4-flash` 的缓存、JSON、retry 成本和世界观闸门。
6. D-007：只有新增持久化字段才提升 `SAVE_FORMAT_VERSION`；具体是否提升在 `v0.10.0-a1` 决定。

用户明确决定 D-006 不纳入本轮：`v0.10.0` 不把 EdgeOne 公测准备作为 rc 必选范围。
