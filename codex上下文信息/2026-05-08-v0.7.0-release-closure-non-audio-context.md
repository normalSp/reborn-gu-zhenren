# 2026-05-08 v0.7.0 非音频发布收束上下文交接

## 当前分支与版本

- 当前分支：`codex/v070-release-closure`
- 当前版本标识：`package.json = 0.7.0-pre.0`
- 当前存档格式：`SAVE_FORMAT_VERSION = 14`

## 本轮已完成

1. P0 provenance 全量发布闸门
   - 扩展 `provenance-gate` 的运行时池覆盖审计。
   - 校准 `training-grounds.json` 的 `_meta.totalGrounds`。
   - 覆盖蛊虫、仙蛊、杀招、NPC、训练场、遭遇、商店、宝黄天、秘境与 GuUseRegistry 的有效来源策略。

2. P0 经济套利合并复验
   - 新增 release 级 20/100/300 回合套利审计。
   - 覆盖宝黄天、成就奖励、势力五账本、仙窍产出、小队战奖励和小队外派收益。
   - 验收口径保持：20 回合买不起六转仙蛊，100 回合一个六转大目标，300 回合多个六转资产但七转仍是战略门槛。

3. 小队外派 UI 与奖励回流
   - 新增持久化 `squadDispatchState`，存档格式升到 14 并补迁移默认值。
   - `SquadFormationPanel` 增加外派区域，支持任务、成员、成功率、风险、持续回合、预期回流和不可派遣原因。
   - 外派奖励走闸门：元石走经济服务，材料只映射到已登记蛊材，未知内容进入传闻/线索。

4. 地形/阵法与十绝体解释显性化
   - `SVGMapPanel` 显示地形/阵法战斗预览：伤害、命中、逃跑和事件风险修正。
   - `AperturePanel` 显示十绝体空窍压力、安全回合、亲和/禁制流派、警告和禁止操作原因。
   - 这两项均为派生 UI，不新增第二套数值源。

5. 长测扩大与专项存档修正
   - `installE2eHarness` 摘要增加外派、地形、十绝体压力和宝黄天全品类池字段。
   - `v070-long.spec.ts` 从 3 条路径扩展到 12 条路径。
   - `v070c_aperture_visual_pressure.json` 原本缺十绝体空窍数据，已补 `mortalAperture`，使专项存档与测试目标一致。

## 已知非阻塞警告

- 长测期间仍有 BGM 缺文件与 public manifest 导入警告：
  - 这是用户明确要求“音频资源真实落地先不管”的范围。
  - 不作为本轮非音频收束失败项。
  - 后续真实音频 P0 需要处理资源文件、manifest 路径和 runtimeEnabled。

## 下一步建议

1. 发布前最后 P0：真实音频资源落地。
   - 人工核对 B 站参考视频曲目与时间戳。
   - 补方源《年轮》、龙公《春泥》、五域 BGM、打击 3、成就解锁和 UI 音效。
   - 更新 `audio-source-manifest.json` 与 `character-bgm-manifest.json` 并启用 runtime。

2. 发布候选包前建议新增一次人工手测记录。
   - 使用 `测试存档/v0.7.0` 的 v070c 专项存档。
   - 覆盖外派、地形、十绝体、宝黄天全品类、事件日志、战斗闪图、成就和 DeepSeek 闸门。

3. 若需要静音发布包或公开演示包，建议新增音频资源未启用时的安静降级开关，避免缺文件 warning 干扰验收。
