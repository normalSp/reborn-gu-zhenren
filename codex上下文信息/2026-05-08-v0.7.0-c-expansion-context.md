## 已验证结论
- Playwright 三路径长测基线可跑，不需要 DeepSeek；若 `.env` 缺失，不视为测试失败。
- BGM runtime manifest 不启用旧合成资源；方源/龙公角色曲已登记但未启用，等待用户放入确认音频。
- 战斗/成就缺真实 SFX 时不会再播放合成音冒充，只写诊断日志；UI 可保留轻量合成 fallback。
- 杀招/重要仙蛊闪图不阻塞战斗结算；没有图片时显示 fallback 色块和名称。
- 地形/阵法只提供轻量修正并夹取，不覆盖 `combat-config.json` 的境界、流派、道痕地基。
- 小队外派和第二空窍宝黄天交易都只产出候选/线索，不能绕过经济、任务、炼蛊和传闻闸门。
- 十绝体灾劫/压力画像为派生计算，未新增持久化字段，因此没有提升 `SAVE_FORMAT_VERSION`。
