## a1 冻结结论

正式玩家主入口收束为：

`地图 / 行动 / 角色 / 蛊道 / 世界 / 记录`

运行时地图文案可保留 `舆图` 风味，例如 `青茅舆图`。

关键归并：

- `行动 + 自由目标` -> `行动` 工作台。
- `蛊虫 / 杀招 / 炼蛊 / 蛊材` -> `蛊道` 工作台。
- `青茅 / 凡战 / 群像 / 大阵` -> 开发/演示抽屉。
- `传承 / 宿命 / 终局 / 商会` -> `世界` 下的境界、剧情、场景化入口。

## 权威边界

- UI 只负责分组、折叠、隐藏、说明候选/只读/未开放状态。
- `ActionPanel` 继续承接当前行动、青茅区域行动、资源 loop、战斗候选。
- `FreeGoalPanel` 继续承接自由目标、路线/交易/社会候选和后续行动。
- World Intent Engine 是入口裁决器/路由器，不能替代战斗、经济、奖励、地点权威。
- DeepSeek 仍只能写叙事、候选、线索、传闻、压力文本。
- MiroFish 仍只是候选材料来源，blocking need 当前为 `not_needed`。
