## 已完成内容

### 1. modifier 真相源

- 新增 `src/canon/modifier-registry.json`
  - 出身：商家、铁家、东海商帮、墨家绿洲等。
  - 天赋：`talent_steady` / `t_steady`、`talent_refinement_savant` / `t_refinement_savant`、`talent_hundred_refinements` / `t_hundred_refinements`、`talent_thrifty` / `t_thrifty`、`talent_barter` / `t_barter`。
  - 道具：`便携炼蛊工具包`。
  - 效果类型：炼蛊成功率、材料消耗、时间消耗、反噬、商会买卖价格、补残方、仙蛊升炼等。

### 2. 运行时解析

- 新增 `src/engine/modifier-engine.ts`
  - 统一解析出身、天赋、库存/材料道具来源。

  - 提供 `applyRefineSuccessModifiers`、`applyMaterialArrayCostModifiers`、`applyMerchantPrice`、`formatModifierBreakdown` 等入口。

### 3. 炼蛊/升炼/残方补全

  - 炼蛊/升炼材料消耗、时间、反噬、仙蛊升炼概率接入 modifier。
- 修改 `src/engine/recipe-discovery.ts`
  - 残方补全成功率走 `fragment_complete` modifier。

### 4. 商会和 UI
