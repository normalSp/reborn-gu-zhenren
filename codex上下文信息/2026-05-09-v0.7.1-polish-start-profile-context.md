# 2026-05-09 v0.7.1-polish 开局身份路由上下文交接

## 用户目标

用户发现：选择南疆一转开局时，如果出身选商家、武家、熊家等，剧情仍把玩家写在古月山寨里，甚至写成古月族人。用户要求先解决 v0.7.1 的发布级身份串线，并把各出身深线剧情写入 v0.8.0 大纲。

## 本轮决策

- v0.7.1 只做“开局可信、身份不串线”，不硬塞各出身长篇剧情。
- v0.8.0 承接“各出身深线剧情与身份长线”。
- 南疆一转采用混合分流：
  - 古月/熊家/白家：青茅山本地分支。
  - 商家：商队、商家外围或商路分支。
  - 武家：南疆超级势力边支，按风道/正道大势力口径，不再混用熊家。
  - 铁家：巡查、执法或炼器外务分支。
  - 散修：山外营地或边缘小寨分支。
- 熊家与武家拆分：熊力蛊与力道口径归熊家寨；武家一转阶段不发放越阶风道蛊。
