## 已验证结论
- 小队战不再使用裸 `Math.random`，结果可复现。
- 小队技能复用 `DuelMove`，会扣真元/仙元、进入冷却、写入 trace。
- 战斗奖励默认是经济闸门预览；B6 已把胜利奖励通过 store 真实回流，材料仍走 `addMaterial`。
- 敌方蛊虫默认死亡/自毁，不做常规掉蛊。
- 成就系统可读取小队胜利、入队次数、满编、小队重伤、阵亡、合击、越级撤退和十绝体觉醒。
- 十绝体在小队战 UI 显示亲和/禁制/高压迫提示，但特有灾劫和完整可视化留到 c。
