## 可优化体验项

- 在创建页增加筛选按钮：只看“运行时生效”、只看“待系统”、只看“叙事效果”，避免高阶天赋列表信息过密。
- 在蛊虫喂养 UI 显示“节流进度：0.35/1”，让玩家理解为什么节俭不是每次都少扣一个食料。
- 在遭遇日志中写明“莫家绿洲降低了危险遭遇风险”，让玩家感知出身/势力效果。
- P2 若做采集/侦察系统，建议优先接商队开局和三王山前夜常用场景，不要一次性铺满所有地域。

## 不确定项与建议沟通

- “每日真元恢复+5%”更适合接入调息/自然恢复，还是接入每日回合结算？这会影响玩家是否需要消耗 AP 安静盘坐。
- “修行突破失败惩罚-10%”是降低失败伤害、降低资源损耗，还是降低境界回退/心魔风险？建议用户决策后再做。
- “采集成功率+15%”需要先明确采集行动是否消耗 AP，以及失败时是否触发野外危险。
- “正道/魔道势力初始冷淡”建议接入声望/NPC 政策，而不是简单给所有 NPC 好感扣数。
