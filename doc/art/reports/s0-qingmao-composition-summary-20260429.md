# S0 Qingmao Composition Summary 2026-04-29

## 结论

现有构图规则已经能避免最明显的“固定左留白”，但还不够支撑第一卷全量重生。最近两张正式图都好看，不过都接近“前景大暗形 + 远处白色人物 + 中央道路”的空间关系。后续需要更换镜头逻辑，而不只是换角色和特效。

## 推荐构图类型

战斗图优先用这些：
- `side-scroll-pursuit`：地狼蛛三战，横向追逐和 S 形山道。
- `worm-eye-monster-gate`：雷冠狼破寨，低机位读巨兽尺度。
- `claustrophobic-cavern-attack`：猴王洞窟战，用洞顶和石柱压迫。
- `vertical-storm-layering`：一代/铁血冷/鹤灾混战，上下分层。
- `encircled-pressure-ring`：狼群、围杀、蛊虫压迫，但不要画成魔法阵。
- `split-level-duel`：崖壁、台阶、上下坡对打，避免平面对峙。

叙事场景优先用这些：
- `tiny-figure-vast-place`：青茅山、学堂、祠堂、山门。
- `threshold-entry`：花酒传承洞口、地下密室、石缝发现。
- `corridor-depth`：调查、追踪、寨门、山道纵深。
- `tableau-ritual-circle`：开窍大典、祭祀、血湖。
- `foreground-witness`：方源旁观、铁家调查、族人见证。
- `object-first-story`：贾金生、赌石、遗物、蛊虫发现。
- `aftermath-wide-silence`：青书牺牲、狼潮后、青茅山毁灭余波。

## 少用构图

- `overhead-crossfire`：青书图已经用过，后续短期少用。
- `low-ground-clash`：方源二战白凝冰已经用得很成功，短期不要再做前景黑影压场。
- `central-void-standoff`：只用于真正的心理对峙，否则容易变双人海报。
- 固定左侧 UI 留白：只给普通 UI 背景用，不给战斗图用。

## 下一张建议

下一张 `earthwolf-spider-third-battle.png` 推荐用 `side-scroll-pursuit`，但细化为“横向追逐 + S 形山道”：千里地狼蛛横穿前景，方源伏在蛛背，白凝冰在后方弯道或山脊形成追杀焦点。这样可以和前两张的低机位/斜俯视拉开。

质量上继续沿用 `quality-lock-master`：大暗形、低碎度、灰绿冷雾、连续边缘光。特别避免把蛛壳、碎石和山道画成满屏小纹理。
