## 使用规则

- 旧英文文件名保留，用于 Playwright 长测兼容。
- 中文文件名用于人工浏览器走查和视频录制准备。
- 文件内容仍是 JSON 存档，可通过游戏内读档或 e2e harness 加载。
- 新增中文开局存档覆盖南疆一转七出身：古月、熊家、白家、商家、武家、铁家、散修。
- 本目录 JSON 均已重建为当前 `formatVersion=21`，凡人蛊师档统一使用正式 `state.aperture` 字段，并补齐 v0.9.0-a3 `trainingGroundState` 道场线索账本默认值。
- 十绝体矩阵覆盖 10 种自然十绝体，并额外覆盖隐藏第十一绝体 `纯梦求真体`。
