## 11. 黑楼兰 — 《长安月下》

### Style Prompt (English)
```
Chinese Heroic Folk Opera, Pipa, Guzheng, Drums, Male Vocals Opera/Pop Powerful, BPM 88, D Minor, Heroic Spirit, Endless Watch, Martial Elegance, Ancient City Night
```

### 中文歌词

[Intro - 琵琶+大鼓 马蹄声远]
长安一片月 万户捣衣声

[Verse 1]
我在城头守了九十九夜
月光把影子拉得很长
他们说黑楼兰是个狠角色
却不知狠是因为无人可倚仗

[Pre-Chorus]
长安灯火万家
没有一盏为我
我只有这片月光
和手中未冷的刀

[Chorus - 戏腔]
长安姑娘——
我在这城墙之上等风也等你
长安城上 月如霜
我的名字刻在每一块城砖里

长安姑娘——
我用豪情换一场悲欢
长安城下 人未还
我饮尽风沙 独守城关

[Verse 2]
生于乱世 长于杀伐
一身肝胆 半生飘零
谁说女子不如男儿烈
我的刀比命还硬三分

[Bridge]
长安长安 长夜漫漫
我用一生等一个答案
那年月下谁在唱不完的歌
如今只剩风在耳边呢喃

[Final Chorus]
长安姑娘——
我已不是当年的模样
但长安城还在 月光还在
我还在 等你回来

[Outro - 琵琶独奏 fade out]
长安...长安...

## 12. 吴帅 — 《虞兮叹》

### Style Prompt (English)
```
Chinese Ancient Opera Tragic, Pipa, Guzheng, War Drums, Female Vocals Opera Tragic Heroic, BPM 85, E Minor, Hero's Last Stand, Historical Epic, Battlefield Elegy
```

### 中文歌词

[Intro - 大鼓+琵琶 战场氛围]
力拔山兮气盖世 时不利兮骓不逝

[Verse 1]
我领千军踏破西漠黄沙
一生征战 不曾低头
在最高处跌入最低谷
虞兮虞兮奈若何

[Pre-Chorus]
四面楚歌时是谁在唱
英雄末路是我自己的战场
我不怕输 只怕输了气节
放下刀容易 放下尊严难

[Chorus - 戏腔悲壮]
虞美人——
我在这战场最中央
虞兮虞兮 无可奈何
英雄泪 不轻弹 只为你而落

虞美人——
就让天地记住我最后的模样
虞兮虞兮 剑已折
但我的脊梁 还在

[Verse 2]
他们说成王败寇是天道
我说天道不过是成者的借口
我败了 但我从未背叛自己
黄沙埋骨 亦英雄

[Bridge]
若有来生 我还做吴帅
还走这一条不归路
虞兮虞兮 此生无悔
只憾未能与她共白首

[Final Chorus]
虞美人——
我在这黄沙中闭上了眼
虞兮虞兮 来世再见
我的故事 风会替我说完

[Outro - 狂风+大鼓渐弱]

## 13. 太白云生 — 《逆流Cry》

### Style Prompt (English)
```
Dark Synth-pop, K-pop Influence, Electronic Drums, Female Vocals Powerful Emotional, BPM 130, C Minor, Betrayal Scene, Cold Yet Emotional, Synthetic Coldness, Dramatic Climax
```

### 中文歌词

[Intro - 合成器Arp+808鼓机]
逆流河中 冰冷刺骨

[Verse 1]
我以为逆流河是回家的路
游了那么久 那么远
直到你的剑刺穿我的背脊
我才明白 河水为何如此冰冷

[Pre-Chorus]
你说这是蛊道的规矩
我对你而言不过是棋子
可我的血是热的
你为什么能面无表情

[Chorus]
Cry for me——
你这逆流河中的杀手
Cry for me——
你的眼睛比冰还冷

我曾在逆流河中逆流而上
以为游到尽头就是岸
现在我沉入河底
看着你的身影越来越远

[Verse 2]
太白是我 云生是我
逆流而上的我用尽了力气
不是为了活下去
是为了问一声：为什么

[Bridge - 电子Drop段]
逆流河不会倒流
你的剑不会收回
我的血融入河水
逆流河从此是我的颜色

[Final Chorus]
Cry for me——
我用最后的力气看你的眼
Cry for me——
在你的瞳孔中寻找一丝温热

逆流河啊 逆流河
请替我记住这一刻
当你的剑刺穿了我
逆流河终于不再逆流

[Outro - 合成器渐弱 fade to silence]

## 14. 墨瑶 — 《独一无二》

```
Indie Chinese Pop Ballad, Electric Guitar Ambient, Synth Pad, Female Vocals Clear Emotional, BPM 84, A Major to F# Minor, Solitary Love Confession, Modern Indie, Intimate Late-night
```

[Intro - 电吉他空间感+合成器]
你是我的独一无二

[Verse 1]
在万千人海中我遇见你
没有预兆 没有理由
像暗室中的一盏孤灯
照亮了我全部的宇宙

[Pre-Chorus]
世人都说你冷漠如冰
我却看见冰层下的星辰
不需要你回应我的温柔
能仰望你就足够

[Chorus]
你是我的独一无二
不需要谁来证明
在这偌大的天地之间
我只认你 我只能是你

你是我的独一无二
哪怕你从未为谁停留
我用余生做一枚琥珀
把你凝固在最温柔的时刻

[Verse 2]
瑶池有墨 纸上有你
我用画笔描摹你的样子
每一笔都是我的呼吸
每一划都是我的宿命

[Bridge - 吉他Solo]
若有来生
我还是会站在同样的位置
用同样的眼神
看你不经意地经过

[Final Chorus]
你是我的独一无二
用尽我全部的词汇
这世间有万千风景
我只取你这一瓢

你是我的独一无二
无人可替 无人可比
就算你永远不会知道
你是我世界里 唯一的颜色

[Outro - 合成器+吉他 fade out]
